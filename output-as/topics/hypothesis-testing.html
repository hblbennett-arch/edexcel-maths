Guided Approach

The standard structure for hypothesis testing questions:

1
State the hypotheses
\(H_0\): \(p = \text{(given value)}\) — the null hypothesis (no change). \(H_1\): \(p > \text{value}\) (one-tailed, testing for increase), \(p < \text{value}\) (one-tailed, decrease), or \(p \neq \text{value}\) (two-tailed). Read the question carefully to determine which.
2
Define the distribution under \(H_0\)
Under the null hypothesis: \(X \sim B(n, p)\) where \(n\) = sample size and \(p\) = the value stated in \(H_0\). State this explicitly.
3
Calculate the probability (p-value approach) OR find the critical region
P-value method: Calculate \(P(X \geq \text{observed})\) or \(P(X \leq \text{observed})\) depending on \(H_1\). Compare to significance level.
Critical region method: Find the largest/smallest value where the cumulative probability just exceeds the significance level.
4
Make a conclusion in context
If p-value < significance level: reject \(H_0\). If p-value > significance level: do not reject \(H_0\). Always state the conclusion in the context of the question (mention the person/claim/situation).

Common Pitfalls (from Examiner Reports)

  • Wrong inequality for the probability: For \(H_1: p > p_0\), you need \(P(X \geq \text{observed})\) NOT \(P(X = \text{observed})\). The test uses cumulative probability in the tail.
  • Wrong direction of \(H_1\): "Claims the proportion has increased" means \(H_1: p > p_0\). "Claims it has decreased" means \(H_1: p < p_0\). "Differs from" means \(H_1: p \neq p_0\) (two-tailed).
  • Two-tailed test: halving the significance level: For a two-tailed test at 5%, compare to 2.5% in each tail. Many candidates compare to 5% and get the wrong conclusion.
  • Conclusion not in context: "Reject H0" alone does not score the final mark. You must relate it to the question: "There is evidence to support the claim that the proportion has increased."
  • Critical region errors: The critical region is where you reject H0. For \(H_1: p > p_0\), the CR is \(X \geq c\) where \(P(X \geq c) \leq \alpha\). Do not confuse with \(P(X \leq c)\).

What Top Candidates Do

  • Write hypotheses using \(p\) (not \(\mu\) or \(x\)) since we are testing a proportion.
  • State the distribution explicitly: "Under \(H_0\), \(X \sim B(n, p)\)".
  • Show the probability calculation clearly, e.g. "\(P(X \geq 16) = 1 - P(X \leq 15) = 0.0377\)".
  • Compare explicitly: "0.0377 < 0.05 so the result is significant".
  • For critical regions, state both the region AND the associated probability (actual significance level).

Paper 2 — June 2018 — Question 3 (7 marks)

Naasir plays a game repeatedly. Each time he plays the game, the probability that he wins is \(\frac{1}{3}\), independent of other games.

(a)(i) Find the probability that he wins exactly 2 out of 15 games. [1 mark]

(a)(ii) Find the probability that he wins more than 5 out of 15 games. [2 marks]

(b) Naasir claims he has found a strategy that improves his chance of winning. He plays 32 games using this strategy and wins 16. Test his claim at the 5% significance level. [4 marks]

View Question Paper View Mark Scheme
Part (a)(i) Step 1: Identify the distribution and calculate P(X = 2) ▶
Each game is an independent trial with a fixed probability of success (\(\frac{1}{3}\)). This is a binomial distribution. We need \(X \sim B(15, \frac{1}{3})\) and want \(P(X = 2)\). We can use the formula or a calculator.

Let \(N\) = number of games Naasir wins. Then \(N \sim B(15, \frac{1}{3})\).

$$P(N = 2) = \binom{15}{2} \left(\frac{1}{3}\right)^2 \left(\frac{2}{3}\right)^{13}$$

Using a calculator:

$$= 0.05995... \approx 0.0599$$

M1 -- Selecting B(15, 1/3)
A1 -- awrt 0.0599
Part (a)(ii) Step 2: Calculate P(X > 5) ▶
"More than 5" means \(X > 5\), which is the same as \(X \geq 6\). The complement is easier: \(P(X > 5) = 1 - P(X \leq 5)\). We can find \(P(X \leq 5)\) directly from cumulative binomial tables or a calculator.

$$P(N > 5) = 1 - P(N \leq 5)$$

Using calculator with B(15, 1/3):

$$= 1 - 0.6184... = 0.3816...$$

$$\approx 0.382$$

A1 -- awrt 0.382
The key skill here is recognising "more than 5" means P(X > 5) = 1 - P(X ≤ 5). A common error is calculating P(X ≥ 5) = 1 - P(X ≤ 4) instead.
Part (b) Step 3: State the hypotheses ▶
Naasir "claims he has found a strategy that improves his chance" -- this means he believes the probability has INCREASED. So the alternative hypothesis is one-tailed: \(p > \frac{1}{3}\). The null hypothesis is always "no change".

$$H_0: p = \frac{1}{3}$$

$$H_1: p > \frac{1}{3}$$

(One-tailed test — testing for an increase)

B1 -- Both hypotheses stated correctly in terms of \(p\)
Hypotheses must be in terms of \(p\) (or \(\pi\)). Writing "H1: p > 1/3" is correct. Writing "H1: p ≠ 1/3" would be wrong here because the claim is specifically about improvement (one direction).
Part (b) Step 4: Define the test distribution and calculate the p-value ▶
Under \(H_0\), the probability is still \(\frac{1}{3}\). He played 32 games, so under \(H_0\): \(X \sim B(32, \frac{1}{3})\). He won 16. Since \(H_1\) is \(p > \frac{1}{3}\), we need the probability of getting a result this extreme or more extreme in the upper tail: \(P(X \geq 16)\).

Under \(H_0\): \(X \sim B(32, \frac{1}{3})\)

Observed value: \(X = 16\)

$$P(X \geq 16) = 1 - P(X \leq 15)$$

Using calculator:

$$= 1 - 0.9623... = 0.03765...$$

M1 -- Selecting B(32, 1/3) as the test model
A1 -- P(X ≥ 16) = awrt 0.0377
Part (b) Step 5: Compare to significance level and conclude ▶
We compare our p-value (0.0377) to the significance level (0.05). Since 0.0377 < 0.05, the result is significant. This means we reject \(H_0\) and accept the alternative -- but we must state this in the context of the original claim.

Since \(0.0377 < 0.05\), the result is significant.

Reject \(H_0\). There is evidence at the 5% level to support Naasir's claim that his strategy improves his chance of winning.

A1 -- Correct conclusion in context: evidence supports Naasir's claim
The conclusion must mention "Naasir" or "his" and "claim" or "strategy" or "method". Simply writing "reject H0" without context loses this mark. Also: you must have correctly compared to 0.05, not 0.025.

Paper 2 Statistics — June 2022 — Question 2 (7 marks)

A manufacturer knows that 8% of bags of sugar from supplier A will be damp. A random sample of 35 bags is taken.

(a)(i) Find the probability that exactly 2 bags are damp. [1 mark]

(a)(ii) Find the probability that more than 3 bags are damp. [2 marks]

(b) Supplier B claims their proportion of damp bags is less than 8%. A sample of 70 bags from B has only 2 damp. Test at the 10% level. [4 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Calculate binomial probabilities ▶
With 35 bags and probability 0.08 of being damp, we have \(X \sim B(35, 0.08)\). Part (i) asks for \(P(X = 2)\) and part (ii) asks for \(P(X > 3)\).

\(X \sim B(35, 0.08)\)

(i)

$$P(X = 2) = \binom{35}{2}(0.08)^2(0.92)^{33} = 0.2459...$$

$$\approx 0.246$$

(ii)

$$P(X > 3) = 1 - P(X \leq 3)$$

Using calculator:

$$= 1 - 0.7618... = 0.2382...$$

$$\approx 0.238$$

B1 -- Identifying B(35, 0.08)
A1 -- P(X = 2) correct
A1 -- P(X > 3) correct
Part (b) Step 2: Set up the hypothesis test ▶
Supplier B claims the proportion is LESS than 8%. This is a one-tailed test in the lower direction. We test at 10% significance. Under H0, we assume the proportion is still 0.08.

$$H_0: p = 0.08$$

$$H_1: p < 0.08$$

Under \(H_0\): \(X \sim B(70, 0.08)\)

Observed: \(X = 2\)

B1 -- Correct hypotheses in terms of \(p\)
M1 -- Selecting B(70, 0.08) as test distribution
Part (b) Step 3: Calculate p-value and conclude ▶
Since \(H_1: p < 0.08\), we look at the LOWER tail. We need \(P(X \leq 2)\) -- the probability of getting 2 or fewer damp bags if the true proportion is 0.08.

$$P(X \leq 2) = P(X=0) + P(X=1) + P(X=2)$$

Using calculator with B(70, 0.08):

$$= 0.0327...$$

Compare: \(0.0327 < 0.10\) (the significance level).

The result is significant. Reject \(H_0\). There is evidence at the 10% level to support Supplier B's claim that their proportion of damp bags is less than 8%.

A1 -- Correct probability P(X ≤ 2) calculated
A1 -- Correct conclusion in context
At 10% significance, p = 0.0327 is significant. Note: if tested at 5%, it would STILL be significant. But if they had used P(X = 2) instead of P(X ≤ 2), they would get the wrong answer.

Paper 2 Statistics — June 2023 — Question 4 (7 marks)

Past information shows that 25% of adults in a large population have a particular allergy. Rylan believes the proportion differs from 25%. He takes a random sample of 50 adults.

(a) Write down the alternative hypothesis for Rylan's test. [1 mark]

(b) Find the critical region for this test at the 5% level. State the probability associated with each tail (as close to 2.5% as possible). [4 marks]

(c) State the actual probability of incorrectly rejecting \(H_0\). [1 mark]

(d) Rylan finds 10 adults with the allergy. State the conclusion. [1 mark]

View Question Paper View Mark Scheme
Part (a) Step 1: Write the alternative hypothesis ▶
"Differs from 25%" means the proportion could be higher OR lower -- this is a TWO-TAILED test. The alternative hypothesis uses \(\neq\).

$$H_1: p \neq 0.25$$

B1 -- \(H_1: p \neq 0.25\)
"Differs from" always means two-tailed (\(\neq\)). If the question said "believes it has increased" that would be one-tailed (\(>\)).
Part (b) Step 2: Find the critical region (lower tail) ▶
For a two-tailed test at 5%, we split the significance: 2.5% in each tail. Under \(H_0\): \(X \sim B(50, 0.25)\). For the lower tail, we need the largest value of \(c\) such that \(P(X \leq c) \leq 0.025\).

Under \(H_0\): \(X \sim B(50, 0.25)\)

Lower tail (seeking \(P(X \leq c) \leq 0.025\)):

\(P(X \leq 7) = 0.0453...\) -- too large (> 0.025)

\(P(X \leq 6) = 0.0194...\) -- within 2.5%

So the lower critical region is \(X \leq 7\). Wait, let's be more careful:

\(P(X \leq 7) = 0.0453 > 0.025\) -- NOT in critical region

\(P(X \leq 6) = 0.0194 \leq 0.025\) -- IN critical region

Lower critical region: \(X \leq 6\) with probability 0.0194

M1 -- Finding cumulative probabilities for lower tail
A1 -- Lower CR: \(X \leq 6\)
Part (b) Step 3: Find the critical region (upper tail) ▶
For the upper tail, we need the smallest value of \(c\) such that \(P(X \geq c) \leq 0.025\). This is equivalent to finding \(c\) where \(P(X \leq c-1) \geq 0.975\).

Upper tail (seeking \(P(X \geq c) \leq 0.025\)):

\(P(X \geq 18) = 1 - P(X \leq 17) = 1 - 0.9675... = 0.0325\) -- too large

\(P(X \geq 19) = 1 - P(X \leq 18) = 1 - 0.9836... = 0.0164\) -- within 2.5%

Upper critical region: \(X \geq 19\) with probability 0.0164

M1 -- Finding cumulative probabilities for upper tail
A1 -- Upper CR: \(X \geq 19\)

Full critical region: \(X \leq 6\) or \(X \geq 19\)

Part (c) Step 4: Actual significance level (Type I error probability) ▶
The actual significance level is the true probability of rejecting \(H_0\) when it is true -- the sum of both tail probabilities. A Type I error is incorrectly rejecting a true \(H_0\), and its probability equals the actual significance level.

Actual significance level = \(P(X \leq 6) + P(X \geq 19)\)

$$= 0.0194 + 0.0164 = 0.0358$$

The probability of incorrectly rejecting \(H_0\) is 0.0358 (or 3.58%).

B1 -- Actual significance level = sum of both tail probabilities (awrt 0.0358)
This is also the probability of a Type I error. Many candidates only gave one tail probability. You must add BOTH tail probabilities together.
Part (d) Step 5: Conclusion for the observed value ▶
Rylan found 10 adults with the allergy. We check whether 10 falls in the critical region. The CR is \(X \leq 6\) or \(X \geq 19\). Since \(6 < 10 < 19\), the value 10 does NOT fall in the critical region.

Since \(10\) does not lie in the critical region (\(X \leq 6\) or \(X \geq 19\)):

Do not reject \(H_0\). There is insufficient evidence at the 5% level to suggest that the proportion of adults with the allergy differs from 25%.

B1 -- Correct conclusion: do not reject \(H_0\), insufficient evidence that proportion differs from 0.25

Paper 2 Statistics — June 2019 — Question 5 (6 marks)

A student's hypothesis test on \(X \sim B(30, 0.15)\) at 5% contains two errors: (i) the alternative hypothesis should be \(H_1: p > 0.15\), and (ii) the probability calculated should be \(P(X \geq 8)\) not \(P(X = 8)\).

(a) Identify the two errors in the student's test. [2 marks]

(b) State whether these errors affect the conclusion. [1 mark]

(c) Find the critical region for the corrected test. [2 marks]

(d) State the actual significance level of this test. [1 mark]

View Question Paper View Mark Scheme
Part (a) Step 1: Identify the errors ▶
The student made two specific errors. In hypothesis testing: (1) the alternative hypothesis must match the claim (here it should be \(>\) not \(\neq\) or \(<\)), and (2) we always use cumulative probability \(P(X \geq \text{observed})\) not the point probability \(P(X = \text{observed})\).

Error 1: The alternative hypothesis should be \(H_1: p > 0.15\) (one-tailed, not two-tailed or wrong direction).

Error 2: The probability should be \(P(X \geq 8)\) not \(P(X = 8)\). We need the cumulative tail probability, not the probability of exactly that value.

B1 -- Identifying that \(H_1\) should use \(>\)
B1 -- Identifying that \(P(X = 8)\) should be \(P(X \geq 8)\)
Part (b) Step 2: Do the errors affect the conclusion? ▶
When corrected: \(P(X \geq 8) = 0.0698\) from \(B(30, 0.15)\). Since \(0.0698 > 0.05\), the correct conclusion is "do not reject \(H_0\)". If the student incorrectly rejected \(H_0\), the errors did change the conclusion.

\(P(X \geq 8) = 0.0698 > 0.05\)

Yes, the errors affect the conclusion. With the correct test, \(H_0\) should NOT be rejected (the result is not significant at 5%).

B1 -- Stating the conclusion is affected, with correct supporting probability
Part (c) Step 3: Find the critical region ▶
For the corrected one-tailed test (\(H_1: p > 0.15\)) at 5%, we need the smallest \(c\) such that \(P(X \geq c) \leq 0.05\). Equivalently, the largest \(c-1\) where \(P(X \leq c-1) \geq 0.95\).

Under \(H_0\): \(X \sim B(30, 0.15)\)

\(P(X \leq 7) = 0.9302...\) so \(P(X \geq 8) = 0.0698 > 0.05\) -- not quite

\(P(X \leq 8) = 0.9722...\) so \(P(X \geq 9) = 0.0278 \leq 0.05\) -- in CR

Critical region: \(X \geq 9\)

M1 -- Using tables to find probability associated with critical value
A1 -- Critical region \(X \geq 9\)
Part (d) Step 4: Actual significance level ▶
The actual significance level is the probability of being in the critical region when \(H_0\) is true. Since the CR is \(X \geq 9\), the actual significance level is \(P(X \geq 9)\).

$$P(X \geq 9) = 1 - P(X \leq 8) = 1 - 0.9722... = 0.0278$$

Actual significance level = 0.0278 (or awrt 2.78%)

B1 -- awrt 0.0278
The actual significance level is always ≤ the nominal level (5% here). It equals the probability of the critical region under H0. This is also the probability of a Type I error.

How to Recognise This Question Type

Look for these signals in the question:

  • Words like "random sample", "probability", "proportion", "claims", "believes"
  • A fixed number of trials with a fixed probability -- this triggers binomial
  • "Test at the __% significance level" -- this is a hypothesis test
  • "Find the critical region" -- find the boundary values for rejection
  • "State the actual significance level" -- find the exact tail probability for the CR
  • Questions about Type I error (rejecting \(H_0\) when true) or Type II error (not rejecting \(H_0\) when false)

Key Formulae

  • Binomial probability: \(P(X = r) = \binom{n}{r} p^r (1-p)^{n-r}\) Learn this
  • Binomial conditions: Fixed \(n\) trials, fixed probability \(p\), independent trials, two outcomes Learn this
  • One-tailed (upper): \(H_1: p > p_0\), calculate \(P(X \geq \text{obs})\), compare to \(\alpha\) Learn this
  • One-tailed (lower): \(H_1: p < p_0\), calculate \(P(X \leq \text{obs})\), compare to \(\alpha\) Learn this
  • Two-tailed: \(H_1: p \neq p_0\), compare to \(\frac{\alpha}{2}\) in the appropriate tail Learn this
  • Type I error: Rejecting \(H_0\) when \(H_0\) is true. Probability = actual significance level. Learn this
