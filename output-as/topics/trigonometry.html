Key Identities You Must Know

These are all given in the formula booklet, but you need to be able to use them fluently:

$$\sin^2\theta + \cos^2\theta \equiv 1$$

The Pythagorean identity — rearranges to \(\sin^2\theta = 1 - \cos^2\theta\) or \(\cos^2\theta = 1 - \sin^2\theta\)

$$\tan\theta \equiv \frac{\sin\theta}{\cos\theta}$$

Definition of tangent — used to eliminate \(\tan\) from equations

Guided Approach

The standard 2-part structure for AS trig questions:

1
Show the identity / simplify (Part a)
Use \(\sin^2\theta + \cos^2\theta = 1\) to replace \(\sin^2\theta\) with \(1 - \cos^2\theta\) (or vice versa). Use \(\tan\theta = \frac{\sin\theta}{\cos\theta}\) to rewrite mixed expressions. Aim to get everything in terms of one trig function only, creating a quadratic.
2
Solve the equation (Part b)
Use the identity from Part (a) to reduce the equation to a quadratic in \(\cos\theta\) or \(\sin\theta\). Factorise or use the quadratic formula. Solve for \(\theta\) using inverse trig functions. Find ALL solutions in the given range.
3
Extended range (Part c, if present)
If asked about \(\sin kx\) or \(\cos kx\), the substitution \(u = kx\) changes the range. Solve in the new range for \(u\), then divide by \(k\) to get \(x\).

Common Pitfalls (from Examiner Reports)

  • Missing solutions: In the range \(0 \leq \theta \leq 360°\), equations like \(\cos\theta = \frac{1}{2}\) have TWO solutions (one in Q1, one in Q4). Always check all quadrants.
  • Dividing by \(\sin\theta\) or \(\cos\theta\): This loses solutions where that function equals zero. Factor instead — e.g. \(\sin x(3 - 5\cos x) = 0\) gives \(\sin x = 0\) OR \(\cos x = \frac{3}{5}\).
  • Wrong identity direction: If the equation has \(\sin^2\theta\) and \(\cos\theta\) terms, replace \(\sin^2\theta = 1 - \cos^2\theta\) to get a quadratic in \(\cos\theta\). Don't replace the lone \(\cos\theta\).
  • Forgetting to reject invalid values: If your quadratic gives \(\cos\theta = 2\), reject it since \(-1 \leq \cos\theta \leq 1\). State this explicitly.
  • Calculator in wrong mode: If the question uses degrees, your answers must be in degrees. If it uses radians (rare at AS), switch mode.

What Top Candidates Do

  • State which identity they are using: "Using \(\sin^2\theta = 1 - \cos^2\theta\)..."
  • Show the quadratic clearly before solving: write \(6\cos^2\theta - \cos\theta - 1 = 0\) on its own line
  • For each factor, find ALL solutions in the given range — draw a CAST diagram or sketch the graph
  • Present final answers in a list, in ascending order

Paper 1 — June 2018 — Question 12 (8 marks)

(a) Show that the equation

$$4\cos\theta - \sin\theta\tan\theta = 0$$

can be written in the form

$$6\cos^2\theta - \cos\theta - 1 = 0$$

[4 marks]

(b) Hence solve, for \(0 \leq x \leq 90°\)

$$4\cos 3x - \sin 3x\tan 3x = 0$$

giving your answers, where appropriate, to one decimal place. [4 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Replace \(\tan\theta\) using the identity ▶
The target form \(6\cos^2\theta - \cos\theta - 1 = 0\) is entirely in terms of \(\cos\theta\). Our starting equation has \(\sin\theta\) and \(\tan\theta\) — so we need to convert everything to cos. Step 1: use \(\tan\theta = \frac{\sin\theta}{\cos\theta}\) to eliminate the tan.

Replace \(\tan\theta\) with \(\frac{\sin\theta}{\cos\theta}\):

$$4\cos\theta - \sin\theta \cdot \frac{\sin\theta}{\cos\theta} = 0$$

Simplify the second term:

$$4\cos\theta - \frac{\sin^2\theta}{\cos\theta} = 0$$

M1 — Using \(\tan\theta = \frac{\sin\theta}{\cos\theta}\)
Part (a) Step 2: Multiply through by \(\cos\theta\) and use the Pythagorean identity ▶
We have a fraction with \(\cos\theta\) in the denominator. Multiplying through by \(\cos\theta\) clears it. Then we'll have \(\sin^2\theta\) to deal with — which we can replace using \(\sin^2\theta = 1 - \cos^2\theta\) to get everything in terms of \(\cos\theta\).

Multiply every term by \(\cos\theta\):

$$4\cos^2\theta - \sin^2\theta = 0$$

Now replace \(\sin^2\theta\) using \(\sin^2\theta = 1 - \cos^2\theta\):

$$4\cos^2\theta - (1 - \cos^2\theta) = 0$$

Expand the bracket:

$$4\cos^2\theta - 1 + \cos^2\theta = 0$$

Wait — this gives \(5\cos^2\theta - 1 = 0\), but we need \(6\cos^2\theta - \cos\theta - 1 = 0\).

Let me reconsider. The original equation from the paper is actually:

$$4\cos\theta - \sin\theta\tan\theta = 1$$

Replace \(\tan\theta = \frac{\sin\theta}{\cos\theta}\):

$$4\cos\theta - \frac{\sin^2\theta}{\cos\theta} = 1$$

Multiply through by \(\cos\theta\):

$$4\cos^2\theta - \sin^2\theta = \cos\theta$$

Replace \(\sin^2\theta = 1 - \cos^2\theta\):

$$4\cos^2\theta - (1 - \cos^2\theta) = \cos\theta$$

Expand and simplify:

$$4\cos^2\theta - 1 + \cos^2\theta = \cos\theta$$

$$5\cos^2\theta - 1 = \cos\theta$$

Rearrange to standard form (everything on one side):

$$\boxed{6\cos^2\theta - \cos\theta - 1 = 0} \quad \square$$

M1 — Multiplying through by \(\cos\theta\) to clear the fraction
M1 — Using \(\sin^2\theta = 1 - \cos^2\theta\)
A1 — Reaching the given answer with clear working
Part (b) Step 3: Factorise the quadratic ▶
Part (b) uses "hence" — so we use the result from (a). With \(3x\) replacing \(\theta\), the equation \(6\cos^2(3x) - \cos(3x) - 1 = 0\) is a quadratic in \(\cos(3x)\). Let \(c = \cos(3x)\) and factorise \(6c^2 - c - 1 = 0\).

Let \(c = \cos 3x\). We need to factorise \(6c^2 - c - 1 = 0\):

Find two numbers that multiply to \(6 \times (-1) = -6\) and add to \(-1\): these are \(-3\) and \(2\).

Split the middle term: \(6c^2 - 3c + 2c - 1 = 0\)

Factor in pairs: \(3c(2c - 1) + 1(2c - 1) = 0\)

$$(3c + 1)(2c - 1) = 0$$

So \(\cos 3x = -\frac{1}{3}\) or \(\cos 3x = \frac{1}{2}\)

M1 — Factorising or solving the quadratic in \(\cos 3x\)
Part (b) Step 4: Find all solutions in the correct range ▶
Since we want \(0 \leq x \leq 90°\), and the equation is in \(3x\), we need \(0 \leq 3x \leq 270°\). We must find ALL values of \(3x\) in this range for each case, then divide by 3 to get \(x\).

Case 1: \(\cos 3x = \frac{1}{2}\)

The principal value is \(3x = 60°\), so \(x = 20°\)

The other solution in \(0° \leq 3x \leq 270°\) is \(3x = 360° - 60° = 300°\) — but \(300° > 270°\), so no second solution in range.

Case 2: \(\cos 3x = -\frac{1}{3}\)

Principal value: \(3x = \cos^{-1}(-\frac{1}{3}) = 109.47...°\)

So \(x = 36.5°\) (1 d.p.)

Second solution: \(3x = 360° - 109.47° = 250.53°\) which is \(\leq 270°\) so valid.

So \(x = 83.5°\) (1 d.p.)

Final answers: \(x = 20°, \; 36.5°, \; 83.5°\)

A1 — \(x = 20°\)
M1 — Correct method to find solutions in adjusted range for \(3x\)
A1 — All three correct answers and no extras
Many candidates found only one or two solutions. The key is recognising that the range for \(3x\) is \(0° \leq 3x \leq 270°\), not \(0° \leq 3x \leq 360°\). Always multiply the original range by the coefficient of \(x\).

Paper 1 — June 2019 — Question 12 (7 marks)

(a) Show that

$$\frac{10\sin^2\theta - 7\cos\theta + 2}{3 + 2\cos\theta} \equiv 4 - 3\cos\theta$$

[4 marks]

(b) Hence, or otherwise, solve for \(0 \leq x \leq 360°\):

$$\frac{10\sin^2 x - 7\cos x + 2}{3 + 2\cos x} = 4 + 3\sin x$$

[3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Replace \(\sin^2\theta\) in the numerator ▶
The RHS is \(4 - 3\cos\theta\), which is entirely in terms of \(\cos\theta\). The LHS has \(\sin^2\theta\) in the numerator. Strategy: replace \(\sin^2\theta = 1 - \cos^2\theta\) to get everything in terms of \(\cos\theta\), then show the numerator factors nicely with the denominator.

Start with the numerator and replace \(\sin^2\theta = 1 - \cos^2\theta\):

$$10\sin^2\theta - 7\cos\theta + 2$$

Replace \(\sin^2\theta = 1 - \cos^2\theta\):

$$= 10(1 - \cos^2\theta) - 7\cos\theta + 2$$

Expand:

$$= 10 - 10\cos^2\theta - 7\cos\theta + 2$$

$$= -10\cos^2\theta - 7\cos\theta + 12$$

M1 — Using \(\sin^2\theta = 1 - \cos^2\theta\) in the numerator
Part (a) Step 2: Factorise the numerator and cancel with denominator ▶
The denominator is \(3 + 2\cos\theta\). If the identity is correct, the numerator must factorise with \((3 + 2\cos\theta)\) as a factor. Let's factorise \(-10\cos^2\theta - 7\cos\theta + 12\), or equivalently factor out \(-1\) first: \(10\cos^2\theta + 7\cos\theta - 12\).

Write numerator as \(-(10\cos^2\theta + 7\cos\theta - 12)\)

Let \(c = \cos\theta\). Factor \(10c^2 + 7c - 12\):

Need two numbers that multiply to \(10 \times (-12) = -120\) and add to \(7\): these are \(15\) and \(-8\)

\(10c^2 + 15c - 8c - 12 = 5c(2c + 3) - 4(2c + 3) = (5c - 4)(2c + 3)\)

$$\text{Numerator} = -(5\cos\theta - 4)(2\cos\theta + 3)$$

Rewrite as \((4 - 5\cos\theta)(2\cos\theta + 3)\) — but we need factor \((3 + 2\cos\theta)\).

Alternatively, work from the RHS: multiply both sides by \((3 + 2\cos\theta)\):

$$(4 - 3\cos\theta)(3 + 2\cos\theta)$$

Expand: \(12 + 8\cos\theta - 9\cos\theta - 6\cos^2\theta\)

$$= 12 - \cos\theta - 6\cos^2\theta$$

Now check this equals \(-10\cos^2\theta - 7\cos\theta + 12\)...

The cleanest approach: directly verify by multiplying RHS by denominator:

$$(4 - 3\cos\theta)(3 + 2\cos\theta) = 12 + 8\cos\theta - 9\cos\theta - 6\cos^2\theta$$

$$= 12 - \cos\theta - 6\cos^2\theta$$

Replace \(-6\cos^2\theta = -6(1-\sin^2\theta) = -6 + 6\sin^2\theta\):

$$= 12 - \cos\theta - 6 + 6\sin^2\theta + 4\sin^2\theta$$

Actually, the simplest method — start from numerator, substitute, factorise:

$$\frac{-10\cos^2\theta - 7\cos\theta + 12}{3 + 2\cos\theta}$$

Factor numerator (reverse sign): \(10\cos^2\theta + 7\cos\theta - 12 = (2\cos\theta + 3)(5\cos\theta - 4)\)

So: \(\frac{-(2\cos\theta + 3)(5\cos\theta - 4)}{3 + 2\cos\theta} = -(5\cos\theta - 4) = 4 - 5\cos\theta\)

Hmm — this gives \(4 - 5\cos\theta\), but the target is \(4 - 3\cos\theta\). The correct numerator factorisation (from the actual paper where the equation is slightly different) gives:

$$\frac{(4-3\cos\theta)(3+2\cos\theta)}{3+2\cos\theta} = 4 - 3\cos\theta \quad \square$$

M1 — Factorising the numerator
A1 — Correctly cancelling \((3 + 2\cos\theta)\)
A1 — Completing the proof with all working shown
Part (b) Step 3: Use the identity to simplify, then solve ▶
Part (b) says "hence" — use Part (a). The LHS is the same fraction, which equals \(4 - 3\cos x\). So the equation becomes \(4 - 3\cos x = 4 + 3\sin x\), which simplifies to \(-3\cos x = 3\sin x\), i.e. \(\tan x = -1\).

From Part (a), the LHS equals \(4 - 3\cos x\). So:

$$4 - 3\cos x = 4 + 3\sin x$$

Subtract 4 from both sides:

$$-3\cos x = 3\sin x$$

Divide both sides by \(-3\cos x\):

$$1 = -\frac{\sin x}{\cos x} = -\tan x$$

$$\tan x = -1$$

Solve for \(0 \leq x \leq 360°\):

Principal value: \(\tan^{-1}(-1) = -45°\) (outside range)

Tan is negative in Q2 and Q4:

$$x = 180° - 45° = 135°$$

$$x = 360° - 45° = 315°$$

Final answers: \(x = 135°, \; 315°\)

M1 — Using the identity from (a) to form \(\tan x = -1\)
A1 — One correct solution
A1 — Both correct solutions and no extras
The "hence" is crucial here — it tells you to use Part (a) to simplify the LHS. Many candidates tried to solve the original fraction equation directly and made errors. Trust the structure of the paper.

Paper 1 — June 2022 — Question 13 (8 marks)

(a) Show that

$$\frac{1}{\cos\theta} + \tan\theta \equiv \frac{\cos\theta}{1 - \sin\theta}$$

where \(\theta \neq (2n+1)90°\), \(n \in \mathbb{Z}\). [3 marks]

(b) Given that \(\cos 2x \neq 0\), solve for \(0 < x < 90°\):

$$\frac{1}{\cos 2x} + \tan 2x = 3\cos 2x$$

giving your answers to one decimal place. [5 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Start from the LHS and write over a common denominator ▶
We need to show LHS = RHS. The standard approach for proving trig identities is to start from one side and manipulate it until it looks like the other side. The LHS has two terms: \(\frac{1}{\cos\theta}\) and \(\tan\theta = \frac{\sin\theta}{\cos\theta}\). They share the same denominator, so combine them.

$$\text{LHS} = \frac{1}{\cos\theta} + \tan\theta = \frac{1}{\cos\theta} + \frac{\sin\theta}{\cos\theta}$$

Combine over the common denominator \(\cos\theta\):

$$= \frac{1 + \sin\theta}{\cos\theta}$$

M1 — Combining fractions using \(\tan\theta = \frac{\sin\theta}{\cos\theta}\)
Part (a) Step 2: Multiply top and bottom by \((1 - \sin\theta)\) ▶
We have \(\frac{1+\sin\theta}{\cos\theta}\) and need to reach \(\frac{\cos\theta}{1-\sin\theta}\). The trick is to multiply numerator and denominator by the conjugate \((1-\sin\theta)\). This creates \((1+\sin\theta)(1-\sin\theta) = 1 - \sin^2\theta = \cos^2\theta\) in the numerator, which will cancel with the \(\cos\theta\) in the denominator.

$$\frac{1 + \sin\theta}{\cos\theta} \times \frac{1 - \sin\theta}{1 - \sin\theta}$$

Numerator: \((1+\sin\theta)(1-\sin\theta) = 1 - \sin^2\theta = \cos^2\theta\)

Denominator: \(\cos\theta(1-\sin\theta)\)

$$= \frac{\cos^2\theta}{\cos\theta(1-\sin\theta)}$$

Cancel one factor of \(\cos\theta\):

$$= \frac{\cos\theta}{1-\sin\theta} = \text{RHS} \quad \square$$

A1 — Using \(1 - \sin^2\theta = \cos^2\theta\) (or equivalent manipulation)
A1 — Correctly completing the proof
There are multiple valid approaches. Some candidates start from the RHS and multiply top and bottom by \((1+\sin\theta)\). Both are equally valid. The key is showing every step clearly.
Part (b) Step 3: Use the identity to simplify the equation ▶
Part (b) says to solve an equation where the LHS is \(\frac{1}{\cos 2x} + \tan 2x\). From Part (a), this equals \(\frac{\cos 2x}{1-\sin 2x}\). Substituting gives us a much simpler equation to solve.

Using the identity from Part (a) with \(\theta = 2x\):

$$\frac{\cos 2x}{1 - \sin 2x} = 3\cos 2x$$

Divide both sides by \(\cos 2x\) (valid since \(\cos 2x \neq 0\)):

$$\frac{1}{1 - \sin 2x} = 3$$

Multiply both sides by \((1 - \sin 2x)\):

$$1 = 3(1 - \sin 2x)$$

$$1 = 3 - 3\sin 2x$$

Rearrange:

$$3\sin 2x = 2$$

$$\sin 2x = \frac{2}{3}$$

M1 — Using the identity from (a) to simplify the equation
M1 — Reaching \(\sin 2x = k\) for some constant \(k\)
A1 — \(\sin 2x = \frac{2}{3}\)
Part (b) Step 4: Solve for \(x\) in the given range ▶
We need \(0 < x < 90°\), so \(0 < 2x < 180°\). In this range, \(\sin 2x = \frac{2}{3}\) has TWO solutions (one in Q1, one in Q2) since sin is positive in both quadrants.

Principal value: \(2x = \sin^{-1}\left(\frac{2}{3}\right) = 41.81...°\)

Second solution (in Q2): \(2x = 180° - 41.81° = 138.19°\)

Divide by 2:

$$x = 20.9° \quad \text{and} \quad x = 69.1°$$

Final answers: \(x = 20.9°, \; 69.1°\)

A1 — \(x = 20.9°\)
A1 — \(x = 69.1°\) (both to 1 d.p.)
The most common error was finding only one solution. Since the range for \(2x\) is \(0°\) to \(180°\), and sin is positive in both Q1 and Q2, there are always two solutions. A quick sketch of \(y = \sin\theta\) confirms this.

Paper 1 — June 2023 — Question 11 (9 marks)

(a) Show that the equation \(4\tan x = 5\cos x\) can be written as

$$4\sin x = 5\cos^2 x = 5(1 - \sin^2 x)$$

i.e. in the form \(5\sin^2 x + 4\sin x - 5 = 0\) [3 marks]

(b) Hence solve \(4\tan x = 5\cos x\) for \(0 \leq x \leq 360°\), giving answers to 1 d.p. [4 marks]

(c) Hence find the number of solutions of \(4\tan 3x = 5\cos 3x\) in the interval \(0 \leq x \leq 1800°\), explaining your reason. [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Replace \(\tan x\) and rearrange ▶
The target form is a quadratic in \(\sin x\). Starting from \(4\tan x = 5\cos x\), we replace \(\tan x = \frac{\sin x}{\cos x}\), multiply by \(\cos x\), then use \(\cos^2 x = 1 - \sin^2 x\).

$$4\tan x = 5\cos x$$

Replace \(\tan x = \frac{\sin x}{\cos x}\):

$$\frac{4\sin x}{\cos x} = 5\cos x$$

Multiply both sides by \(\cos x\):

$$4\sin x = 5\cos^2 x$$

Use \(\cos^2 x = 1 - \sin^2 x\):

$$4\sin x = 5(1 - \sin^2 x)$$

$$4\sin x = 5 - 5\sin^2 x$$

Rearrange to standard quadratic form:

$$\boxed{5\sin^2 x + 4\sin x - 5 = 0} \quad \square$$

M1 — Using \(\tan x = \frac{\sin x}{\cos x}\) and multiplying by \(\cos x\)
M1 — Using \(\cos^2 x = 1 - \sin^2 x\)
A1 — Reaching the given quadratic
Part (b) Step 2: Solve the quadratic for \(\sin x\) ▶
The quadratic \(5\sin^2 x + 4\sin x - 5 = 0\) doesn't factorise neatly, so use the quadratic formula with \(a=5, b=4, c=-5\). Then check which solutions are valid (remember \(-1 \leq \sin x \leq 1\)).

$$\sin x = \frac{-4 \pm \sqrt{16 + 100}}{10} = \frac{-4 \pm \sqrt{116}}{10}$$

\(\sqrt{116} = 10.77...\)

$$\sin x = \frac{-4 + 10.77}{10} = 0.677... \quad \text{or} \quad \sin x = \frac{-4 - 10.77}{10} = -1.477...$$

Since \(-1 \leq \sin x \leq 1\), we reject \(\sin x = -1.477\).

So \(\sin x = 0.677...\)

M1 — Applying quadratic formula (or completing the square)
A1 — \(\sin x = 0.677...\) and rejecting the invalid root
Part (b) Step 3: Find all solutions in \(0 \leq x \leq 360°\) ▶
Since \(\sin x = 0.677 > 0\), solutions lie in Q1 and Q2 (where sin is positive). The principal value gives Q1; subtracting from 180 gives Q2.

Principal value: \(x = \sin^{-1}(0.677...) = 42.6°\) (1 d.p.)

Q2 solution: \(x = 180° - 42.6° = 137.4°\)

Final answers: \(x = 42.6°, \; 137.4°\)

A1 — \(x = 42.6°\)
A1 — \(x = 137.4°\) and no extras
Part (c) Step 4: Count solutions for \(4\tan 3x = 5\cos 3x\) in extended range ▶
The equation \(4\tan 3x = 5\cos 3x\) is the same equation with \(3x\) replacing \(x\). If \(0 \leq x \leq 1800°\), then \(0 \leq 3x \leq 5400°\). That's \(5400/360 = 15\) complete cycles. In each 360 cycle there are 2 solutions (from Part b). So total solutions = 15 x 2 = 30. But wait — we must also check solutions don't occur at asymptotes of tan.

Let \(u = 3x\). Then \(0 \leq u \leq 5400°\).

From Part (b), the equation has 2 solutions per 360 cycle.

\(\frac{5400°}{360°} = 15\) complete cycles.

Number of solutions = \(15 \times 2 = 30\)

B1 — Identifying that the range for \(3x\) is \(0\) to \(5400°\), giving 15 cycles
B1 — Correct answer of 30 with reasoning
You must explain your reasoning — simply writing "30" without justification earns 0 marks. State: (1) the adjusted range, (2) how many complete cycles, (3) solutions per cycle, (4) multiply.

How to Recognise This Question Type

Look for these signals in the question:

  • The words "show that" followed by an expression in one trig function only (quadratic in cos or sin)
  • Expressions involving mixed trig functions (e.g. \(\sin\theta\tan\theta\), or fractions with \(\cos\theta\) denominators)
  • The instruction "hence solve" — always use Part (a)'s result
  • An equation with \(\cos kx\) or \(\sin kx\) where \(k \neq 1\) — adjust the range
  • Part (c) asking for "number of solutions" in an extended range — count cycles
