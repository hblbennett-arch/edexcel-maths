Guided Approach

Vector questions at AS level typically involve position vectors in 2D or 3D:

1
Find displacement vectors using \(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\)
The vector FROM point A TO point B is found by subtracting: position vector of B minus position vector of A. This is the single most important formula for vector questions.
2
Use parallel/collinear conditions
Two vectors are parallel if one is a scalar multiple of the other: \(\overrightarrow{AB} = k\overrightarrow{CD}\). Points A, B, C are collinear if \(\overrightarrow{AB} = k\overrightarrow{AC}\) (or any pair sharing a point).
3
Magnitude and perpendicularity
Magnitude: \(|\mathbf{v}| = \sqrt{x^2 + y^2 + z^2}\). Perpendicular: \(\mathbf{a} \cdot \mathbf{b} = 0\) (dot product equals zero). At AS level, the dot product \(\mathbf{a} \cdot \mathbf{b} = a_1b_1 + a_2b_2 + a_3b_3\).

Common Pitfalls (from Examiner Reports)

  • Wrong direction: \(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\), NOT \(\mathbf{a} - \mathbf{b}\). Getting this backwards gives a sign error throughout.
  • Collinear vs parallel: Parallel vectors have the same direction but need not share a point. Collinear means the points lie on the same straight line — you need to show parallel vectors AND a shared point.
  • Dot product errors: \(\mathbf{a} \cdot \mathbf{b} = a_1b_1 + a_2b_2\) (multiply corresponding components and ADD). Not \(a_1b_2 + a_2b_1\).
  • Forgetting to state conclusions: If asked "show these are perpendicular", you must compute the dot product AND explicitly state "= 0, therefore perpendicular".

What Top Candidates Do

  • Draw a quick diagram labelling all points with their position vectors.
  • Write \(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\) as a formula before substituting numbers.
  • For "show collinear": find two displacement vectors sharing a common point, show one is a scalar multiple of the other, then state "since they share point X and are parallel, A, B, C are collinear."

AS Paper 1 — June 2019 — Question 13 (7 marks)

Relative to a fixed origin O, the position vectors of A, B, and C are:

\(\mathbf{a} = 2\mathbf{i} + 3\mathbf{j} - \mathbf{k}\), \(\mathbf{b} = 4\mathbf{i} + \mathbf{j} + 3\mathbf{k}\), \(\mathbf{c} = 7\mathbf{i} - 2\mathbf{j} + 9\mathbf{k}\)

(a) Find \(\overrightarrow{AB}\) and \(\overrightarrow{AC}\). [2 marks]

(b) Hence show that A, B, and C are collinear and state the ratio AB:BC. [3 marks]

(c) Find the exact length of AC. [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find \(\overrightarrow{AB}\) and \(\overrightarrow{AC}\) ▶
Use the fundamental formula: the vector from A to B is \(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\). Subtract corresponding components.

$$\overrightarrow{AB} = \mathbf{b} - \mathbf{a} = (4-2)\mathbf{i} + (1-3)\mathbf{j} + (3-(-1))\mathbf{k}$$

$$= 2\mathbf{i} - 2\mathbf{j} + 4\mathbf{k}$$

$$\overrightarrow{AC} = \mathbf{c} - \mathbf{a} = (7-2)\mathbf{i} + (-2-3)\mathbf{j} + (9-(-1))\mathbf{k}$$

$$= 5\mathbf{i} - 5\mathbf{j} + 10\mathbf{k}$$

A1 — \(\overrightarrow{AB} = 2\mathbf{i} - 2\mathbf{j} + 4\mathbf{k}\)
A1 — \(\overrightarrow{AC} = 5\mathbf{i} - 5\mathbf{j} + 10\mathbf{k}\)
Part (b) Step 2: Show A, B, C are collinear ▶
To prove collinearity, we need to show that \(\overrightarrow{AB}\) and \(\overrightarrow{AC}\) are parallel (one is a scalar multiple of the other). Since they share the point A, parallel implies collinear.

Compare \(\overrightarrow{AB} = 2\mathbf{i} - 2\mathbf{j} + 4\mathbf{k}\) with \(\overrightarrow{AC} = 5\mathbf{i} - 5\mathbf{j} + 10\mathbf{k}\):

Check each component ratio: \(\frac{5}{2} = \frac{-5}{-2} = \frac{10}{4} = 2.5\)

$$\overrightarrow{AC} = \frac{5}{2}\overrightarrow{AB}$$

Since \(\overrightarrow{AC} = \frac{5}{2}\overrightarrow{AB}\) and both vectors start from A, the points A, B, and C lie on the same straight line (are collinear).

M1 — Identifying the scalar relationship between the vectors
A1 — Correctly showing \(\overrightarrow{AC} = \frac{5}{2}\overrightarrow{AB}\)
Part (b) Step 3: Find the ratio AB:BC ▶
Since \(\overrightarrow{AC} = \frac{5}{2}\overrightarrow{AB}\), point B lies between A and C, and the distance AC is \(\frac{5}{2}\) times the distance AB. So \(AB = \frac{2}{5}\) of AC, meaning \(BC = AC - AB = \frac{3}{5}\) of AC. The ratio AB:BC = 2:3.

\(\overrightarrow{AC} = \frac{5}{2}\overrightarrow{AB}\)

This means AC = \(\frac{5}{2}\) AB, so AB = \(\frac{2}{5}\) AC

Therefore BC = AC - AB = \(\frac{5}{2}\)AB - AB = \(\frac{3}{2}\)AB

$$\text{AB:BC} = 1:\frac{3}{2} = 2:3$$

A1 — AB:BC = 2:3
Many candidates found AB:AC = 2:5 instead of AB:BC = 2:3. Read the question carefully: it asks for the ratio of the two PARTS (AB and BC), not AB to the whole (AC).
Part (c) Step 4: Find the exact length of AC ▶
The length (magnitude) of a vector \(\mathbf{v} = a\mathbf{i} + b\mathbf{j} + c\mathbf{k}\) is \(|\mathbf{v}| = \sqrt{a^2 + b^2 + c^2}\). Apply this to \(\overrightarrow{AC}\).

$$|\overrightarrow{AC}| = |5\mathbf{i} - 5\mathbf{j} + 10\mathbf{k}|$$

$$= \sqrt{5^2 + (-5)^2 + 10^2}$$

$$= \sqrt{25 + 25 + 100}$$

$$= \sqrt{150} = \sqrt{25 \times 6} = 5\sqrt{6}$$

M1 — Using the magnitude formula with their \(\overrightarrow{AC}\)
A1 — \(5\sqrt{6}\) (exact form required)
The question says "exact length" so you must leave the answer in surd form: \(5\sqrt{6}\), not the decimal 12.2. Always simplify the surd: \(\sqrt{150} = 5\sqrt{6}\).

AS Paper 1 — June 2022 — Question 11 (8 marks)

The points A, B, and C have position vectors relative to a fixed origin O:

\(\mathbf{a} = \begin{pmatrix} 1 \\ 3 \end{pmatrix}\), \(\mathbf{b} = \begin{pmatrix} 5 \\ 1 \end{pmatrix}\), \(\mathbf{c} = \begin{pmatrix} 7 \\ 5 \end{pmatrix}\)

(a) Find \(\overrightarrow{AB}\) and show that \(|\overrightarrow{AB}| = 2\sqrt{5}\). [2 marks]

(b) Show that \(\overrightarrow{AB}\) is perpendicular to \(\overrightarrow{BC}\). [3 marks]

(c) Find the position vector of the point M, the midpoint of AC. [1 mark]

(d) Hence find the position vector of the point D such that ABCD is a rectangle. [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find \(\overrightarrow{AB}\) and its magnitude ▶
Same formula as before: \(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\). Then find the magnitude using Pythagoras in 2D.

$$\overrightarrow{AB} = \mathbf{b} - \mathbf{a} = \begin{pmatrix} 5-1 \\ 1-3 \end{pmatrix} = \begin{pmatrix} 4 \\ -2 \end{pmatrix}$$

$$|\overrightarrow{AB}| = \sqrt{4^2 + (-2)^2} = \sqrt{16 + 4} = \sqrt{20} = 2\sqrt{5} \quad \square$$

M1 — Finding \(\overrightarrow{AB}\) correctly
A1 — Showing \(|\overrightarrow{AB}| = 2\sqrt{5}\)
Part (b) Step 2: Show \(\overrightarrow{AB}\) is perpendicular to \(\overrightarrow{BC}\) ▶
Two vectors are perpendicular if and only if their dot product equals zero. First find \(\overrightarrow{BC}\), then compute the dot product \(\overrightarrow{AB} \cdot \overrightarrow{BC}\).

First find \(\overrightarrow{BC}\):

$$\overrightarrow{BC} = \mathbf{c} - \mathbf{b} = \begin{pmatrix} 7-5 \\ 5-1 \end{pmatrix} = \begin{pmatrix} 2 \\ 4 \end{pmatrix}$$

Now compute the dot product:

$$\overrightarrow{AB} \cdot \overrightarrow{BC} = \begin{pmatrix} 4 \\ -2 \end{pmatrix} \cdot \begin{pmatrix} 2 \\ 4 \end{pmatrix} = (4)(2) + (-2)(4) = 8 - 8 = 0$$

Since \(\overrightarrow{AB} \cdot \overrightarrow{BC} = 0\), the vectors are perpendicular. \(\square\)

M1 — Finding \(\overrightarrow{BC}\)
M1 — Computing the dot product
A1 — Showing = 0 and stating "therefore perpendicular"
You must explicitly state "= 0, therefore perpendicular" (or equivalent). Simply writing "8 - 8 = 0" without a conclusion statement does not earn the final mark.
Part (c) Step 3: Find the midpoint M of AC ▶
The midpoint of two points is the average of their position vectors: \(\mathbf{m} = \frac{\mathbf{a} + \mathbf{c}}{2}\).

$$\mathbf{m} = \frac{\mathbf{a} + \mathbf{c}}{2} = \frac{1}{2}\begin{pmatrix} 1+7 \\ 3+5 \end{pmatrix} = \frac{1}{2}\begin{pmatrix} 8 \\ 8 \end{pmatrix} = \begin{pmatrix} 4 \\ 4 \end{pmatrix}$$

B1 — \(\mathbf{m} = \begin{pmatrix} 4 \\ 4 \end{pmatrix}\)
Part (d) Step 4: Find position vector of D for rectangle ABCD ▶
In a rectangle (or any parallelogram), the diagonals bisect each other. So the midpoint of AC must equal the midpoint of BD. Since M is the midpoint of AC, it's also the midpoint of BD. Using \(\mathbf{m} = \frac{\mathbf{b} + \mathbf{d}}{2}\), we can solve for \(\mathbf{d}\).

Midpoint of BD = M:

$$\frac{\mathbf{b} + \mathbf{d}}{2} = \begin{pmatrix} 4 \\ 4 \end{pmatrix}$$

$$\mathbf{d} = 2\begin{pmatrix} 4 \\ 4 \end{pmatrix} - \mathbf{b} = \begin{pmatrix} 8 \\ 8 \end{pmatrix} - \begin{pmatrix} 5 \\ 1 \end{pmatrix} = \begin{pmatrix} 3 \\ 7 \end{pmatrix}$$

$$\mathbf{d} = \begin{pmatrix} 3 \\ 7 \end{pmatrix} \quad \text{i.e. } D = (3, 7)$$

M1 — Using the property that diagonals bisect (or equivalent method)
A1 — \(\mathbf{d} = 3\mathbf{i} + 7\mathbf{j}\)
Alternative method: since ABCD is a rectangle, \(\overrightarrow{AD} = \overrightarrow{BC}\). So \(\mathbf{d} - \mathbf{a} = \overrightarrow{BC} = \begin{pmatrix} 2 \\ 4 \end{pmatrix}\), giving \(\mathbf{d} = \begin{pmatrix} 1 \\ 3 \end{pmatrix} + \begin{pmatrix} 2 \\ 4 \end{pmatrix} = \begin{pmatrix} 3 \\ 7 \end{pmatrix}\). Both methods are equally valid.

Key Formulae

Essential Vector Operations

Displacement vector: \(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\)

Magnitude: \(|\mathbf{v}| = \sqrt{v_1^2 + v_2^2 + v_3^2}\)

Midpoint: \(\mathbf{m} = \frac{\mathbf{a} + \mathbf{b}}{2}\)

Dot product: \(\mathbf{a} \cdot \mathbf{b} = a_1b_1 + a_2b_2 + a_3b_3\)

Key Conditions

Parallel: \(\mathbf{a} = k\mathbf{b}\) for some scalar \(k\)

Perpendicular: \(\mathbf{a} \cdot \mathbf{b} = 0\)

Collinear (A, B, C): \(\overrightarrow{AB} = k\overrightarrow{AC}\) (parallel AND share a point)

Unit vector: \(\hat{\mathbf{v}} = \frac{\mathbf{v}}{|\mathbf{v}|}\)

Parallelogram/Rectangle Properties

In parallelogram ABCD: \(\overrightarrow{AB} = \overrightarrow{DC}\) and \(\overrightarrow{AD} = \overrightarrow{BC}\)

Diagonals bisect each other: midpoint of AC = midpoint of BD
