Guided Approach

The key operations with functions at AS level:

1
Composite functions: \(fg(x)\) means "apply \(g\) first, then \(f\)"
Replace \(x\) in \(f\) with the entire expression \(g(x)\). Common mistake: doing it the wrong way round. \(fg(x) = f(g(x))\) is NOT the same as \(gf(x) = g(f(x))\).
2
Inverse functions: swap \(x\) and \(y\), then rearrange
Write \(y = f(x)\), swap \(x\) and \(y\) to get \(x = f(y)\), then rearrange to make \(y\) the subject. The domain of \(f^{-1}\) is the range of \(f\), and vice versa.
3
Domain and range
Domain = set of allowed inputs (what \(x\) can be). Range = set of possible outputs (what \(f(x)\) can be). Watch for: division by zero (exclude from domain), square roots of negatives, logarithms of non-positives.

Common Pitfalls (from Examiner Reports)

  • Order of composition: \(fg(x)\) means "do \(g\) first, then \(f\)". Many candidates compute \(gf(x)\) instead. Remember: read right to left.
  • Domain of inverse: The domain of \(f^{-1}\) equals the RANGE of \(f\). Candidates who just copy the original domain lose marks.
  • Not rearranging fully for inverse: After swapping \(x\) and \(y\), some candidates leave the answer with \(y\) still buried in a fraction or square root.
  • Forgetting domain restrictions: If \(f\) has domain \(x > 2\), this affects what inputs are valid for composites like \(fg\).

What Top Candidates Do

  • For composites: write \(fg(x) = f[\text{replace x with } g(x)]\) to make the substitution explicit.
  • For inverses: lay out "Let \(y = f(x)\). Swap: \(x = ...\). Rearrange: \(y = ...\). Therefore \(f^{-1}(x) = ...\)"
  • Always state domain and range using correct notation: e.g. "\(x \in \mathbb{R}, x \neq 3\)" or "\(f(x) > 0\)".

AS Paper 1 — June 2018 — Question 11 (7 marks)

The functions \(f\) and \(g\) are defined as:

\(f(x) = 3x + 2, \quad x \in \mathbb{R}\)

\(g(x) = x^2 - 4, \quad x \in \mathbb{R}\)

(a) Find \(fg(x)\). [2 marks]

(b) Solve \(fg(x) = g(x)\). [5 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find \(fg(x)\) — substitute \(g(x)\) into \(f\) ▶
\(fg(x)\) means \(f(g(x))\) — we apply \(g\) first (giving \(x^2 - 4\)), then feed that result into \(f\). So wherever \(f\) has an \(x\), we replace it with \(g(x) = x^2 - 4\).

$$fg(x) = f(g(x)) = f(x^2 - 4)$$

Replace \(x\) in \(f(x) = 3x + 2\) with \((x^2 - 4)\):

$$= 3(x^2 - 4) + 2$$

$$= 3x^2 - 12 + 2$$

$$= 3x^2 - 10$$

M1 — Substituting \(g(x)\) into \(f\) (correct order)
A1 — \(fg(x) = 3x^2 - 10\)
Part (b) Step 2: Set up and solve \(fg(x) = g(x)\) ▶
We need to solve \(3x^2 - 10 = x^2 - 4\). This is a quadratic equation — rearrange to standard form, then solve by factoring or using the quadratic formula.

$$3x^2 - 10 = x^2 - 4$$

Subtract \(x^2 - 4\) from both sides:

$$2x^2 - 6 = 0$$

Factor:

$$2(x^2 - 3) = 0$$

$$x^2 = 3$$

$$x = \pm\sqrt{3}$$

M1 — Setting \(fg(x) = g(x)\) and forming a quadratic
A1 — Correct simplified quadratic: \(2x^2 - 6 = 0\)
M1 — Solving the quadratic
A1 — \(x = \sqrt{3}\)
A1 — \(x = -\sqrt{3}\) (both roots needed)
Candidates who found \(fg(x)\) incorrectly in part (a) could still earn method marks here by correctly setting up and solving their equation. Both solutions must be given — writing only the positive root loses the final A1.

AS Paper 1 — June 2019 — Question 12 (8 marks)

The function \(f\) is defined as \(f(x) = \frac{2x + 1}{x - 3}, \quad x \in \mathbb{R}, x \neq 3\)

(a) Find \(f^{-1}(x)\) and state its domain. [4 marks]

(b) Find \(ff(x)\), giving your answer in its simplest form. [4 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find the inverse — swap and rearrange ▶
To find \(f^{-1}(x)\): write \(y = \frac{2x+1}{x-3}\), swap \(x\) and \(y\) to get \(x = \frac{2y+1}{y-3}\), then rearrange to make \(y\) the subject. The key technique for rational functions is to multiply both sides by the denominator, then collect all \(y\) terms on one side.

Let \(y = \frac{2x + 1}{x - 3}\). Swap \(x\) and \(y\):

$$x = \frac{2y + 1}{y - 3}$$

Multiply both sides by \((y - 3)\):

$$x(y - 3) = 2y + 1$$

Expand:

$$xy - 3x = 2y + 1$$

Collect \(y\) terms on one side:

$$xy - 2y = 3x + 1$$

Factor out \(y\):

$$y(x - 2) = 3x + 1$$

Divide both sides by \((x - 2)\):

$$y = \frac{3x + 1}{x - 2}$$

$$f^{-1}(x) = \frac{3x + 1}{x - 2}$$

M1 — Attempting to swap and rearrange (or equivalent method)
A1 — Correct algebra reaching \(\frac{3x+1}{x-2}\)
Part (a) Step 2: State the domain of \(f^{-1}\) ▶
The domain of \(f^{-1}\) equals the range of \(f\). For \(f(x) = \frac{2x+1}{x-3}\), the horizontal asymptote is \(y = 2\) (leading coefficients: \(\frac{2}{1}\)), so the range of \(f\) is \(y \neq 2\). Therefore the domain of \(f^{-1}\) is \(x \neq 2\).

Domain of \(f^{-1}\): \(x \in \mathbb{R}, x \neq 2\)

(This also makes sense from the formula since \(f^{-1}(x) = \frac{3x+1}{x-2}\) is undefined when \(x = 2\).)

B1 — Domain: \(x \neq 2\) (or equivalent notation)
A1 — Full answer with correct working
Some candidates wrote \(x \neq 3\) (copying the original domain) instead of \(x \neq 2\). Remember: the domain of the inverse is the range of the original function.
Part (b) Step 3: Find \(ff(x)\) — substitute \(f(x)\) into itself ▶
\(ff(x) = f(f(x))\). We replace every \(x\) in \(f\) with \(f(x) = \frac{2x+1}{x-3}\). This creates a compound fraction that we'll need to simplify carefully.

$$ff(x) = f\left(\frac{2x+1}{x-3}\right) = \frac{2\left(\frac{2x+1}{x-3}\right) + 1}{\left(\frac{2x+1}{x-3}\right) - 3}$$

Simplify numerator: \(2 \cdot \frac{2x+1}{x-3} + 1 = \frac{2(2x+1)}{x-3} + \frac{x-3}{x-3} = \frac{4x+2+x-3}{x-3} = \frac{5x-1}{x-3}\)

Simplify denominator: \(\frac{2x+1}{x-3} - 3 = \frac{2x+1}{x-3} - \frac{3(x-3)}{x-3} = \frac{2x+1-3x+9}{x-3} = \frac{-x+10}{x-3}\)

Divide the fractions:

$$ff(x) = \frac{\frac{5x-1}{x-3}}{\frac{-x+10}{x-3}} = \frac{5x-1}{-x+10} = \frac{5x-1}{10-x}$$

$$ff(x) = \frac{5x - 1}{10 - x}$$

M1 — Substituting \(f(x)\) into \(f\) (correct structure)
M1 — Multiplying out compound fractions
A1 — Correct simplified numerator: \(5x - 1\)
A1 — Correct final answer: \(\frac{5x-1}{10-x}\)
The compound fraction algebra is where most marks are lost. The common error is forgetting to find a common denominator when adding/subtracting fractions. Writing the numerator and denominator simplification separately (as shown) helps avoid errors.

AS Paper 1 — June 2022 — Question 9 (9 marks)

The function \(f\) is defined as \(f(x) = (x - 3)^2 + 1, \quad x \geq 3\)

(a) State the range of \(f\). [1 mark]

(b) Find \(f^{-1}(x)\) and state its domain. [4 marks]

(c) Explain why the inverse of \(f\) would not exist without the domain restriction \(x \geq 3\). [1 mark]

View Question Paper View Mark Scheme
Part (a) Step 1: Find the range of \(f\) ▶
Since \(x \geq 3\), the expression \((x-3)^2\) achieves its minimum value of 0 when \(x = 3\), so \(f(x) = (x-3)^2 + 1 \geq 0 + 1 = 1\). As \(x\) increases beyond 3, \(f(x)\) increases without bound.

Range of \(f\): \(f(x) \geq 1\) (or equivalently \(y \geq 1\))

B1 — \(f(x) \geq 1\)
Part (b) Step 2: Find the inverse ▶
Swap \(x\) and \(y\), then rearrange. Since we have a squared term, we'll take a square root — and because the domain restricts \(x \geq 3\) (so \(x - 3 \geq 0\)), we take the POSITIVE root only.

Let \(y = (x-3)^2 + 1\). Swap:

$$x = (y-3)^2 + 1$$

Subtract 1:

$$x - 1 = (y-3)^2$$

Take square root (positive only, since original domain gives \(y - 3 \geq 0\)):

$$\sqrt{x-1} = y - 3$$

Add 3:

$$y = 3 + \sqrt{x-1}$$

$$f^{-1}(x) = 3 + \sqrt{x - 1}$$

Domain of \(f^{-1}\): \(x \geq 1\) (this is the range of \(f\))

M1 — Swapping and attempting to rearrange
A1 — Taking positive root (justified by domain)
A1 — \(f^{-1}(x) = 3 + \sqrt{x-1}\)
B1 — Domain: \(x \geq 1\)
Part (c) Step 3: Explain why domain restriction is needed ▶
A function only has an inverse if it is one-to-one (each output comes from exactly one input). Without the restriction \(x \geq 3\), the quadratic \((x-3)^2 + 1\) is NOT one-to-one because it is symmetric about \(x = 3\) — both \(x = 2\) and \(x = 4\) give the same output.

Without the domain restriction, \(f\) would not be one-to-one (it would be a many-to-one function). For example, both \(x = 2\) and \(x = 4\) give \(f(x) = 2\). A function must be one-to-one to have an inverse.

B1 — Explanation that \(f\) would not be one-to-one without the restriction (or equivalent reasoning)
Simply saying "it's a quadratic so it doesn't have an inverse" is too vague. You need to explain WHY: "the function is not one-to-one" or "two different x values map to the same y value" or "it fails the horizontal line test".

Key Concepts

Composition Rules

\(fg(x) = f(g(x))\) — apply \(g\) first, then \(f\)

\(gf(x) = g(f(x))\) — apply \(f\) first, then \(g\)

\(ff(x) = f(f(x))\) — apply \(f\) twice

Inverse Function Properties

Domain of \(f^{-1}\) = Range of \(f\)

Range of \(f^{-1}\) = Domain of \(f\)

\(f(f^{-1}(x)) = x\) and \(f^{-1}(f(x)) = x\)

Graph of \(f^{-1}\) is a reflection of graph of \(f\) in the line \(y = x\)

Graph Transformations

\(f(x) + a\): translate UP by \(a\)

\(f(x + a)\): translate LEFT by \(a\)

\(af(x)\): stretch vertically by factor \(a\)

\(f(ax)\): stretch horizontally by factor \(\frac{1}{a}\)
