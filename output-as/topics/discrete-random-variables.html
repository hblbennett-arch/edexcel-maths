Key Formulae

Sum of probabilities:

$$\sum_{\text{all } x} P(X = x) = 1$$

Expected Value (Mean):

$$E(X) = \mu = \sum x \cdot P(X = x)$$

Variance:

$$\text{Var}(X) = E(X^2) - [E(X)]^2$$

where \(E(X^2) = \sum x^2 \cdot P(X = x)\)

Standard Deviation:

$$\sigma = \sqrt{\text{Var}(X)}$$

Key Concepts

What is a Discrete Random Variable?

A random variable \(X\) that can only take specific (countable) values, each with an associated probability. The probability distribution lists all possible values and their probabilities.

Validity check: A probability distribution is valid if and only if:
(1) All probabilities are between 0 and 1: \(0 \leq P(X = x) \leq 1\)
(2) The sum of all probabilities equals 1: \(\sum P(X = x) = 1\)

Finding Unknown Probabilities

Use the fact that \(\sum P(X = x) = 1\) to find unknown probabilities in a distribution table.

Common question type: "The random variable \(X\) has distribution given by..." with one probability expressed in terms of a constant \(k\). Use \(\sum P = 1\) to form an equation and solve for \(k\).

E(X) and Var(X) — Calculation Method

Always set up a systematic table:

\(x\)\(P(X=x)\)\(x \cdot P(X=x)\)\(x^2 \cdot P(X=x)\)
............
Totals:\(E(X)\)\(E(X^2)\)

Then: \(\text{Var}(X) = E(X^2) - [E(X)]^2\)

Linear Functions of X

If \(Y = aX + b\), then:
$$E(Y) = aE(X) + b$$ $$\text{Var}(Y) = a^2 \text{Var}(X)$$ Note: The constant \(b\) does NOT affect the variance.

Worked Examples

Example 1: Finding a Distribution from Symmetry June 2018 Q5

A fair spinner can take values 1, 2, 3 or 4. The random variable \(X\) represents the score. Given that \(P(X = 4) = P(X = 2) = 0.35\) and the distribution is symmetric about \(x = 2.5\):
(a) Write down the complete probability distribution of \(X\). [2]
(b) The spinner is spun 60 times. Find the probability of landing on 4 more than half the time. [3]
(c) Given \(Y = \frac{12}{X}\), find \(P(Y \geq X)\). [3]
Step 1: Use symmetry to find all probabilities
Symmetric about 2.5 means: \(P(X=1) = P(X=4)\) and \(P(X=2) = P(X=3)\).
Given \(P(X=4) = P(X=2) = 0.35\):
- \(P(X=4) = 0.35\), so \(P(X=1) = 0.35\)... but wait, that gives total \(> 1\).

Actually: \(P(X=2) = 0.35\) is given, and \(P(X=4) = P(X=2)\), so \(P(X=4) = 0.35\).
By symmetry: \(P(X=1) = P(X=4)\) ... no. Rereading: the symmetry is that values equidistant from 2.5 have equal probabilities.
So \(P(X=1) = P(X=4)\) and \(P(X=2) = P(X=3)\).

Given \(P(X=2) = 0.35\) and \(P(X=4) = P(X=2) = 0.35\), this means \(P(X=4) = 0.35\).
Then \(P(X=1) = P(X=4) = 0.35\)... Total = \(0.35 + 0.35 + 0.35 + P(X=3) = 1\), giving \(P(X=3) = -0.05\).

Let me re-read. The actual question states: \(P(X=4) = P(X=2)\) (from the symmetry) and uses \(\sum P = 1\):
\(P(X=1) + 0.35 + P(X=3) + 0.35 = 1\)
Since \(P(X=1) = P(X=4) = 0.35\)... That's not right. Let me use the mark scheme:

From the mark scheme: \(P(X=4) = P(X=2) = 0.35\) and \(P(X=1) = P(X=3)\).
\(\sum P = 1\): \(P(X=1) + 0.35 + P(X=3) + 0.35 = 1\)
\(2 \times P(X=1) = 0.30\), so \(P(X=1) = P(X=3) = 0.15\).
Step 2: Write the distribution table
\(x\)1234
\(P(X=x)\)0.150.350.150.35
Check: \(0.15 + 0.35 + 0.15 + 0.35 = 1\) ✓
Step 3: P(more than 30 out of 60 land on 4)
Let \(A\) = number of times spinner lands on 4 in 60 spins.
\(A \sim B(60, 0.35)\)
"More than half" means \(A > 30\):
\(P(A > 30) = 1 - P(A \leq 30) = 1 - 0.99411... = 0.00589\)
\(\approx 0.00589\) (awrt 3 s.f.)
Step 4: Find P(Y >= X) where Y = 12/X
We need \(P\left(\frac{12}{X} \geq X\right)\), i.e. \(P(12 \geq X^2)\), i.e. \(P(X^2 \leq 12)\).
Since \(X\) takes values 1, 2, 3, 4:
- \(X=1\): \(1^2 = 1 \leq 12\) ✓
- \(X=2\): \(2^2 = 4 \leq 12\) ✓
- \(X=3\): \(3^2 = 9 \leq 12\) ✓
- \(X=4\): \(4^2 = 16 > 12\) ✗

So \(P(Y \geq X) = P(X=1) + P(X=2) + P(X=3) = 0.15 + 0.35 + 0.15 = 0.65\)

Actually using the mark scheme approach: \(Y - X \geq 0\) means \(\frac{12}{X} - X \geq 0\), i.e. \(\frac{12 - X^2}{X} \geq 0\). Since \(X > 0\), need \(12 - X^2 \geq 0\), i.e. \(X \leq \sqrt{12} \approx 3.46\).
So \(X \in \{1, 2, 3\}\): \(P = 0.15 + 0.35 + 0.15 = \mathbf{0.65}\)

(a) \(P(X=1) = 0.15\), \(P(X=2) = 0.35\), \(P(X=3) = 0.15\), \(P(X=4) = 0.35\)

(b) \(P(A > 30) \approx 0.00589\)

(c) \(P(Y \geq X) = 0.65\)

Example 2: Two Spinners and Forming X = mR + nG June 2022 Q5

Manon has two biased spinners.
Red spinner \(R\): \(P(R=2) = \frac{1}{4}\), \(P(R=3) = \frac{3}{4}\)
Green spinner \(G\): \(P(G=1) = \frac{2}{3}\), \(P(G=4) = \frac{1}{3}\)

Manon spins each spinner once and adds the scores.
(a)(i) Find the probability that the sum is 7. [1]
(a)(ii) Find the probability that the sum is less than 4. [2]

The random variable \(X = mR + nG\) where \(m\) and \(n\) are integers.
\(P(X = 20) = \frac{1}{6}\) and \(P(X = 50) = \frac{1}{4}\)
(b) Find \(m\) and \(n\). [5]
Step 1: List all possible outcomes for R + G
\(G=1\) (prob \(\frac{2}{3}\))\(G=4\) (prob \(\frac{1}{3}\))
\(R=2\) (prob \(\frac{1}{4}\))Sum = 3, prob = \(\frac{1}{4}\times\frac{2}{3} = \frac{2}{12} = \frac{1}{6}\)Sum = 6, prob = \(\frac{1}{4}\times\frac{1}{3} = \frac{1}{12}\)
\(R=3\) (prob \(\frac{3}{4}\))Sum = 4, prob = \(\frac{3}{4}\times\frac{2}{3} = \frac{6}{12} = \frac{1}{2}\)Sum = 7, prob = \(\frac{3}{4}\times\frac{1}{3} = \frac{3}{12} = \frac{1}{4}\)
Step 2: Find P(sum = 7) and P(sum < 4)
(i) Sum = 7 only when \(R=3\) and \(G=4\):
\(P(\text{sum} = 7) = \frac{3}{4} \times \frac{1}{3} = \frac{1}{4}\)

(ii) Sum < 4 means sum = 3 (only when \(R=2\) and \(G=1\)):
\(P(\text{sum} < 4) = \frac{1}{4} \times \frac{2}{3} = \frac{1}{6}\)
Step 3: Find possible values of X = mR + nG
The four possible values of \(X = mR + nG\) are:
- \(R=2, G=1\): \(X = 2m + n\) with prob \(\frac{1}{6}\)
- \(R=2, G=4\): \(X = 2m + 4n\) with prob \(\frac{1}{12}\)
- \(R=3, G=1\): \(X = 3m + n\) with prob \(\frac{1}{2}\)
- \(R=3, G=4\): \(X = 3m + 4n\) with prob \(\frac{1}{4}\)
Step 4: Match probabilities to find m and n
We're told \(P(X=20) = \frac{1}{6}\) and \(P(X=50) = \frac{1}{4}\).

From our table:
- Prob \(\frac{1}{6}\) corresponds to \(2m + n\), so \(2m + n = 20\) ... (1)
- Prob \(\frac{1}{4}\) corresponds to \(3m + 4n\), so \(3m + 4n = 50\) ... (2)

From (1): \(n = 20 - 2m\)
Substituting into (2): \(3m + 4(20-2m) = 50\)
\(3m + 80 - 8m = 50\)
\(-5m = -30\)
\(m = 6\)
\(n = 20 - 12 = 8\)

\(m = 6\), \(n = 8\)
Step 5: Verify
Check all values of \(X\) with \(m=6, n=8\):
- \(R=2, G=1\): \(X = 12+8 = 20\) ✓ (prob \(\frac{1}{6}\) ✓)
- \(R=2, G=4\): \(X = 12+32 = 44\)
- \(R=3, G=1\): \(X = 18+8 = 26\)
- \(R=3, G=4\): \(X = 18+32 = 50\) ✓ (prob \(\frac{1}{4}\) ✓)

(a)(i) \(P(\text{sum} = 7) = \frac{1}{4}\)

(a)(ii) \(P(\text{sum} < 4) = \frac{1}{6}\)

(b) \(m = 6\), \(n = 8\)

Example 3: E(X) and Var(X) Calculation Style: June 2023/2024

The random variable \(X\) has the following probability distribution:
\(x\)12345
\(P(X=x)\)0.1\(k\)0.3\(k\)0.1
(a) Find the value of \(k\). [2]
(b) Find \(E(X)\) and \(Var(X)\). [5]
(c) Find \(E(3X - 2)\) and \(Var(3X - 2)\). [2]
Step 1: Find k using sum of probabilities = 1
\(\sum P(X=x) = 1\):
\(0.1 + k + 0.3 + k + 0.1 = 1\)
\(0.5 + 2k = 1\)
\(2k = 0.5\)
\(\boxed{k = 0.25}\)
Step 2: Calculate E(X)
\(E(X) = \sum x \cdot P(X=x)\)
\(= 1(0.1) + 2(0.25) + 3(0.3) + 4(0.25) + 5(0.1)\)
\(= 0.1 + 0.5 + 0.9 + 1.0 + 0.5\)
\(\boxed{E(X) = 3}\)
Step 3: Calculate E(X^2)
\(E(X^2) = \sum x^2 \cdot P(X=x)\)
\(= 1^2(0.1) + 2^2(0.25) + 3^2(0.3) + 4^2(0.25) + 5^2(0.1)\)
\(= 0.1 + 1.0 + 2.7 + 4.0 + 2.5\)
\(= 10.3\)
Step 4: Calculate Var(X)
\(\text{Var}(X) = E(X^2) - [E(X)]^2\)
\(= 10.3 - 3^2\)
\(= 10.3 - 9\)
\(\boxed{\text{Var}(X) = 1.3}\)
Step 5: Find E(3X-2) and Var(3X-2)
Using the linear transformation rules:
\(E(3X - 2) = 3E(X) - 2 = 3(3) - 2 = \boxed{7}\)

\(\text{Var}(3X - 2) = 3^2 \cdot \text{Var}(X) = 9 \times 1.3 = \boxed{11.7}\)

Remember: adding/subtracting a constant does not change the variance!

(a) \(k = 0.25\)

(b) \(E(X) = 3\), \(\text{Var}(X) = 1.3\)

(c) \(E(3X-2) = 7\), \(\text{Var}(3X-2) = 11.7\)

Example 4: Forming a Distribution from Context Style: June 2019/2022

A bag contains 3 red and 2 blue balls. Two balls are drawn at random without replacement. The random variable \(X\) represents the number of red balls drawn.
(a) Find the probability distribution of \(X\). [4]
(b) Find \(E(X)\). [2]
(c) Find \(\text{Var}(X)\). [2]
Step 1: Identify possible values of X
Drawing 2 balls from 3 red + 2 blue (without replacement).
\(X\) = number of red balls drawn. Possible values: \(X = 0, 1, 2\).
Step 2: Calculate each probability
Total ways to choose 2 from 5: \(\binom{5}{2} = 10\)

\(P(X=0)\): Both blue = \(\frac{\binom{2}{2}}{\binom{5}{2}} = \frac{1}{10}\)

\(P(X=1)\): One red, one blue = \(\frac{\binom{3}{1}\binom{2}{1}}{\binom{5}{2}} = \frac{6}{10} = \frac{3}{5}\)

\(P(X=2)\): Both red = \(\frac{\binom{3}{2}}{\binom{5}{2}} = \frac{3}{10}\)

Check: \(\frac{1}{10} + \frac{6}{10} + \frac{3}{10} = \frac{10}{10} = 1\) ✓
Step 3: Write the distribution table
\(x\)012
\(P(X=x)\)\(\frac{1}{10}\)\(\frac{6}{10}\)\(\frac{3}{10}\)
Step 4: Calculate E(X)
\(E(X) = 0\left(\frac{1}{10}\right) + 1\left(\frac{6}{10}\right) + 2\left(\frac{3}{10}\right)\)
\(= 0 + \frac{6}{10} + \frac{6}{10} = \frac{12}{10} = \boxed{1.2}\)
Step 5: Calculate Var(X)
\(E(X^2) = 0^2\left(\frac{1}{10}\right) + 1^2\left(\frac{6}{10}\right) + 2^2\left(\frac{3}{10}\right)\)
\(= 0 + \frac{6}{10} + \frac{12}{10} = \frac{18}{10} = 1.8\)

\(\text{Var}(X) = E(X^2) - [E(X)]^2 = 1.8 - 1.2^2 = 1.8 - 1.44 = \boxed{0.36}\)

(a)

\(x\)012
\(P(X=x)\)0.10.60.3

(b) \(E(X) = 1.2\)

(c) \(\text{Var}(X) = 0.36\)

Exam Tips

1. Always check your distribution sums to 1. This is the most common error and the easiest mark to lose.
2. For Var(X), always use the formula \(E(X^2) - [E(X)]^2\). Calculate \(E(X^2)\) as a separate column in your table. Never try to do it in your head.
3. Common mistake: \(\text{Var}(X) \neq E(X^2) - E(X)\). You must square the mean: \([E(X)]^2\).
4. For "forming a distribution" questions: Systematically list all outcomes using a sample space diagram or tree diagram, then group by the value of \(X\).
5. Linear transformations: Remember that \(\text{Var}(aX+b) = a^2\text{Var}(X)\). The \(+b\) has NO effect on variance. This is because shifting data doesn't change its spread.
