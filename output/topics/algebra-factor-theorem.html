Guided Approach: Factor Theorem

The factor theorem states: if \(f(a) = 0\), then \((x - a)\) is a factor of \(f(x)\). The standard approach:

1
Use the factor theorem condition
If told that \((x - a)\) is a factor, then \(f(a) = 0\). Substitute \(x = a\) into \(f(x)\) and set the result equal to zero. This gives an equation you can solve for unknown constants.
2
Perform algebraic division (if needed)
Once you know the factor, divide \(f(x)\) by \((x - a)\) using polynomial long division or by comparing coefficients. This gives you the remaining quadratic factor.
3
Factorise the quadratic (if needed)
Factor or use the quadratic formula on the remaining quadratic to find all roots. Check: can the quadratic be factored, or does the discriminant tell you about real solutions?
4
Solve any inequality or state conclusions
If asked for an inequality, factorise fully, sketch a sign diagram or number line, and state the solution set with correct inequality signs. If asked to prove something about solutions, use the discriminant \(b^2 - 4ac\).

Common Pitfalls (from Examiner Reports)

  • Arithmetic errors in substitution: When substituting \(x = -3\) into \(3x^3 + 2ax^2 - 4x + 5a\), students frequently get \((-3)^3 = -27\) wrong or lose a negative sign. Write each term separately.
  • Wrong value substituted: If \((x + 3)\) is a factor, substitute \(x = -3\), NOT \(x = 3\). The factor \((x - a)\) means substitute \(x = a\), so \((x + 3) = (x - (-3))\) means \(x = -3\).
  • Division errors in algebraic long division: The most common mistake is subtracting incorrectly (forgetting to change signs). Write out each subtraction step fully rather than doing it mentally.
  • Wrong direction for inequality: After finding critical values (e.g., \(x = -2\) and \(x = 5\)), students write \(x < -2\) or \(x > 5\) when it should be \(-2 < x < 5\) (or vice versa). Always test a value in each region.
  • Forgetting to check factors systematically: When told "factorise completely", try \(x = \pm 1, \pm 2, \pm 3, ...\) systematically. Students waste time guessing randomly.

What Top Candidates Do

  • Write \(f(a) = 0\) explicitly as a clear first line — this earns the method mark even if arithmetic subsequently fails.
  • For algebraic division, use a clear column layout with every term written out (including \(0x\) placeholder terms when there are gaps in degree).
  • When solving quadratic inequalities, always sketch the parabola and mark the roots — this avoids the "wrong direction" error.
  • State the discriminant result and its meaning: "\(b^2 - 4ac = -7 < 0\), so no real roots, hence no further solutions."

Paper 1 — June 2019 — Question 1 (3 marks)

$$f(x) = 3x^3 + 2ax^2 - 4x + 5a$$

Given that \((x + 3)\) is a factor of \(f(x)\), find the value of the constant \(a\).

View Question Paper View Mark Scheme
Full solution Step 1: Apply the factor theorem ▶
The factor theorem says: if \((x + 3)\) is a factor of \(f(x)\), then \(f(-3) = 0\). Note carefully: \((x + 3) = (x - (-3))\), so we substitute \(x = -3\). This is the single most important step — state it clearly.

Since \((x + 3)\) is a factor of \(f(x)\), by the factor theorem:

$$f(-3) = 0$$

M1 — Stating or using f(-3) = 0 (factor theorem applied correctly)
Full solution Step 2: Substitute \(x = -3\) and evaluate each term ▶
We substitute \(x = -3\) into every term of \(f(x) = 3x^3 + 2ax^2 - 4x + 5a\). The key is careful arithmetic with negatives and powers. Writing each term separately reduces errors dramatically.

$$f(-3) = 3(-3)^3 + 2a(-3)^2 - 4(-3) + 5a$$

Calculate each term separately:

\(3(-3)^3 = 3 \times (-27) = -81\)

\(2a(-3)^2 = 2a \times 9 = 18a\)

\(-4(-3) = 12\)

\(5a = 5a\)

$$f(-3) = -81 + 18a + 12 + 5a$$

$$f(-3) = 23a - 69$$

M1 — Correct substitution of x = -3 into f(x) leading to a linear equation in a
Full solution Step 3: Set equal to zero and solve for \(a\) ▶
We set our expression equal to zero (the factor theorem condition) and solve the resulting linear equation for \(a\).

$$23a - 69 = 0$$

Add 69 to both sides:

$$23a = 69$$

Divide both sides by 23:

$$\boxed{a = 3}$$

A1 — Correct value: \(a = 3\)
This was well-answered overall, but a significant minority substituted \(x = 3\) instead of \(x = -3\). Remember: \((x + 3)\) as a factor means the root is at \(x = -3\).

Paper 1 — June 2022 — Question 2 (3 marks)

$$f(x) = (x - 4)(x^2 - 3x + k) - 42$$

where \(k\) is a constant.

Given that \((x + 2)\) is a factor of \(f(x)\), find the value of \(k\).

View Question Paper View Mark Scheme
Full solution Step 1: Apply the factor theorem — identify what to substitute ▶
Since \((x + 2)\) is a factor of \(f(x)\), the factor theorem tells us \(f(-2) = 0\). This question is slightly different because \(f(x)\) is given in a partially factored form. We don't need to expand it first — we can just substitute \(x = -2\) directly into the given expression.

By the factor theorem: \((x + 2)\) is a factor \(\implies f(-2) = 0\)

M1 — Using f(-2) = 0
Full solution Step 2: Substitute \(x = -2\) into the given expression ▶
The beauty of this question is that we can substitute directly without expanding. We evaluate each bracket at \(x = -2\): \((x-4)\) becomes \((-6)\), and \((x^2-3x+k)\) becomes \((4+6+k) = (10+k)\).

$$f(-2) = (-2 - 4)((-2)^2 - 3(-2) + k) - 42$$

Evaluate each bracket:

\((-2 - 4) = -6\)

\((-2)^2 - 3(-2) + k = 4 + 6 + k = 10 + k\)

$$f(-2) = (-6)(10 + k) - 42$$

$$f(-2) = -60 - 6k - 42$$

$$f(-2) = -102 - 6k$$

A1 — Correct simplified expression in terms of k
Full solution Step 3: Set equal to zero and solve ▶

$$-102 - 6k = 0$$

Add 102 to both sides:

$$-6k = 102$$

Divide by -6:

$$\boxed{k = -17}$$

A1 — Correct value: \(k = -17\)
A common error was expanding \((x-4)(x^2-3x+k)\) fully before substituting — this wastes time and introduces more opportunities for arithmetic errors. Substituting first is much cleaner.

Paper 1 — June 2023 — Question 2 (6 marks)

In this question you must show all stages of your working.

$$f(x) = 4x^3 + 5x^2 - 10x + 4a, \quad x \in \mathbb{R}$$

where \(a\) is a positive constant.

Given \((x - a)\) is a factor of \(f(x)\),

(a) show that \(a(4a^2 + 5a - 6) = 0\) [2 marks]

(b) Hence (i) find the value of \(a\) (ii) use algebra to find the exact solutions of \(f(x) = 3\) [4 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Apply the factor theorem with \(x = a\) ▶
Since \((x - a)\) is a factor, by the factor theorem \(f(a) = 0\). Here the root IS the same letter as the unknown constant — this is what makes the question slightly harder. We substitute \(x = a\) into the cubic.

By the factor theorem, \((x - a)\) is a factor \(\implies f(a) = 0\):

$$f(a) = 4a^3 + 5a^2 - 10a + 4a = 0$$

Combine the \(a\) terms: \(-10a + 4a = -6a\):

$$4a^3 + 5a^2 - 6a = 0$$

Factor out \(a\):

$$\boxed{a(4a^2 + 5a - 6) = 0} \quad \square$$

M1 — Substituting \(x = a\) and setting f(a) = 0
A1 — Reaching the given expression with correct working shown
Part (b)(i) Step 2: Find the value of \(a\) ▶
From \(a(4a^2 + 5a - 6) = 0\), either \(a = 0\) or \(4a^2 + 5a - 6 = 0\). But we're told \(a\) is a positive constant, so \(a \neq 0\). We solve the quadratic \(4a^2 + 5a - 6 = 0\) and pick the positive root.

Since \(a\) is a positive constant, \(a \neq 0\). So we need:

$$4a^2 + 5a - 6 = 0$$

Factorise: look for factors of \(4 \times (-6) = -24\) that add to 5. These are 8 and \(-3\):

$$4a^2 + 8a - 3a - 6 = 0$$

$$4a(a + 2) - 3(a + 2) = 0$$

$$(4a - 3)(a + 2) = 0$$

So \(a = \frac{3}{4}\) or \(a = -2\)

Since \(a\) is positive: \(\boxed{a = \frac{3}{4}}\)

A1 — \(a = \frac{3}{4}\) (with valid reasoning for rejecting other solutions)
Part (b)(ii) Step 3: Write \(f(x)\) using the known factor and divide ▶
Now we know \(a = \frac{3}{4}\), so the factor is \((x - \frac{3}{4})\) and \(f(x) = 4x^3 + 5x^2 - 10x + 3\). To solve \(f(x) = 3\), we first write \(f(x) - 3 = 0\), then factorise using our known factor. We can also note that \((x - \frac{3}{4})\) being a factor of \(f(x)\) means \((4x - 3)\) is a factor of \(4f(x)\) — or equivalently, we divide \(f(x)\) by \((x - \frac{3}{4})\).

With \(a = \frac{3}{4}\): \(f(x) = 4x^3 + 5x^2 - 10x + 3\)

We need to solve \(f(x) = 3\):

$$4x^3 + 5x^2 - 10x + 3 = 3$$

$$4x^3 + 5x^2 - 10x = 0$$

Factor out \(x\):

$$x(4x^2 + 5x - 10) = 0$$

M1 — Setting f(x) = 3, simplifying, and factoring out x (or equivalent approach)
Part (b)(ii) Step 4: Solve the remaining quadratic ▶
One solution is \(x = 0\). For the other solutions, we solve \(4x^2 + 5x - 10 = 0\) using the quadratic formula (it doesn't factorise neatly). "Exact solutions" means we leave surds in our answer.

From \(x(4x^2 + 5x - 10) = 0\):

Either \(x = 0\), or \(4x^2 + 5x - 10 = 0\)

Apply the quadratic formula with \(a=4\), \(b=5\), \(c=-10\):

$$x = \frac{-5 \pm \sqrt{25 + 160}}{8} = \frac{-5 \pm \sqrt{185}}{8}$$

The three exact solutions are:

$$\boxed{x = 0, \quad x = \frac{-5 + \sqrt{185}}{8}, \quad x = \frac{-5 - \sqrt{185}}{8}}$$

M1 — Using the quadratic formula (or completing the square) on the remaining quadratic
A1 — All three correct exact solutions stated
Many candidates lost marks by not giving all three solutions. The \(x = 0\) solution was frequently omitted. "Exact solutions" means surds are required — decimal approximations score 0.

Paper 2 — June 2018 — Question 6 (5 marks)

$$f(x) = -3x^3 + 8x^2 - 9x + 10, \quad x \in \mathbb{R}$$

(a) (i) Calculate \(f(2)\)
(ii) Write \(f(x)\) as a product of two algebraic factors. [3 marks]

Using the answer to (a)(ii),

(b) prove that there are exactly two real solutions to the equation \(-3y^6 + 8y^4 - 9y^2 + 10 = 0\) [2 marks]

View Question Paper View Mark Scheme
Part (a)(i) Step 1: Calculate \(f(2)\) ▶
When part (i) asks you to calculate \(f(2)\) before part (ii) asks you to factorise, this is a strong hint. If \(f(2) = 0\), then \((x - 2)\) is a factor by the factor theorem. This is the examiner guiding you toward the factorisation.

$$f(2) = -3(2)^3 + 8(2)^2 - 9(2) + 10$$

\(= -3(8) + 8(4) - 18 + 10\)

\(= -24 + 32 - 18 + 10\)

$$f(2) = 0$$

B1 — \(f(2) = 0\) (shown with working)
Part (a)(ii) Step 2: Use algebraic division to factorise \(f(x)\) ▶
Since \(f(2) = 0\), we know \((x - 2)\) is a factor. Now we divide \(f(x)\) by \((x - 2)\) to find the other factor (a quadratic). We can use polynomial long division or compare coefficients. Let's use the comparing coefficients method since it's often quicker for cubics.

Since \(f(2) = 0\), \((x - 2)\) is a factor. Write \(f(x) = (x - 2)(ax^2 + bx + c)\):

Comparing the leading coefficient: \(-3x^3\) must come from \(x \cdot ax^2\), so \(a = -3\)

Comparing the constant term: \(+10\) must come from \((-2)(c)\), so \(c = -5\)

Comparing the \(x^2\) coefficient: \(8 = -3 \times (-2) + b \times 1 = 6 + b\), so \(b = 2\)

Check \(x\) coefficient: \(-9 = (-3)(-2) + (2)(1) \cdot ... \) — let's verify:

\((x-2)(-3x^2 + 2x - 5)\): the \(x\) coefficient is \(2 \times (-2) + (-5) \times 1 = -4 - 5 = -9\) ✔

$$\boxed{f(x) = (x - 2)(-3x^2 + 2x - 5)}$$

M1 — Dividing f(x) by (x - 2) using any valid method
A1 — Correct factorisation: \((x-2)(-3x^2+2x-5)\)
Part (b) Step 3: Connect to the equation in \(y\) ▶
The equation \(-3y^6 + 8y^4 - 9y^2 + 10 = 0\) looks like our cubic but with \(y^2\) in place of \(x\). If we let \(x = y^2\), then \(f(x) = 0\) becomes the given equation. From our factorisation, \(f(x) = 0\) when \(x = 2\) or when \(-3x^2+2x-5 = 0\). We need to determine which give real values of \(y\).

Let \(x = y^2\). Then the equation \(-3y^6 + 8y^4 - 9y^2 + 10 = 0\) becomes \(f(y^2) = 0\).

From Part (a): \(f(x) = 0\) when \((x - 2) = 0\) or \(-3x^2 + 2x - 5 = 0\)

Case 1: \(x = 2\), i.e., \(y^2 = 2\), giving \(y = \pm\sqrt{2}\). This gives 2 real solutions.

Case 2: \(-3x^2 + 2x - 5 = 0\). Check the discriminant:

$$\Delta = b^2 - 4ac = (2)^2 - 4(-3)(-5) = 4 - 60 = -56$$

Since \(\Delta = -56 < 0\), this quadratic has no real roots.

So \(x = y^2\) has no real solutions from Case 2.

Therefore there are exactly two real solutions: \(y = \sqrt{2}\) and \(y = -\sqrt{2}\). \(\square\)

M1 — Recognising the substitution \(x = y^2\) and considering both factors
A1 — Complete proof: discriminant < 0 for the quadratic, so only \(y^2 = 2\) gives solutions, hence exactly 2
You must explicitly show that the quadratic factor has no real roots (by finding the discriminant) — you cannot just state it. Also, you must state that \(y^2 = 2\) gives exactly 2 values of \(y\) (the positive and negative square root).

How to Recognise This Question Type

Look for these signals in the question:

  • The phrase "given that \((x + a)\) is a factor" or "given that \(f(a) = 0\)"
  • A cubic polynomial with an unknown constant — the factor theorem gives an equation to find it
  • Part (i) asks to "calculate \(f(p)\)" for a specific value — this is a hint that \((x-p)\) is a factor
  • The instruction "factorise completely" or "write as a product of factors"
  • A problem where you need to find intersection points of a line and curve (substitute and solve the resulting cubic/quadratic)
  • The phrase "prove that there are exactly \(n\) real solutions" — use the discriminant after factorising
  • Questions asking for "the set of values" or using inequality notation — quadratic inequality after factorising

Key Algebraic Skills

  • Factor theorem: \(f(a) = 0 \iff (x-a)\) is a factor of \(f(x)\)
  • Remainder theorem: When \(f(x)\) is divided by \((x-a)\), the remainder is \(f(a)\)
  • Polynomial long division: Divide \(f(x)\) by \((x-a)\) to find the quotient (the other factor)
  • Discriminant: For \(ax^2+bx+c=0\): if \(b^2-4ac > 0\) then 2 real roots; if \(= 0\) then 1 repeated root; if \(< 0\) then no real roots
  • Quadratic inequality: Factorise, find critical values, sketch the parabola, read off the solution region
  • Simultaneous equations: Substitute the linear equation into the quadratic, solve the resulting quadratic
