The Approach

The standard approach for Edexcel circle questions:

1
Get the circle into standard form
If given \(x^2 + y^2 + 2gx + 2fy + c = 0\), complete the square on \(x\) and \(y\) separately to get \((x-a)^2 + (y-b)^2 = r^2\). The centre is \((a, b)\) and the radius is \(r\). Watch signs carefully: \(x^2 + 6x\) becomes \((x+3)^2 - 9\).
2
Use the key geometric fact: tangent is perpendicular to radius
If a line is tangent to the circle at point \(P\), then the line from the centre to \(P\) is perpendicular to the tangent. This means their gradients multiply to \(-1\). Use this to find equations of tangents or to locate the point of tangency.
3
For intersection/tangency problems: substitute and use discriminant
Substitute the line equation into the circle equation to get a quadratic. Then: 2 intersections means \(b^2 - 4ac > 0\); tangent means \(b^2 - 4ac = 0\); no intersection means \(b^2 - 4ac < 0\). Solve for the unknown parameter.
4
For two-circle problems: use distance between centres
Two circles intersect when the distance between centres satisfies \(|r_1 - r_2| < d < r_1 + r_2\). They touch externally when \(d = r_1 + r_2\) and internally when \(d = |r_1 - r_2|\).

Common Pitfalls (from Examiner Reports)

  • Sign errors when reading off the centre: From \((x+3)^2 + (y-5)^2 = 25\), the centre is \((-3, 5)\), NOT \((3, -5)\). Candidates frequently get the signs wrong when extracting coordinates from the completed square form.
  • Forgetting that tangent is perpendicular to radius: This is THE key geometric fact. If the tangent has gradient \(m\), the radius to that point has gradient \(-\frac{1}{m}\). Many candidates try to use the discriminant when perpendicularity gives a quicker route.
  • Algebraic errors in simultaneous equations: When substituting a line into a circle equation, expanding \((2x+k)^2\) or \((x-3)^2\) incorrectly is the most common source of lost marks. Expand brackets carefully and collect terms.
  • Confusing \(r^2\) with \(r\): After completing the square, candidates often write \(r = 25\) when they mean \(r^2 = 25\), so \(r = 5\). Always take the square root at the end.
  • Wrong discriminant condition: Tangent requires \(b^2-4ac = 0\) (exactly one solution), not \(> 0\). For "intersects at 2 points" you need \(b^2-4ac > 0\). Read the question carefully.

What Top Candidates Do

  • When completing the square, lay out the work clearly: write \(x^2 - 10x = (x-5)^2 - 25\) as a separate line before combining.
  • Sketch a diagram even if not asked. Mark the centre, radius, tangent point, and perpendicular relationship. This prevents sign errors and guides the algebra.
  • For discriminant questions: explicitly state the condition ("\(b^2 - 4ac = 0\) for tangency") before substituting values. This earns method marks even if subsequent algebra fails.
  • Always give the radius as a simplified surd when asked (e.g. \(3\sqrt{2}\), not \(\sqrt{18}\)).

Paper 1 — June 2018 — Question 6 (10 marks)

The circle \(C\) has centre \(A\) with coordinates \((7, 5)\). The line \(l\), with equation \(y = 2x + 1\), is the tangent to \(C\) at the point \(P\).

(a) Show that an equation of the line \(PA\) is \(2y + x = 17\) [3 marks]

(b) Find an equation for \(C\). [4 marks]

(c) The line with equation \(y = 2x + k\), \(k \neq 1\), is also a tangent to \(C\). Find the value of the constant \(k\). [3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Use perpendicularity to find the gradient of PA ▶
The tangent \(l\) touches the circle at \(P\), and \(A\) is the centre. The key geometric fact is: the radius to a point of tangency is always perpendicular to the tangent at that point. So the line \(PA\) is perpendicular to \(l\). If we know the gradient of \(l\), we can find the gradient of \(PA\) using the perpendicular gradient rule: \(m_1 \times m_2 = -1\).

The tangent \(l\) has equation \(y = 2x + 1\), so its gradient is \(2\).

Since \(PA\) is perpendicular to the tangent (radius meets tangent at right angles):

$$\text{Gradient of } PA = -\frac{1}{2}$$

M1 — Uses perpendicular gradients to deduce gradient of PA is \(-\frac{1}{2}\)
Part (a) Step 2: Find the equation of line PA ▶
We now know the gradient of \(PA\) is \(-\frac{1}{2}\) and it passes through \(A(7, 5)\). We use the point-gradient form \(y - y_1 = m(x - x_1)\) to write the equation, then rearrange to the given form \(2y + x = 17\).

Using \(y - y_1 = m(x - x_1)\) with \(m = -\frac{1}{2}\) and \((7, 5)\):

$$y - 5 = -\frac{1}{2}(x - 7)$$

Multiply both sides by 2:

$$2y - 10 = -(x - 7)$$

Expand the right side:

$$2y - 10 = -x + 7$$

Rearrange: add \(x\) and add 10 to both sides:

$$\boxed{2y + x = 17} \quad \square$$

M1 — Finding the equation of a line with changed gradient through \((7, 5)\)
A1* — Completing the proof with no errors: \(2y + x = 17\)
Part (b) Step 3: Find the coordinates of P by solving simultaneously ▶
To write the equation of the circle, we need the radius. The radius equals the distance \(|PA|\). We know \(A(7,5)\) already, so we need the coordinates of \(P\). Point \(P\) lies on both line \(l\) (the tangent \(y = 2x + 1\)) and line \(PA\) (\(2y + x = 17\)). So we solve these simultaneously.

Solve \(y = 2x + 1\) and \(2y + x = 17\) simultaneously:

Substitute \(y = 2x + 1\) into \(2y + x = 17\):

$$2(2x + 1) + x = 17$$

Expand:

$$4x + 2 + x = 17$$

Simplify:

$$5x = 15$$

$$x = 3$$

Find \(y\): \(y = 2(3) + 1 = 7\)

$$P = (3, 7)$$

M1 — Solving \(2y + x = 17\) and \(y = 2x + 1\) simultaneously
A1 — \(P = (3, 7)\)
Part (b) Step 4: Find the radius and write the equation of C ▶
The equation of a circle with centre \((a, b)\) and radius \(r\) is \((x-a)^2 + (y-b)^2 = r^2\). We know the centre is \(A(7, 5)\). The radius is the distance from \(A\) to \(P\), which we find using Pythagoras: \(r^2 = (x_2-x_1)^2 + (y_2-y_1)^2\). Note: we need \(r^2\) for the equation, so there's no need to square root.

$$r^2 = (7-3)^2 + (5-7)^2 = 16 + 4 = 20$$

The equation of \(C\) is:

$$\boxed{(x-7)^2 + (y-5)^2 = 20}$$

M1 — Using Pythagoras to find the radius (or \(r^2\)) from \(P(3,7)\) and \(A(7,5)\)
A1 — \((x-7)^2 + (y-5)^2 = 20\)
Do not write \((x-7)^2 + (y-5)^2 = \sqrt{20}\) — the right-hand side must be \(r^2\), not \(r\). The examiner reported this as a common error.
Part (c) Step 5: Find the second tangent using the discriminant ▶
We need another line \(y = 2x + k\) that is also tangent to \(C\). A line is tangent to a circle when it meets the circle at exactly one point, which means the quadratic (from substituting the line into the circle) has discriminant = 0. Alternatively, since both tangents have gradient 2, the second tangent point can be found using the perpendicular from centre to line. Here we use the discriminant method.

Substitute \(y = 2x + k\) into \((x-7)^2 + (y-5)^2 = 20\):

$$(x-7)^2 + (2x+k-5)^2 = 20$$

Expand \((x-7)^2 = x^2 - 14x + 49\):

Expand \((2x+k-5)^2 = 4x^2 + 4(k-5)x + (k-5)^2\):

$$x^2 - 14x + 49 + 4x^2 + 4(k-5)x + (k-5)^2 = 20$$

Collect terms:

$$5x^2 + (4k - 34)x + (k^2 - 10k + 54) = 0$$

For tangency, the discriminant \(= 0\):

$$(4k-34)^2 - 4(5)(k^2-10k+54) = 0$$

Expand: \(16k^2 - 272k + 1156 - 20k^2 + 200k - 1080 = 0\)

$$-4k^2 - 72k + 76 = 0$$

Divide by \(-4\):

$$k^2 + 18k - 19 = 0$$

Factorise:

$$(k + 19)(k - 1) = 0$$

$$k = -19 \quad \text{or} \quad k = 1$$

Since the question states \(k \neq 1\) (that's the original tangent):

$$\boxed{k = -19}$$

M1 — Substituting \(y = 2x + k\) into the circle and forming a quadratic
M1 — Using the discriminant condition \(b^2 - 4ac = 0\) and solving for \(k\)
A1 — \(k = -19\)
Candidates who found both \(k = 1\) and \(k = -19\) but failed to reject \(k = 1\) lost the final mark. Always check constraints given in the question.

Paper 1 — October 2021 — Question 7 (9 marks)

The circle \(C\) has equation

$$x^2 + y^2 - 10x + 4y + 11 = 0$$

(a) Find (i) the coordinates of the centre of \(C\), (ii) the exact radius of \(C\), giving your answer as a simplified surd. [4 marks]

(b) The line \(l\) has equation \(y = 3x + k\) where \(k\) is a constant. Given that \(l\) is a tangent to \(C\), find the possible values of \(k\), giving your answers as simplified surds. [5 marks]

View Question Paper View Mark Scheme
Part (a)(i) Step 1: Complete the square to find the centre ▶
The equation is in expanded form \(x^2 + y^2 - 10x + 4y + 11 = 0\). We need to rewrite it in standard form \((x-a)^2 + (y-b)^2 = r^2\) by completing the square on both \(x\) and \(y\) separately. The centre will then be \((a, b)\).

Complete the square on \(x\) terms and \(y\) terms separately:

Group the \(x\) terms: \(x^2 - 10x\). Half of \(-10\) is \(-5\), so:

$$x^2 - 10x = (x - 5)^2 - 25$$

Group the \(y\) terms: \(y^2 + 4y\). Half of \(4\) is \(2\), so:

$$y^2 + 4y = (y + 2)^2 - 4$$

Substitute back into the equation:

$$(x-5)^2 - 25 + (y+2)^2 - 4 + 11 = 0$$

Collect the constants:

$$(x-5)^2 + (y+2)^2 = 25 + 4 - 11 = 18$$

The centre is \(\boxed{(5, -2)}\)

M1 — Attempts to complete the square by halving both the \(x\) and \(y\) coefficients
A1 — Correct centre: \((5, -2)\)
A common error is writing the centre as \((5, 2)\) instead of \((5, -2)\). Remember: from \((y+2)^2\), the centre's \(y\)-coordinate is \(-2\), not \(+2\).
Part (a)(ii) Step 2: Find the radius ▶
From the standard form \((x-5)^2 + (y+2)^2 = 18\), the right-hand side is \(r^2\). So \(r^2 = 18\) and we need \(r = \sqrt{18}\) simplified as a surd. To simplify: find the largest perfect square factor of 18 (which is 9), then \(\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}\).

From \((x-5)^2 + (y+2)^2 = 18\):

$$r^2 = 18$$

$$r = \sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}$$

The radius is \(\boxed{3\sqrt{2}}\)

M1 — Correct strategy for finding the radius (square root of the RHS)
A1 — \(r = 3\sqrt{2}\) (simplified surd; do not accept \(\sqrt{18}\) or \(\pm 3\sqrt{2}\))
Part (b) Step 3: Substitute the line into the circle equation ▶
The line \(y = 3x + k\) is tangent to \(C\). This means substituting into the circle gives a quadratic with exactly one solution, i.e. discriminant = 0. We substitute \(y = 3x + k\) into the original circle equation (or the completed square form) and simplify to a quadratic in \(x\).

Substitute \(y = 3x + k\) into \(x^2 + y^2 - 10x + 4y + 11 = 0\):

$$x^2 + (3x+k)^2 - 10x + 4(3x+k) + 11 = 0$$

Expand \((3x+k)^2 = 9x^2 + 6kx + k^2\):

Expand \(4(3x+k) = 12x + 4k\):

$$x^2 + 9x^2 + 6kx + k^2 - 10x + 12x + 4k + 11 = 0$$

Collect terms in \(x^2\), \(x\), and constants:

$$10x^2 + (6k + 2)x + (k^2 + 4k + 11) = 0$$

M1 — Substitutes \(y = 3x + k\) into the circle equation and expands brackets
A1 — Correct 3-term quadratic: \(10x^2 + (6k+2)x + (k^2+4k+11) = 0\)
Part (b) Step 4: Apply the discriminant condition for tangency ▶
For the line to be a tangent, the quadratic must have exactly one repeated root. This means \(b^2 - 4ac = 0\). We identify \(a = 10\), \(b = (6k+2)\), \(c = (k^2+4k+11)\), substitute into the discriminant formula, and solve for \(k\).

For tangency: \(b^2 - 4ac = 0\)

$$(6k+2)^2 - 4(10)(k^2+4k+11) = 0$$

Expand \((6k+2)^2 = 36k^2 + 24k + 4\):

Expand \(4(10)(k^2+4k+11) = 40k^2 + 160k + 440\):

$$36k^2 + 24k + 4 - 40k^2 - 160k - 440 = 0$$

Simplify:

$$-4k^2 - 136k - 436 = 0$$

Divide by \(-4\):

$$k^2 + 34k + 109 = 0$$

M1 — Recognises the requirement to use \(b^2 - 4ac = 0\) where \(b\) and \(c\) are expressions in \(k\)
Part (b) Step 5: Solve the quadratic in \(k\) using the formula ▶
The quadratic \(k^2 + 34k + 109 = 0\) does not factorise neatly, so we use the quadratic formula. The discriminant is \(34^2 - 4(1)(109) = 1156 - 436 = 720\). We need to simplify \(\sqrt{720}\) as a surd.

$$k = \frac{-34 \pm \sqrt{34^2 - 4(1)(109)}}{2}$$

$$k = \frac{-34 \pm \sqrt{1156 - 436}}{2}$$

$$k = \frac{-34 \pm \sqrt{720}}{2}$$

Simplify \(\sqrt{720} = \sqrt{144 \times 5} = 12\sqrt{5}\):

$$k = \frac{-34 \pm 12\sqrt{5}}{2}$$

$$k = -17 \pm 6\sqrt{5}$$

$$\boxed{k = -17 + 6\sqrt{5} \quad \text{or} \quad k = -17 - 6\sqrt{5}}$$

M1 — Solves the quadratic in \(k\) using the formula
A1 — Correct simplified values: \(k = -17 \pm 6\sqrt{5}\)
Both values of \(k\) are required. Candidates who left the answer as \(\frac{-34 \pm 12\sqrt{5}}{2}\) without simplifying lost the A mark. The question explicitly asks for "simplified surds".

Paper 2 — October 2020 — Question 14 (7 marks)

A circle \(C\) with radius \(r\)

  • lies only in the 1st quadrant
  • touches the \(x\)-axis and touches the \(y\)-axis

The line \(l\) has equation \(2x + y = 12\)

(a) Show that the \(x\)-coordinates of the points of intersection of \(l\) with \(C\) satisfy
\(5x^2 + (2r - 48)x + (r^2 - 24r + 144) = 0\) [3 marks]

(b) Given also that \(l\) is a tangent to \(C\), find the two possible values of \(r\), giving your answers as fully simplified surds. [4 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Write down the equation of the circle ▶
The circle touches both axes and lies in the 1st quadrant. If it touches the \(x\)-axis, the centre is at height \(r\) above it. If it touches the \(y\)-axis, the centre is at distance \(r\) from it. So the centre is \((r, r)\) and the equation is \((x-r)^2 + (y-r)^2 = r^2\). This is the key insight that connects the geometric conditions to an algebraic equation.

Since the circle touches both axes and has radius \(r\), the centre must be at \((r, r)\).

The equation of \(C\) is:

$$(x - r)^2 + (y - r)^2 = r^2$$

B1 — Deduces the correct equation of the circle
Part (a) Step 2: Substitute the line and simplify to the given quadratic ▶
We need to find where \(l: 2x + y = 12\) meets \(C\). Rearrange the line as \(y = 12 - 2x\) and substitute into the circle equation. Then expand and simplify to show we get the given quadratic. Since it's a "show that", every step must be visible.

From \(2x + y = 12\): \(y = 12 - 2x\). Substitute into \((x-r)^2 + (y-r)^2 = r^2\):

$$(x - r)^2 + (12 - 2x - r)^2 = r^2$$

Expand \((x-r)^2 = x^2 - 2rx + r^2\):

Expand \((12-2x-r)^2 = 4x^2 + r^2 + 144 - 48x + 4rx - 24r\):

Let us verify: \((12-2x-r)^2 = (12-r)^2 - 2(12-r)(2x) + (2x)^2\)

\(= 144 - 24r + r^2 - 4x(12-r) + 4x^2\)

\(= 4x^2 - 48x + 4rx + 144 - 24r + r^2\)

Now add both expanded expressions:

$$x^2 - 2rx + r^2 + 4x^2 - 48x + 4rx + 144 - 24r + r^2 = r^2$$

Collect \(x^2\) terms: \(5x^2\)

Collect \(x\) terms: \(-2rx - 48x + 4rx = (2r-48)x\)

Collect constants: \(r^2 + 144 - 24r + r^2 = 2r^2 - 24r + 144\)

$$5x^2 + (2r-48)x + 2r^2 - 24r + 144 = r^2$$

Subtract \(r^2\) from both sides:

$$\boxed{5x^2 + (2r - 48)x + (r^2 - 24r + 144) = 0} \quad \square$$

M1 — Substitutes \(y = 12 - 2x\) into the circle equation and expands
A1* — Correct and accurate algebra leading to the given answer
Part (b) Step 3: Apply the discriminant condition for tangency ▶
For \(l\) to be tangent to \(C\), the quadratic in \(x\) must have exactly one solution (the line touches the circle at one point). This means discriminant \(= 0\). We have \(a = 5\), \(b = (2r-48)\), \(c = (r^2 - 24r + 144)\). Setting \(b^2 - 4ac = 0\) gives us an equation in \(r\) to solve.

For tangency: \(b^2 - 4ac = 0\)

$$(2r - 48)^2 - 4(5)(r^2 - 24r + 144) = 0$$

Expand \((2r-48)^2 = 4r^2 - 192r + 2304\):

Expand \(20(r^2-24r+144) = 20r^2 - 480r + 2880\):

$$4r^2 - 192r + 2304 - 20r^2 + 480r - 2880 = 0$$

Simplify:

$$-16r^2 + 288r - 576 = 0$$

Divide by \(-16\):

$$r^2 - 18r + 36 = 0$$

M1 — Uses \(b^2 - 4ac = 0\) with appropriate values
A1 — Correct quadratic equation in \(r\): \(r^2 - 18r + 36 = 0\)
Part (b) Step 4: Solve for \(r\) using the quadratic formula ▶
The quadratic \(r^2 - 18r + 36 = 0\) does not factorise neatly, so we apply the quadratic formula. The discriminant is \(18^2 - 4(36) = 324 - 144 = 180\). We simplify \(\sqrt{180} = 6\sqrt{5}\) and divide through.

$$r = \frac{18 \pm \sqrt{324 - 144}}{2}$$

$$r = \frac{18 \pm \sqrt{180}}{2}$$

Simplify: \(\sqrt{180} = \sqrt{36 \times 5} = 6\sqrt{5}\)

$$r = \frac{18 \pm 6\sqrt{5}}{2}$$

$$r = 9 \pm 3\sqrt{5}$$

$$\boxed{r = 9 + 3\sqrt{5} \quad \text{or} \quad r = 9 - 3\sqrt{5}}$$

dM1 — Correct attempt to solve the quadratic in \(r\) (dependent on previous M)
A1 — Both answers in fully simplified surd form: \(r = 9 \pm 3\sqrt{5}\)
Both values are geometrically valid: the smaller circle (\(r = 9 - 3\sqrt{5} \approx 2.29\)) touches the line on one side of the first quadrant, and the larger circle (\(r = 9 + 3\sqrt{5} \approx 15.71\)) touches it on the other side. Candidates must simplify the surds fully.

Paper 1 — June 2023 — Question 10 (9 marks)

A circle \(C\) has equation

$$x^2 + y^2 + 6kx - 2ky + 7 = 0$$

where \(k\) is a constant.

(a) Find in terms of \(k\), (i) the coordinates of the centre of \(C\), (ii) the radius of \(C\). [3 marks]

(b) The line with equation \(y = 2x - 1\) intersects \(C\) at 2 distinct points. Find the range of possible values of \(k\). [6 marks]

View Question Paper View Mark Scheme
Part (a)(i) Step 1: Find the centre in terms of \(k\) ▶
The general circle equation \(x^2 + y^2 + 2gx + 2fy + c = 0\) has centre \((-g, -f)\) and radius \(\sqrt{g^2 + f^2 - c}\). Here we have \(2g = 6k\) so \(g = 3k\), and \(2f = -2k\) so \(f = -k\). Alternatively, complete the square: the \(x\) terms are \(x^2 + 6kx = (x+3k)^2 - 9k^2\) and the \(y\) terms are \(y^2 - 2ky = (y-k)^2 - k^2\).

Complete the square on both variables:

\(x^2 + 6kx = (x + 3k)^2 - 9k^2\)

\(y^2 - 2ky = (y - k)^2 - k^2\)

Substituting back:

$$(x+3k)^2 - 9k^2 + (y-k)^2 - k^2 + 7 = 0$$

$$(x+3k)^2 + (y-k)^2 = 10k^2 - 7$$

The centre is \(\boxed{(-3k,\; k)}\)

B1 — Correct centre \((-3k, k)\)
Part (a)(ii) Step 2: Find the radius in terms of \(k\) ▶
From the standard form \((x+3k)^2 + (y-k)^2 = 10k^2 - 7\), we read off \(r^2 = 10k^2 - 7\). The radius is the positive square root of this.

From \((x+3k)^2 + (y-k)^2 = 10k^2 - 7\):

$$r = \boxed{\sqrt{10k^2 - 7}}$$

M1 — Correct strategy for finding \(r^2\) by completing the square and collecting terms
A1ft — Radius is \(\sqrt{10k^2 - 7}\)
Do not allow \(\pm\sqrt{10k^2 - 7}\) and do not accept \(10k^2 - 7\) as the radius (that is \(r^2\)). The answer must be explicitly written as the square root.
Part (b) Step 3: Substitute the line into the circle equation ▶
The line \(y = 2x - 1\) intersects the circle at 2 distinct points. We substitute into the circle equation to get a quadratic in \(x\). For 2 distinct intersection points, we need the discriminant to be strictly positive (\(b^2 - 4ac > 0\)).

Substitute \(y = 2x - 1\) into \(x^2 + y^2 + 6kx - 2ky + 7 = 0\):

$$x^2 + (2x-1)^2 + 6kx - 2k(2x-1) + 7 = 0$$

Expand \((2x-1)^2 = 4x^2 - 4x + 1\):

Expand \(-2k(2x-1) = -4kx + 2k\):

$$x^2 + 4x^2 - 4x + 1 + 6kx - 4kx + 2k + 7 = 0$$

Collect terms:

$$5x^2 + (2k - 4)x + (2k + 8) = 0$$

M1 — Substitutes \(y = 2x - 1\) and collects terms to form a quadratic
A1 — Correct quadratic: \(5x^2 + (2k-4)x + (2k+8) = 0\)
Part (b) Step 4: Apply discriminant condition for 2 distinct intersections ▶
For 2 distinct intersection points, we need \(b^2 - 4ac > 0\). Here \(a = 5\), \(b = (2k-4)\), \(c = (2k+8)\). We compute the discriminant, set it greater than zero, and solve the resulting quadratic inequality.

For 2 distinct points: \(b^2 - 4ac > 0\)

$$(2k-4)^2 - 4(5)(2k+8) > 0$$

Expand \((2k-4)^2 = 4k^2 - 16k + 16\):

Expand \(20(2k+8) = 40k + 160\):

$$4k^2 - 16k + 16 - 40k - 160 > 0$$

Simplify:

$$4k^2 - 56k - 144 > 0$$

Divide by 4:

$$k^2 - 14k - 36 > 0$$

dM1 — Sets up \(b^2 - 4ac > 0\) and attempts to find critical values
Part (b) Step 5: Solve the quadratic inequality ▶
We need to solve \(k^2 - 14k - 36 > 0\). First find the critical values (roots) using the quadratic formula, then determine the solution set. Since the coefficient of \(k^2\) is positive, the parabola opens upward, so the quadratic is positive outside the roots.

Find the critical values of \(k^2 - 14k - 36 = 0\):

$$k = \frac{14 \pm \sqrt{196 + 144}}{2} = \frac{14 \pm \sqrt{340}}{2}$$

Simplify: \(\sqrt{340} = \sqrt{4 \times 85} = 2\sqrt{85}\)

$$k = \frac{14 \pm 2\sqrt{85}}{2} = 7 \pm \sqrt{85}$$

Since the parabola \(k^2 - 14k - 36\) opens upward, the quadratic is positive when \(k\) is outside the roots:

$$\boxed{k < 7 - \sqrt{85} \quad \text{or} \quad k > 7 + \sqrt{85}}$$

A1 — Correct critical values: \(7 \pm \sqrt{85}\)
ddM1 — Selects the outside region for their critical values
A1 — Correct final answer: \(k < 7 - \sqrt{85}\) or \(k > 7 + \sqrt{85}\)

You can use set notation: \(\{k : k < 7 - \sqrt{85}\} \cup \{k : k > 7 + \sqrt{85}\}\). Do NOT use "and" between the two conditions — use "or".

How to Recognise This Question Type

Look for these signals in the question:

  • An equation with \(x^2 + y^2\) terms (both squared, both coefficient 1)
  • The words "circle", "centre", "radius", "tangent"
  • A Part (a) asking you to "find the centre and radius" — this is the setup for later parts
  • A line equation \(y = mx + c\) combined with a circle — signals substitution and discriminant
  • The phrase "tangent to the circle" tells you to use \(b^2 - 4ac = 0\)
  • Two circles: expect to use the distance between centres and the triangle inequality on radii
  • Usually appears in the Q3–10 range, worth 5–9 marks

Key Formulae Summary

  • Standard form: Learn this \((x-a)^2 + (y-b)^2 = r^2\) — centre \((a,b)\), radius \(r\)
  • Expanded form: Learn this \(x^2 + y^2 + 2gx + 2fy + c = 0\) — centre \((-g, -f)\), radius \(\sqrt{g^2 + f^2 - c}\)
  • Tangent perpendicular to radius: Learn this if tangent has gradient \(m\), radius to that point has gradient \(-\frac{1}{m}\)
  • Tangency via discriminant: Learn this substitute line into circle, set \(b^2 - 4ac = 0\)
  • Two circles intersect: Learn this \(|r_1 - r_2| < d < r_1 + r_2\) where \(d\) = distance between centres
