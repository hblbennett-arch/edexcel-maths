The Approach

The standard structure for Edexcel vector questions:

1
Identify what vectors you have and what you need
Write down the given position vectors. Identify what the question asks: a direction vector (\(\overrightarrow{AB}\)), a magnitude (\(|\overrightarrow{AB}|\)), or a geometric relationship (parallel, collinear). Remember: \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\) (end minus start).
2
Find the direction vectors you need
Subtract position vectors to get direction vectors between points. For geometric proofs, you need to compare vectors: if \(\overrightarrow{AB} = k\overrightarrow{CD}\) for scalar \(k\), then AB is parallel to CD. If points are collinear, one vector is a scalar multiple of another.
3
Apply the geometric condition and conclude
For parallel: show one vector = scalar multiple of another. For collinearity: show the vectors share a common point AND are parallel. For magnitude: use \(|\mathbf{v}| = \sqrt{x^2 + y^2 + z^2}\). Always write an explicit conclusion statement.

Common Pitfalls (from Examiner Reports)

  • Direction errors (wrong way around): \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\), NOT \(\overrightarrow{OA} - \overrightarrow{OB}\). The end point comes first in the subtraction. Getting this backwards reverses the sign.
  • Confusing position vectors and direction vectors: A position vector gives the location of a point relative to O. A direction vector (like \(\overrightarrow{AB}\)) gives the displacement between two points. They are different things.
  • Not concluding geometric proofs: Showing \(\overrightarrow{OC} = 2\overrightarrow{AB}\) is NOT enough. You must explicitly state "therefore OC is parallel to AB" and "therefore OABC is a trapezium." The conclusion earns the A mark.
  • Forgetting "only one pair" for trapezium: A trapezium has exactly one pair of parallel sides. If the question says "trapezium" you may need to show the other pair is NOT parallel.
  • Using coordinates instead of vectors: Writing \((3, -8, 2)\) instead of \(3\mathbf{i} - 8\mathbf{j} + 2\mathbf{k}\) loses the A mark. Use proper vector notation.

What Top Candidates Do

  • Write the subtraction out in full: \(\overrightarrow{AB} = (3\mathbf{i} - 3\mathbf{j} - 4\mathbf{k}) - (2\mathbf{i} + 5\mathbf{j} - 6\mathbf{k})\) before simplifying.
  • State the parallel condition clearly: "Since \(\overrightarrow{OC} = 2\overrightarrow{AB}\), OC is parallel to AB."
  • For collinearity, show two things: (1) vectors are parallel AND (2) they share a common point.
  • For "show that" magnitude questions, show the individual squares: \(\sqrt{5^2 + 3^2 + 2^2} = \sqrt{25 + 9 + 4} = \sqrt{38}\).
  • Keep vector notation consistent throughout (either \(\mathbf{i}, \mathbf{j}, \mathbf{k}\) or column vectors, not a mix).

Paper 1 — October 2020 — Question 3 (4 marks)

Relative to a fixed origin O:

  • point A has position vector \(2\mathbf{i} + 5\mathbf{j} - 6\mathbf{k}\)
  • point B has position vector \(3\mathbf{i} - 3\mathbf{j} - 4\mathbf{k}\)
  • point C has position vector \(2\mathbf{i} - 16\mathbf{j} + 4\mathbf{k}\)

(a) Find \(\overrightarrow{AB}\). [2 marks]

(b) Show that quadrilateral OABC is a trapezium, giving reasons for your answer. [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find \(\overrightarrow{AB}\) ▶
We need to find the direction vector from A to B. The formula is always "end minus start": \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\). This gives the displacement you'd travel going from A to B.

Apply \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\):

$$\overrightarrow{AB} = (3\mathbf{i} - 3\mathbf{j} - 4\mathbf{k}) - (2\mathbf{i} + 5\mathbf{j} - 6\mathbf{k})$$

Subtract component by component: \(3-2\), \(-3-5\), \(-4-(-6)\)

$$\overrightarrow{AB} = \mathbf{i} - 8\mathbf{j} + 2\mathbf{k}$$

M1 — Attempts subtraction of position vectors (either way around)
A1 — Correct: \(\mathbf{i} - 8\mathbf{j} + 2\mathbf{k}\)
The answer must be in vector form. Writing \((1, -8, 2)\) as coordinates loses the A1 mark. Column vector notation is acceptable.
Part (b) Step 2: Show OABC is a trapezium ▶
A trapezium has one pair of parallel sides. The vertices are O, A, B, C so the sides are OA, AB, BC, and OC. We need to show that one pair is parallel. Two vectors are parallel if one is a scalar multiple of the other. Let's compare \(\overrightarrow{OC}\) with \(\overrightarrow{AB}\) since we already know \(\overrightarrow{AB}\).

We already have \(\overrightarrow{AB} = \mathbf{i} - 8\mathbf{j} + 2\mathbf{k}\).

The position vector of C gives us \(\overrightarrow{OC}\) directly:

$$\overrightarrow{OC} = 2\mathbf{i} - 16\mathbf{j} + 4\mathbf{k}$$

Now compare:

$$\overrightarrow{OC} = 2\mathbf{i} - 16\mathbf{j} + 4\mathbf{k} = 2(\mathbf{i} - 8\mathbf{j} + 2\mathbf{k}) = 2\overrightarrow{AB}$$

Since \(\overrightarrow{OC} = 2\overrightarrow{AB}\), the vectors are parallel (one is a scalar multiple of the other).

Therefore OC is parallel to AB, so OABC is a trapezium.

M1 — Compares \(\overrightarrow{OC}\) with \(\overrightarrow{AB}\) by stating one is a scalar multiple of the other
A1 — Complete argument: correct calculation, states parallel, concludes "trapezium"
You must include all three elements: (1) show the scalar multiple relationship \(\overrightarrow{OC} = 2\overrightarrow{AB}\), (2) state "therefore parallel", (3) conclude "trapezium". Just showing the calculation without the conclusion loses the A1.

Paper 2 — June 2018 — Question 2 (5 marks)

Relative to a fixed origin O:

  • the point A has position vector \(2\mathbf{i} + 3\mathbf{j} - 4\mathbf{k}\)
  • the point B has position vector \(4\mathbf{i} - 2\mathbf{j} + 3\mathbf{k}\)
  • the point C has position vector \(a\mathbf{i} + 5\mathbf{j} - 2\mathbf{k}\), where \(a\) is a constant and \(a < 0\)

D is the point such that \(\overrightarrow{AB} = \overrightarrow{BD}\).

(a) Find the position vector of D. [2 marks]

Given \(|\overrightarrow{AC}| = 4\)

(b) Find the value of \(a\). [3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find \(\overrightarrow{AB}\) ▶
The condition \(\overrightarrow{AB} = \overrightarrow{BD}\) tells us that D is the point you reach by continuing past B by the same displacement as A to B. First we need to find \(\overrightarrow{AB}\), then use it to locate D.

Find \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\):

$$\overrightarrow{AB} = (4\mathbf{i} - 2\mathbf{j} + 3\mathbf{k}) - (2\mathbf{i} + 3\mathbf{j} - 4\mathbf{k})$$

$$= 2\mathbf{i} - 5\mathbf{j} + 7\mathbf{k}$$

Part (a) Step 2: Find the position vector of D ▶
Since \(\overrightarrow{BD} = \overrightarrow{AB}\), we can find D by adding \(\overrightarrow{AB}\) to the position of B. The position vector of D is \(\overrightarrow{OD} = \overrightarrow{OB} + \overrightarrow{BD} = \overrightarrow{OB} + \overrightarrow{AB}\).

Since \(\overrightarrow{BD} = \overrightarrow{AB} = 2\mathbf{i} - 5\mathbf{j} + 7\mathbf{k}\):

$$\overrightarrow{OD} = \overrightarrow{OB} + \overrightarrow{BD}$$

$$= (4\mathbf{i} - 2\mathbf{j} + 3\mathbf{k}) + (2\mathbf{i} - 5\mathbf{j} + 7\mathbf{k})$$

$$= 6\mathbf{i} - 7\mathbf{j} + 10\mathbf{k}$$

M1 — Strategy to find \(\overrightarrow{OD}\), e.g. \(\overrightarrow{OD} = \overrightarrow{OB} + \overrightarrow{AB}\) or equivalent
A1 — Correct position vector: \(6\mathbf{i} - 7\mathbf{j} + 10\mathbf{k}\)
Alternative methods: \(\overrightarrow{OD} = 2\overrightarrow{OB} - \overrightarrow{OA}\) also works. The key is recognising that \(\overrightarrow{AB} = \overrightarrow{BD}\) means B is the midpoint of AD. Giving coordinates \((6, -7, 10)\) without proper vector form loses the A1.
Part (b) Step 3: Use magnitude condition to find \(a\) ▶
We're told \(|\overrightarrow{AC}| = 4\). The magnitude formula is \(|\overrightarrow{AC}| = \sqrt{(a-2)^2 + (5-3)^2 + (-2-(-4))^2}\). Setting this equal to 4 gives us an equation in \(a\). Since \(a < 0\), we'll need to consider which solution satisfies that condition.

Find \(\overrightarrow{AC}\):

$$\overrightarrow{AC} = (a\mathbf{i} + 5\mathbf{j} - 2\mathbf{k}) - (2\mathbf{i} + 3\mathbf{j} - 4\mathbf{k}) = (a-2)\mathbf{i} + 2\mathbf{j} + 2\mathbf{k}$$

Apply the magnitude condition \(|\overrightarrow{AC}| = 4\):

$$\sqrt{(a-2)^2 + 2^2 + 2^2} = 4$$

Square both sides:

$$(a-2)^2 + 4 + 4 = 16$$

Simplify:

$$(a-2)^2 = 8$$

Take square root:

$$a - 2 = \pm 2\sqrt{2}$$

$$a = 2 + 2\sqrt{2} \quad \text{or} \quad a = 2 - 2\sqrt{2}$$

Since \(a < 0\), and \(2 + 2\sqrt{2} \approx 4.83 > 0\), while \(2 - 2\sqrt{2} \approx -0.83 < 0\):

$$\boxed{a = 2 - 2\sqrt{2}}$$

M1 — Setting up Pythagoras with \(|\overrightarrow{AC}| = 4\) correctly
A1 — Correct simplification to \((a-2)^2 = 8\)
A1 — Correct final answer \(a = 2 - 2\sqrt{2}\) (selecting negative root)

Paper 2 — June 2022 — Question 13 (6 marks)

Relative to a fixed origin O:

  • the point A has position vector \(4\mathbf{i} - 3\mathbf{j} + 5\mathbf{k}\)
  • the point B has position vector \(4\mathbf{j} + 6\mathbf{k}\)
  • the point C has position vector \(-16\mathbf{i} + p\mathbf{j} + 10\mathbf{k}\), where \(p\) is a constant

Given that A, B and C lie on a straight line,

(a) find the value of \(p\). [3 marks]

The line segment OB is extended to a point D so that CD is parallel to OA.

(b) Find \(|\overrightarrow{OD}|\), writing your answer as a fully simplified surd. [3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find direction vector \(\overrightarrow{AB}\) ▶
If A, B, and C are collinear (lie on a straight line), then \(\overrightarrow{AC}\) must be a scalar multiple of \(\overrightarrow{AB}\). We'll find \(\overrightarrow{AB}\) first, then use the collinearity condition to find \(p\).

Find \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\):

$$\overrightarrow{AB} = (0\mathbf{i} + 4\mathbf{j} + 6\mathbf{k}) - (4\mathbf{i} - 3\mathbf{j} + 5\mathbf{k})$$

$$= -4\mathbf{i} + 7\mathbf{j} + \mathbf{k}$$

M1 — Attempts at least two of the relevant vectors (AB, AC, or BC)
Part (a) Step 2: Use collinearity to find \(p\) ▶
For A, B, C to be collinear, \(\overrightarrow{AC}\) must equal \(\lambda \overrightarrow{AB}\) for some scalar \(\lambda\). We can use the \(\mathbf{i}\) or \(\mathbf{k}\) components to find \(\lambda\), then use the \(\mathbf{j}\) component to find \(p\).

Find \(\overrightarrow{AC}\):

$$\overrightarrow{AC} = (-16\mathbf{i} + p\mathbf{j} + 10\mathbf{k}) - (4\mathbf{i} - 3\mathbf{j} + 5\mathbf{k})$$

$$= -20\mathbf{i} + (p+3)\mathbf{j} + 5\mathbf{k}$$

For collinearity, \(\overrightarrow{AC} = \lambda \overrightarrow{AB}\). Comparing \(\mathbf{i}\) components:

$$-20 = \lambda \times (-4) \implies \lambda = 5$$

Check with \(\mathbf{k}\) components: \(5 = 5 \times 1 = 5\) ✓

Now use the \(\mathbf{j}\) component:

$$p + 3 = 5 \times 7 = 35$$

$$\boxed{p = 32}$$

dM1 — Uses the parallel/collinear condition to set up an equation for \(p\)
A1 — \(p = 32\)
Part (b) Step 3: Find \(|\overrightarrow{OD}|\) using the parallel condition ▶
D lies on OB extended, so \(\overrightarrow{OD} = \lambda \overrightarrow{OB} = \lambda(4\mathbf{j} + 6\mathbf{k})\) for some \(\lambda > 1\). The condition is that CD is parallel to OA. If we find \(\overrightarrow{CD}\) in terms of \(\lambda\) and force it to be parallel to \(\overrightarrow{OA} = 4\mathbf{i} - 3\mathbf{j} + 5\mathbf{k}\), we can determine \(\lambda\).

Since D lies on OB extended: \(\overrightarrow{OD} = \lambda(4\mathbf{j} + 6\mathbf{k}) = 4\lambda\mathbf{j} + 6\lambda\mathbf{k}\)

Find \(\overrightarrow{CD} = \overrightarrow{OD} - \overrightarrow{OC}\):

$$\overrightarrow{CD} = (4\lambda\mathbf{j} + 6\lambda\mathbf{k}) - (-16\mathbf{i} + 32\mathbf{j} + 10\mathbf{k})$$

$$= 16\mathbf{i} + (4\lambda - 32)\mathbf{j} + (6\lambda - 10)\mathbf{k}$$

For CD parallel to \(\overrightarrow{OA} = 4\mathbf{i} - 3\mathbf{j} + 5\mathbf{k}\), ratios of components must be equal:

Using \(\mathbf{i}\) and \(\mathbf{j}\) components: \(\frac{16}{4} = \frac{4\lambda - 32}{-3}\)

$$4 = \frac{4\lambda - 32}{-3}$$

$$-12 = 4\lambda - 32$$

$$4\lambda = 20 \implies \lambda = 5$$

Therefore \(\overrightarrow{OD} = 5(4\mathbf{j} + 6\mathbf{k}) = 20\mathbf{j} + 30\mathbf{k}\)

Find the magnitude:

$$|\overrightarrow{OD}| = \sqrt{20^2 + 30^2} = \sqrt{400 + 900} = \sqrt{1300}$$

Simplify: \(\sqrt{1300} = \sqrt{100 \times 13} = 10\sqrt{13}\)

$$\boxed{|\overrightarrow{OD}| = 10\sqrt{13}}$$

M1 — Deduces \(\overrightarrow{OD} = \lambda\overrightarrow{OB}\) and attempts \(\overrightarrow{CD}\)
dM1 — Correct use of "CD parallel to OA" to find \(\lambda\)
A1 — \(|\overrightarrow{OD}| = 10\sqrt{13}\)
The answer must be a fully simplified surd. Writing \(\sqrt{1300}\) without simplifying to \(10\sqrt{13}\) loses the final A1. Alternative method: use similar triangles with the ratio from part (a).

Paper 1 — June 2022 — Question 10 (6 marks)

Figure 3 shows a sketch of a parallelogram PQRS.

Given that:

  • \(\overrightarrow{PQ} = 2\mathbf{i} + 3\mathbf{j} - 4\mathbf{k}\)
  • \(\overrightarrow{QR} = 5\mathbf{i} - 2\mathbf{k}\)

(a) Show that parallelogram PQRS is a rhombus. [2 marks]

(b) Find the exact area of the rhombus PQRS. [4 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Show the parallelogram is a rhombus ▶
A rhombus is a parallelogram with all sides equal. Since PQRS is already given as a parallelogram, we know opposite sides are equal (\(|\overrightarrow{PQ}| = |\overrightarrow{SR}|\) and \(|\overrightarrow{QR}| = |\overrightarrow{PS}|\)). So we just need to show that two adjacent sides are also equal: \(|\overrightarrow{PQ}| = |\overrightarrow{QR}|\).

Find the magnitudes of two adjacent sides:

$$|\overrightarrow{PQ}| = \sqrt{2^2 + 3^2 + (-4)^2} = \sqrt{4 + 9 + 16} = \sqrt{29}$$

$$|\overrightarrow{QR}| = \sqrt{5^2 + 0^2 + (-2)^2} = \sqrt{25 + 0 + 4} = \sqrt{29}$$

Since \(|\overrightarrow{PQ}| = |\overrightarrow{QR}| = \sqrt{29}\), and PQRS is a parallelogram, all four sides have equal length. Therefore PQRS is a rhombus.

M1 — Attempts both \(|\overrightarrow{PQ}|\) and \(|\overrightarrow{QR}|\)
A1 — Shows both equal \(\sqrt{29}\) and concludes "rhombus"
You must show the calculation for both magnitudes — not just state they are equal. The conclusion "therefore rhombus" (or equivalent) is required for the A1.
Part (b) Step 2: Find the diagonals of the rhombus ▶
The area of a rhombus is \(\frac{1}{2} \times d_1 \times d_2\) where \(d_1\) and \(d_2\) are the lengths of the diagonals. The diagonals of parallelogram PQRS are PR and QS. Using the parallelogram law: \(\overrightarrow{PR} = \overrightarrow{PQ} + \overrightarrow{QR}\) and \(\overrightarrow{QS} = -\overrightarrow{PQ} + \overrightarrow{PS} = \overrightarrow{QR} - \overrightarrow{PQ}\) (since \(\overrightarrow{PS} = \overrightarrow{QR}\) in a parallelogram).

Find diagonal \(\overrightarrow{PR}\):

$$\overrightarrow{PR} = \overrightarrow{PQ} + \overrightarrow{QR} = (2\mathbf{i} + 3\mathbf{j} - 4\mathbf{k}) + (5\mathbf{i} - 2\mathbf{k})$$

$$= 7\mathbf{i} + 3\mathbf{j} - 6\mathbf{k}$$

Find diagonal \(\overrightarrow{QS}\):

In a parallelogram, \(\overrightarrow{PS} = \overrightarrow{QR}\), so \(\overrightarrow{QS} = \overrightarrow{QP} + \overrightarrow{PS} = -\overrightarrow{PQ} + \overrightarrow{QR}\)

$$\overrightarrow{QS} = -(2\mathbf{i} + 3\mathbf{j} - 4\mathbf{k}) + (5\mathbf{i} - 2\mathbf{k})$$

$$= 3\mathbf{i} - 3\mathbf{j} + 2\mathbf{k}$$

M1 — Attempts both diagonal vectors PR and QS
A1 — Both correct: \(\overrightarrow{PR} = 7\mathbf{i} + 3\mathbf{j} - 6\mathbf{k}\) and \(\overrightarrow{QS} = 3\mathbf{i} - 3\mathbf{j} + 2\mathbf{k}\)
Part (b) Step 3: Calculate the area using diagonal lengths ▶
The area of a rhombus = \(\frac{1}{2} |d_1| |d_2|\) where \(d_1\) and \(d_2\) are the diagonal vectors. This works because the diagonals of a rhombus are perpendicular (we could verify: \(\overrightarrow{PR} \cdot \overrightarrow{QS} = 21 - 9 - 12 = 0\)).

Find the diagonal lengths:

$$|\overrightarrow{PR}| = \sqrt{7^2 + 3^2 + (-6)^2} = \sqrt{49 + 9 + 36} = \sqrt{94}$$

$$|\overrightarrow{QS}| = \sqrt{3^2 + (-3)^2 + 2^2} = \sqrt{9 + 9 + 4} = \sqrt{22}$$

Area of rhombus PQRS:

$$\text{Area} = \frac{1}{2} \times |\overrightarrow{PR}| \times |\overrightarrow{QS}| = \frac{1}{2} \times \sqrt{94} \times \sqrt{22}$$

Simplify: \(\sqrt{94 \times 22} = \sqrt{2068} = \sqrt{4 \times 517} = 2\sqrt{517}\)

$$= \frac{1}{2} \times 2\sqrt{517} = \sqrt{517}$$

$$\boxed{\text{Area} = \sqrt{517}}$$

dM1 — Correct method for area using \(\frac{1}{2} |\overrightarrow{PR}| \times |\overrightarrow{QS}|\)
A1 — \(\sqrt{517}\)
The diagonals of a rhombus are perpendicular, so the formula \(\frac{1}{2} d_1 d_2\) applies. This is the key insight. Alternatively, you could split into triangles and use base-times-height, but the diagonal approach is cleanest here.

How to Recognise This Question Type

Look for these signals in the question:

  • "Relative to a fixed origin O" — this is the standard preamble for all vector questions
  • Points given with position vectors using \(\mathbf{i}\), \(\mathbf{j}\), \(\mathbf{k}\) notation
  • Keywords: parallel, collinear, trapezium, rhombus, parallelogram
  • Finding \(|\overrightarrow{AB}|\) or a condition like \(|\overrightarrow{AC}| = k\)
  • "Show that..." with a geometric conclusion
  • Usually Q2-Q5 on the paper (early, moderate difficulty), or Q10-Q13 for harder geometric proofs

Key Formulae and Results

Position Vectors and Distance Formula Learn this

  • Direction vector: \(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\) (end minus start)
  • Magnitude: \(|\mathbf{a}| = \sqrt{x^2 + y^2 + z^2}\) for \(\mathbf{a} = x\mathbf{i} + y\mathbf{j} + z\mathbf{k}\)
  • Midpoint: M is midpoint of AB \(\implies \overrightarrow{OM} = \frac{1}{2}(\overrightarrow{OA} + \overrightarrow{OB})\)
  • Section formula: Point dividing AB in ratio \(m:n\) has position vector \(\frac{n\overrightarrow{OA} + m\overrightarrow{OB}}{m+n}\)

Scalar (Dot) Product Learn this

  • Parallel: \(\mathbf{a} \parallel \mathbf{b} \iff \mathbf{a} = k\mathbf{b}\) for some scalar \(k\)
  • Collinear: A, B, C collinear \(\iff \overrightarrow{AC} = \lambda \overrightarrow{AB}\) (parallel AND share a point)

Vector Equation of a Line Learn this

  • Trapezium: Exactly one pair of parallel sides
  • Rhombus: Parallelogram with all sides equal (diagonals are perpendicular)
