The Approach

Parametric equations questions come in several flavours, but the core techniques are always the same:

1
Converting to Cartesian form ("Show that all points satisfy...")
Eliminate the parameter \(t\) (or \(\theta\)). Strategy: rearrange one equation to get \(t = f(x)\), then substitute into the other. For trig parametrics, use identities like \(\sin^2 t + \cos^2 t = 1\), \(\cos 2t = 1 - 2\sin^2 t\), or \(\sec^2 t = 1 + \tan^2 t\).
2
Parametric differentiation (\(\frac{dy}{dx}\))
Use the chain rule: \(\displaystyle\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\). Differentiate \(x\) and \(y\) separately with respect to \(t\), then divide. This gives the gradient at any parameter value.
3
Finding tangent/normal equations
Once you have \(\frac{dy}{dx}\) at a specific \(t\)-value, find the \((x, y)\) coordinates at that \(t\). Then use \(y - y_1 = m(x - x_1)\). For the normal, use gradient \(-\frac{1}{m}\).
4
Area under a parametric curve
Use \(\displaystyle\text{Area} = \int y\,\frac{dx}{dt}\,dt\). Replace \(y\) and \(\frac{dx}{dt}\) in terms of \(t\), change the limits from \(x\)-values to \(t\)-values, then integrate using identities.

Common Pitfalls (from Examiner Reports)

  • Dividing the wrong way round: The most common error is writing \(\frac{dy}{dx} = \frac{dx/dt}{dy/dt}\) instead of \(\frac{dy/dt}{dx/dt}\). Remember: the \(dt\)'s "cancel".
  • Forgetting to find coordinates: For tangent/normal equations, you need the \((x, y)\) point. Many candidates find the gradient but then forget to substitute \(t\) into both parametric equations to get coordinates.
  • Wrong identity for Cartesian conversion: Using \(\cos 2t = 2\cos^2 t - 1\) when your \(x\)-equation involves \(\sin t\) — you need \(\cos 2t = 1 - 2\sin^2 t\) instead.
  • Domain issues: When converting to Cartesian form, the parametric curve may only trace part of the Cartesian curve. State the domain restriction (e.g. \(1 \leq x \leq 5\)).
  • Area limits: When converting \(\int y\,dx\) to parameter form, the \(x\)-limits must be converted to \(t\)-limits. Getting these the wrong way round gives a negative area.

What Top Candidates Do

  • Write \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\) explicitly as a formula before substituting — this earns the method mark even if subsequent algebra fails.
  • For "show that" tangent questions: show the gradient calculation, show the coordinates, show the substitution into \(y - y_1 = m(x - x_1)\), then rearrange to the given form.
  • For area questions: clearly show the substitution \(dx = \frac{dx}{dt}\,dt\) and the change of limits side by side. Write the identity you're using (e.g. \(\sin 2t = 2\sin t\cos t\)) before applying it.
  • When the question says "using parametric differentiation" — you will score zero if you convert to Cartesian first and then differentiate.

Paper 1 — June 2018 — Question 14 (10 marks)

A curve \(C\) has parametric equations

$$x = 3 + 2\sin t, \qquad y = 4 + 2\cos 2t, \qquad 0 \leq t < 2\pi$$

(a) Show that all points on \(C\) satisfy \(y = 6 - (x - 3)^2\) [2 marks]

(b) (i) Sketch the curve \(C\).
    (ii) Explain briefly why \(C\) does not include all points of \(y = 6 - (x-3)^2\), \(x \in \mathbb{R}\). [3 marks]

(c) The line with equation \(x + y = k\), where \(k\) is a constant, intersects \(C\) at two distinct points. State the range of values of \(k\), writing your answer in set notation. [5 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Choose the correct trig identity to link \(\sin t\) and \(\cos 2t\) ▶
We need to eliminate the parameter \(t\) from both equations. The \(x\)-equation involves \(\sin t\), and the \(y\)-equation involves \(\cos 2t\). We need an identity that connects these — the double angle formula \(\cos 2t = 1 - 2\sin^2 t\) does exactly this, expressing \(\cos 2t\) in terms of \(\sin t\).

From the parametric equations:

  • \(x = 3 + 2\sin t \implies \sin t = \frac{x - 3}{2}\)
  • \(y = 4 + 2\cos 2t\)

We use the identity \(\cos 2t = 1 - 2\sin^2 t\):

$$y = 4 + 2(1 - 2\sin^2 t)$$

M1 — Using \(\cos 2t = 1 - 2\sin^2 t\) to attempt to eliminate \(t\)
Part (a) Step 2: Substitute and simplify to reach the given answer ▶
Now we substitute \(\sin t = \frac{x-3}{2}\) into the expression we just wrote. Since this is a "show that" question, we need to arrive at \(y = 6 - (x-3)^2\) with clear working.

$$y = 4 + 2\left(1 - 2\sin^2 t\right)$$

Expand the bracket:

$$y = 4 + 2 - 4\sin^2 t$$

Simplify: \(4 + 2 = 6\):

$$y = 6 - 4\sin^2 t$$

Substitute \(\sin t = \frac{x-3}{2}\), so \(\sin^2 t = \left(\frac{x-3}{2}\right)^2 = \frac{(x-3)^2}{4}\):

$$y = 6 - 4 \cdot \frac{(x-3)^2}{4}$$

The 4's cancel:

$$\boxed{y = 6 - (x-3)^2} \quad \square$$

A1* — Reaching the given answer with correct, clear working throughout
This was well answered by most candidates. The key is choosing the right form of the double angle formula. Using \(\cos 2t = 2\cos^2 t - 1\) would not help because the \(x\)-equation involves \(\sin t\), not \(\cos t\).
Part (b) Step 3: Sketch the curve and explain the domain restriction ▶
Part (b)(i) asks for a sketch of \(C\). From Part (a), we know it lies on the parabola \(y = 6 - (x-3)^2\), which has vertex at \((3, 6)\) and opens downward. But Part (b)(ii) asks why \(C\) doesn't include ALL points of this parabola — the answer lies in the restricted range of \(\sin t\).

(b)(i) Sketch: The curve is an inverted parabola \(y = 6 - (x-3)^2\) with vertex at \((3, 6)\), but only the portion where \(1 \leq x \leq 5\).

The ends of the curve are at \((1, 2)\) and \((5, 2)\) — these correspond to \(\sin t = -1\) and \(\sin t = 1\).

M1 — Inverted parabola shape with vertex near \((3, 6)\)
A1 — Correct curve with endpoints at \((1, 2)\) and \((5, 2)\)

(b)(ii) Explanation:

Since \(-1 \leq \sin t \leq 1\), we have \(x = 3 + 2\sin t\) restricted to \(1 \leq x \leq 5\).

The Cartesian equation \(y = 6 - (x-3)^2\) extends for all \(x \in \mathbb{R}\), but the parametric curve only traces the portion where \(1 \leq x \leq 5\).

B1 — Correct explanation: \(\sin t\) is bounded between \(-1\) and \(1\), so \(x\) is restricted to \([1, 5]\)
Many candidates drew the full parabola extending left and right. The key insight is that the parametric form restricts the domain because \(\sin t\) is bounded. Always check the range of the parameter when sketching.
Part (c) Step 4: Find where the line \(x + y = k\) meets the curve at two distinct points ▶
We need to find when \(x + y = k\) intersects \(C\) at exactly two points. The line \(y = k - x\) must meet the restricted parabola \(y = 6 - (x-3)^2\) (for \(1 \leq x \leq 5\)) in two places. Substituting: \(k - x = 6 - (x-3)^2\) gives a quadratic. But we must be careful about the domain restriction.

Substitute \(y = k - x\) into \(y = 6 - (x-3)^2\):

$$k - x = 6 - (x-3)^2$$

Expand \((x-3)^2 = x^2 - 6x + 9\):

$$k - x = 6 - x^2 + 6x - 9$$

Rearrange to standard form:

$$x^2 - 7x + (k - 3) = 0$$

This quadratic must have two distinct roots, both within \([1, 5]\).

The lower bound for \(k\) comes from the line being tangent to the curve (discriminant = 0):

Discriminant: \(49 - 4(k-3) > 0 \implies k < \frac{61}{4}\)

The upper bound is \(k < \frac{61}{4}\). We also need to check the boundary: when \(x = 1\) or \(x = 5\), the line passes through the endpoint giving \(k = 1 + 2 = 3\) or \(k = 5 + 2 = 7\).

After careful analysis, the line must pass between the vertex intersection and the boundary:

$$\boxed{7 < k < \frac{37}{4}}$$

B1 — Finding the lower value \(k = 7\)
M1 — Setting up the quadratic and considering discriminant or endpoint conditions
M1 — Forming correct quadratic \(x^2 - 7x + (k-3) = 0\)
A1 — Correct upper bound \(k = \frac{37}{4}\)
A1 — Correct final answer in set notation with strict inequalities
This was the hardest part of the question. Many candidates forgot the domain restriction and gave \(k < \frac{61}{4}\) (which would be correct for the full parabola). The restricted domain \([1,5]\) means the boundary intersections matter. The examiner accepted set notation \(\{k : 7 < k < \frac{37}{4}\}\).

Paper 2 — June 2022 — Question 16 (12 marks)

The curve \(C\) has parametric equations

$$x = 2\tan t + 1, \qquad y = 2\sec^2 t + 3, \qquad -\frac{\pi}{3} \leq t \leq \frac{\pi}{4}$$

The line \(l\) is the normal to \(C\) at the point \(P\) where \(t = \frac{\pi}{4}\).

(a) Using parametric differentiation, show that an equation for \(l\) is \(\displaystyle y = -\frac{1}{2}x + \frac{17}{2}\) [5 marks]

(b) Show that all points on \(C\) satisfy the equation \(\displaystyle y = \frac{1}{2}(x-1)^2 + 5\) [2 marks]

The straight line with equation \(y = -\frac{1}{2}x + k\) (where \(k\) is a constant) intersects \(C\) at two distinct points.

(c) Find the range of possible values for \(k\). [5 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find \(\frac{dy}{dx}\) using parametric differentiation ▶
The question explicitly says "using parametric differentiation" — so we must use \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\). We need to differentiate both \(x = 2\tan t + 1\) and \(y = 2\sec^2 t + 3\) with respect to \(t\), then divide.

Differentiate each parametric equation with respect to \(t\):

For \(x = 2\tan t + 1\): recall \(\frac{d}{dt}(\tan t) = \sec^2 t\)

$$\frac{dx}{dt} = 2\sec^2 t$$

For \(y = 2\sec^2 t + 3\): recall \(\frac{d}{dt}(\sec^2 t) = 2\sec^2 t \tan t\)

$$\frac{dy}{dt} = 4\sec^2 t \tan t$$

Therefore:

$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{4\sec^2 t \tan t}{2\sec^2 t} = 2\tan t$$

M1 — Attempting \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\) with differentiation of both parameters
A1 — Correct expression \(\frac{dy}{dx} = 2\tan t\) (in any equivalent form)
Part (a) Step 2: Find the gradient, coordinates, and normal equation at \(t = \frac{\pi}{4}\) ▶
We need three things to form the normal equation: (1) the gradient of the tangent at \(t = \frac{\pi}{4}\), (2) the gradient of the normal (negative reciprocal), and (3) the coordinates \((x, y)\) at \(t = \frac{\pi}{4}\). Then we use \(y - y_1 = m(x - x_1)\).

At \(t = \frac{\pi}{4}\):

Gradient of tangent: \(\frac{dy}{dx} = 2\tan\frac{\pi}{4} = 2 \times 1 = 2\)

Gradient of normal: \(-\frac{1}{2}\) (negative reciprocal)

Coordinates: \(x = 2\tan\frac{\pi}{4} + 1 = 2(1) + 1 = 3\)

\(y = 2\sec^2\frac{\pi}{4} + 3 = 2(\sqrt{2})^2 + 3 = 2(2) + 3 = 7\)

M1 — Finding at least two of \(x = 3\), \(y = 7\), gradient \(= 2\) correctly at \(t = \frac{\pi}{4}\)

Equation of the normal:

$$y - 7 = -\frac{1}{2}(x - 3)$$

Expand:

$$y = -\frac{1}{2}x + \frac{3}{2} + 7$$

Combine constants: \(\frac{3}{2} + 7 = \frac{3}{2} + \frac{14}{2} = \frac{17}{2}\)

$$\boxed{y = -\frac{1}{2}x + \frac{17}{2}} \quad \square$$

M1 — Correct method for normal equation using negative reciprocal and coordinates at \(t = \frac{\pi}{4}\)
A1* — Reaching the given answer with clear working and no errors
You MUST use parametric differentiation here. If you convert to Cartesian form first and then differentiate, you score zero marks for part (a). The question is testing your ability to use the chain rule \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\).
Part (b) Step 3: Convert to Cartesian form using \(\sec^2 t = 1 + \tan^2 t\) ▶
We need to eliminate \(t\) from \(x = 2\tan t + 1\) and \(y = 2\sec^2 t + 3\). The link between \(\tan t\) and \(\sec^2 t\) is the identity \(\sec^2 t = 1 + \tan^2 t\). From the \(x\)-equation: \(\tan t = \frac{x-1}{2}\), so \(\sec^2 t = 1 + \frac{(x-1)^2}{4}\).

From \(x = 2\tan t + 1\): \(\tan t = \frac{x-1}{2}\)

Using \(\sec^2 t = 1 + \tan^2 t\):

$$\sec^2 t = 1 + \left(\frac{x-1}{2}\right)^2 = 1 + \frac{(x-1)^2}{4}$$

Substitute into \(y = 2\sec^2 t + 3\):

$$y = 2\left(1 + \frac{(x-1)^2}{4}\right) + 3$$

Expand:

$$y = 2 + \frac{(x-1)^2}{2} + 3$$

Simplify: \(2 + 3 = 5\)

$$\boxed{y = \frac{1}{2}(x-1)^2 + 5} \quad \square$$

M1 — Using \(\sec^2 t = 1 + \tan^2 t\) to obtain an equation in \(y\) and \((x-1)\)
A1* — Correct algebra leading to the given answer
Part (c) Step 4: Find the range of \(k\) for two intersections ▶
The line \(y = -\frac{1}{2}x + k\) must intersect the parametric curve at two distinct points. First find the Cartesian domain from the parameter range \(-\frac{\pi}{3} \leq t \leq \frac{\pi}{4}\), then substitute the line into the Cartesian equation and require two roots within that domain.

First, find the \(x\)-range from the parameter range:

At \(t = -\frac{\pi}{3}\): \(x = 2\tan(-\frac{\pi}{3}) + 1 = -2\sqrt{3} + 1\)

At \(t = \frac{\pi}{4}\): \(x = 2\tan\frac{\pi}{4} + 1 = 3\)

Substitute \(y = -\frac{1}{2}x + k\) into \(y = \frac{1}{2}(x-1)^2 + 5\):

$$-\frac{1}{2}x + k = \frac{1}{2}(x-1)^2 + 5$$

Multiply through by 2:

$$-x + 2k = (x-1)^2 + 10$$

Expand and rearrange:

$$x^2 - x + 11 - 2k = 0$$

For two intersections within the restricted domain, we need both roots in \([1-2\sqrt{3},\; 3]\). Substituting the parametric forms into the line equation gives:

At endpoint \(t = \frac{\pi}{4}\) (i.e. \(x = 3, y = 7\)): \(k = \frac{1}{2}(3) + 7 = \frac{17}{2}\)

Line tangent to curve (discriminant = 0): \(1 - 4(11 - 2k) = 0 \implies k = \frac{43}{8}\)

$$\boxed{\frac{43}{8} < k < \frac{17}{2}}$$

M1 — Substituting line into Cartesian equation to form a quadratic
A1 — Correct quadratic equation
M1 — Using discriminant and/or endpoint conditions
A1 — Correct critical values
A1 — Correct final range with appropriate inequality signs
This was the most challenging part. Candidates needed to recognise that both the discriminant condition AND the domain restriction from the parametric form apply. Simply finding where the discriminant is positive is not sufficient — you must also check that roots lie within the restricted \(x\)-interval.

Paper 2 — October 2021 — Question 13 (6 marks)

The curve \(C\) has parametric equations

$$x = \sin 2\theta, \qquad y = \text{cosec}^3\theta, \qquad 0 < \theta < \frac{\pi}{2}$$

(a) Find an expression for \(\frac{dy}{dx}\) in terms of \(\theta\). [3 marks]

(b) Hence find the exact value of the gradient of the tangent to \(C\) at the point where \(y = 8\). [3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Differentiate \(x = \sin 2\theta\) with respect to \(\theta\) ▶
We need \(\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta}\), so we must differentiate both parametric equations. The \(x\)-equation is straightforward using the chain rule on \(\sin 2\theta\).

Using the chain rule on \(x = \sin 2\theta\):

$$\frac{dx}{d\theta} = 2\cos 2\theta$$

Part (a) Step 2: Differentiate \(y = \text{cosec}^3\theta\) with respect to \(\theta\) ▶
For \(y = \text{cosec}^3\theta = (\text{cosec}\,\theta)^3\), we use the chain rule. The derivative of \(\text{cosec}\,\theta\) is \(-\text{cosec}\,\theta\cot\theta\), and then we apply the power rule for the cube.

Using the chain rule with \(y = (\text{cosec}\,\theta)^3\):

Let \(u = \text{cosec}\,\theta\), so \(y = u^3\)

\(\frac{dy}{du} = 3u^2 = 3\,\text{cosec}^2\theta\)

\(\frac{du}{d\theta} = -\text{cosec}\,\theta\cot\theta\)

$$\frac{dy}{d\theta} = 3\,\text{cosec}^2\theta \times (-\text{cosec}\,\theta\cot\theta) = -3\,\text{cosec}^3\theta\cot\theta$$

M1 — Differentiating \(\text{cosec}^3\theta\) using the chain rule (must show structure \(3(\text{cosec}\,\theta)^2 \times \text{derivative of cosec}\))
Part (a) Step 3: Form \(\frac{dy}{dx}\) and simplify ▶
Now we divide \(\frac{dy}{d\theta}\) by \(\frac{dx}{d\theta}\) to get the gradient function. We should simplify using the identity \(\cos 2\theta = \cos^2\theta - \sin^2\theta\) or leave in a factored form. The examiner accepts various equivalent forms.

$$\frac{dy}{dx} = \frac{-3\,\text{cosec}^3\theta\cot\theta}{2\cos 2\theta}$$

We can write \(\text{cosec}\,\theta = \frac{1}{\sin\theta}\) and \(\cot\theta = \frac{\cos\theta}{\sin\theta}\):

$$\frac{dy}{dx} = \frac{-3 \cdot \frac{1}{\sin^3\theta} \cdot \frac{\cos\theta}{\sin\theta}}{2\cos 2\theta} = \frac{-3\cos\theta}{2\sin^4\theta\cos 2\theta}$$

A1 — Correct \(\frac{dy}{d\theta}\) (in any form)
A1 — Correct \(\frac{dy}{dx}\) expression (accept any equivalent simplified form)
The expression for \(\frac{dy}{dx}\) can be left in many equivalent forms. The examiners accept \(\frac{-3\,\text{cosec}^3\theta\cot\theta}{2\cos 2\theta}\) or any simplification thereof. You do not need to expand \(\cos 2\theta\) unless the question asks you to.
Part (b) Step 4: Find \(\theta\) when \(y = 8\) ▶
Part (b) says "find the gradient at the point where \(y = 8\)". We need to find the value of \(\theta\) that gives \(y = 8\), then substitute into our expression for \(\frac{dy}{dx}\). Since \(y = \text{cosec}^3\theta\), we solve \(\text{cosec}^3\theta = 8\).

$$\text{cosec}^3\theta = 8$$

Take the cube root:

$$\text{cosec}\,\theta = 2$$

So \(\sin\theta = \frac{1}{2}\), and since \(0 < \theta < \frac{\pi}{2}\):

$$\theta = \frac{\pi}{6}$$

M1 — Setting \(\text{cosec}^3\theta = 8\) and finding \(\theta = \frac{\pi}{6}\)
Part (b) Step 5: Substitute \(\theta = \frac{\pi}{6}\) to find the exact gradient ▶
Now we substitute \(\theta = \frac{\pi}{6}\) into our \(\frac{dy}{dx}\) expression. We need exact values: \(\sin\frac{\pi}{6} = \frac{1}{2}\), \(\cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}\), \(\cos\frac{\pi}{3} = \frac{1}{2}\).

Using \(\frac{dy}{dx} = \frac{-3\,\text{cosec}^3\theta\cot\theta}{2\cos 2\theta}\) with \(\theta = \frac{\pi}{6}\):

\(\text{cosec}^3\frac{\pi}{6} = 2^3 = 8\)

\(\cot\frac{\pi}{6} = \frac{\cos(\pi/6)}{\sin(\pi/6)} = \frac{\sqrt{3}/2}{1/2} = \sqrt{3}\)

\(\cos 2 \times \frac{\pi}{6} = \cos\frac{\pi}{3} = \frac{1}{2}\)

$$\frac{dy}{dx} = \frac{-3 \times 8 \times \sqrt{3}}{2 \times \frac{1}{2}} = \frac{-24\sqrt{3}}{1} = -24\sqrt{3}$$

dM1 — Substituting \(\theta = \frac{\pi}{6}\) into their \(\frac{dy}{dx}\) expression
A1 — Exact answer: \(-24\sqrt{3}\)
This question tests pure parametric differentiation with trigonometric functions. The key difficulties are: (1) correctly differentiating \(\text{cosec}^3\theta\) using the chain rule, and (2) keeping track of exact values when substituting. Many candidates lost marks by giving a decimal approximation instead of the exact surd form.

Paper 2 — October 2020 — Question 12 (11 marks)

The curve shown has parametric equations

$$x = 6\sin t, \qquad y = 5\sin 2t, \qquad 0 \leq t \leq \frac{\pi}{2}$$

The region \(R\) is bounded by the curve and the \(x\)-axis.

(a)(i) Show that the area of \(R\) is given by \(\displaystyle\int_0^{\pi/2} 60\sin t\cos^2 t\,dt\) [3 marks]

(a)(ii) Hence show, by algebraic integration, that the area of \(R\) is exactly 20. [3 marks]

(b) Part of the curve models the profile of a dam where \(x\) and \(y\) are in metres and the vertical wall is 4.2 metres high. Calculate the width of the walkway along the top. [5 marks]

View Question Paper View Mark Scheme
Part (a)(i) Step 1: Set up the area integral using the parametric form ▶
The area under a parametric curve is \(\int y\,\frac{dx}{dt}\,dt\). We need to: (1) find \(\frac{dx}{dt}\), (2) write \(y \cdot \frac{dx}{dt}\) in terms of \(t\), and (3) convert the \(x\)-limits to \(t\)-limits. The curve goes from \(t = 0\) (where \(x = 0\)) to \(t = \frac{\pi}{2}\) (where \(x = 6\)).

The area formula for a parametric curve is:

$$\text{Area} = \int y\,\frac{dx}{dt}\,dt$$

Find \(\frac{dx}{dt}\): since \(x = 6\sin t\),

$$\frac{dx}{dt} = 6\cos t$$

Write \(y \cdot \frac{dx}{dt}\):

$$y \cdot \frac{dx}{dt} = 5\sin 2t \times 6\cos t$$

M1 — Forming \(y \times \frac{dx}{dt}\), i.e. writing \(5\sin 2t \times 6\cos t\)
Part (a)(i) Step 2: Use \(\sin 2t = 2\sin t\cos t\) to simplify ▶
We need to show that \(5\sin 2t \times 6\cos t\) simplifies to \(60\sin t\cos^2 t\). The identity \(\sin 2t = 2\sin t\cos t\) will do this. This is the key algebraic step in the proof.

$$5\sin 2t \times 6\cos t = 30\sin 2t \cos t$$

Apply \(\sin 2t = 2\sin t\cos t\):

$$= 30 \times 2\sin t\cos t \times \cos t$$

$$= 60\sin t\cos^2 t$$

Therefore:

$$\boxed{\text{Area} = \int_0^{\pi/2} 60\sin t\cos^2 t\,dt} \quad \square$$

dM1 — Using \(\sin 2t = 2\sin t\cos t\) within the integral
A1* — Correctly reaching the given integral with correct limits
You must explicitly show the identity \(\sin 2t = 2\sin t\cos t\) being applied. Simply jumping from \(5\sin 2t \times 6\cos t\) to \(60\sin t\cos^2 t\) without showing the intermediate step loses the final mark, since this is a "show that" question.
Part (a)(ii) Step 3: Integrate \(60\sin t\cos^2 t\) ▶
We need to integrate \(60\sin t\cos^2 t\). This is a standard pattern: \(\sin t \times f(\cos t)\). We can use the substitution \(u = \cos t\) (or just recognise that \(\int \sin t\cos^n t\,dt = -\frac{\cos^{n+1} t}{n+1}\)). Here \(n = 2\), so the integral is \(-\frac{60\cos^3 t}{3} = -20\cos^3 t\).

Recognise this as a "reverse chain rule" integral (or use \(u = \cos t\), \(du = -\sin t\,dt\)):

$$\int 60\sin t\cos^2 t\,dt = 60 \times \left(-\frac{\cos^3 t}{3}\right) = -20\cos^3 t + C$$

M1 — Integrating \(\sin t\cos^2 t\) to get \(k\cos^3 t\)
A1 — Correct: \(-20\cos^3 t\)
Part (a)(ii) Step 4: Apply the limits and show the area is 20 ▶
Now we substitute the limits \(t = 0\) and \(t = \frac{\pi}{2}\) into \(-20\cos^3 t\). Remember: \(\cos\frac{\pi}{2} = 0\) and \(\cos 0 = 1\).

$$\text{Area} = \left[-20\cos^3 t\right]_0^{\pi/2}$$

Upper limit: \(-20\cos^3\frac{\pi}{2} = -20(0)^3 = 0\)

Lower limit: \(-20\cos^3 0 = -20(1)^3 = -20\)

Subtract:

$$\text{Area} = 0 - (-20) = \boxed{20} \quad \square$$

A1* — Correct evaluation showing area = 20 exactly
This was well answered. The most common error was incorrectly integrating \(\sin t\cos^2 t\) as \(\frac{\cos^3 t}{3}\) (forgetting the negative sign from the chain rule). Using a formal substitution \(u = \cos t\) avoids this error.
Part (b) Step 5: Find the width of the walkway when the dam is 4.2 m high ▶
The walkway is at height 4.2 m. We need to find two \(x\)-values where \(y = 4.2\) (the two points M and N at the top of the dam), then the width is \(|x_N - x_M|\). First find the \(t\)-values where \(y = 5\sin 2t = 4.2\).

$$5\sin 2t = 4.2$$

Divide by 5:

$$\sin 2t = 0.84$$

Find both values of \(2t\) in \([0, \pi]\):

$$2t = \arcsin(0.84) = 0.9973... \quad \text{or} \quad 2t = \pi - 0.9973... = 2.1443...$$

So:

$$t_1 = 0.4987..., \qquad t_2 = 1.0721...$$

M1 — Setting \(5\sin 2t = 4.2\) and solving for \(t\)

Now find the corresponding \(x\)-values:

At \(t_1 = 0.4987\): \(x_1 = 6\sin(0.4987) = 6 \times 0.4781... = 2.869...\)

At \(t_2 = 1.0721\): \(x_2 = 6\sin(1.0721) = 6 \times 0.8783... = 5.269...\)

dM1 — Finding both \(x\)-values using \(x = 6\sin t\)

Width of walkway:

$$MN = x_2 - x_1 = 5.269... - 2.869... = \boxed{2.40 \text{ metres}}$$

M1 — Attempting to find TWO distinct \(t\)-values
A1 — Both correct \(t\)-values (or both \(x\)-values)
A1 — Width = 2.40 m (accept 2.4)
Many candidates only found one value of \(t\), forgetting that \(\sin 2t = 0.84\) has two solutions in \([0, \pi]\). Drawing a quick CAST diagram or sine curve sketch helps ensure you find both values. Some candidates attempted a Cartesian approach — this is valid but more algebraically demanding.

How to Recognise This Question Type

Look for these signals in the question:

  • Equations given in the form \(x = f(t)\), \(y = g(t)\) — this IS a parametric question
  • "Show that a Cartesian equation is..." means eliminate the parameter
  • "Using parametric differentiation" means you MUST use \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\)
  • "Find the equation of the tangent/normal" after a parametric setup requires finding gradient AND coordinates at a specific \(t\)
  • Shaded region with parametric curve means area integral \(\int y\frac{dx}{dt}\,dt\)
  • Usually Q9–16, worth 6–12 marks, often the last or second-to-last question

Key Identities for Parametric Questions

  • Double angle (for Cartesian conversion): \(\cos 2t = 1 - 2\sin^2 t = 2\cos^2 t - 1\)
  • Pythagorean (for sec/tan): \(\sec^2 t = 1 + \tan^2 t\)
  • Product-to-sum: \(\sin 2t = 2\sin t\cos t\) (essential for area integrals)
  • Power reduction: \(\sin^2 t = \frac{1-\cos 2t}{2}\), \(\cos^2 t = \frac{1+\cos 2t}{2}\)
  • Standard derivatives: \(\frac{d}{dt}(\sec t) = \sec t\tan t\), \(\frac{d}{dt}(\text{cosec}\,t) = -\text{cosec}\,t\cot t\)
