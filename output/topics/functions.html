The Approach

How to tackle any Edexcel functions question systematically:

1
Identify the type
Is it asking for a composite (fg or gf), an inverse (\(f^{-1}\)), a domain/range, or a transformation? The notation tells you: \(fg(x)\) means "apply g first, then f"; \(f^{-1}\) means "undo f"; transformations involve \(f(x+a)\), \(af(x)\), etc.
2
For composites: work from the inside out
For \(fg(x)\): substitute the entire expression for \(g(x)\) wherever you see \(x\) in \(f\). For a numerical value like \(fg(2)\): find \(g(2)\) first, then substitute that result into \(f\). Never apply \(f\) first — the order matters.
3
For inverses: swap and solve
Write \(y = f(x)\), then rearrange to make \(x\) the subject. Finally replace \(x\) with \(f^{-1}(x)\) and \(y\) with \(x\). Always state the domain of \(f^{-1}\) — it equals the range of \(f\).
4
For transformations: apply to coordinates
\(f(x-a)\): translate \(a\) units right. \(f(x)+a\): translate \(a\) units up. \(af(x)\): stretch vertically by factor \(a\). \(f(ax)\): stretch horizontally by factor \(\frac{1}{a}\). \(f(-x)\): reflect in y-axis. \(-f(x)\): reflect in x-axis. Apply to specific point coordinates.

Common Pitfalls (from Examiner Reports)

  • Wrong order for composites: \(fg(x)\) means "g first, then f" — but many candidates compute \(f\) first. The function closest to the \(x\) is applied first. Think of it as \(f(g(x))\) — work from the innermost bracket outward.
  • Forgetting to restrict the domain for the inverse: When finding \(f^{-1}(x)\), you must state its domain. The domain of \(f^{-1}\) = range of \(f\). Without this you lose the final mark every time.
  • Incorrect transformation direction: \(f(x+2)\) is a translation 2 units LEFT (not right). \(f(2x)\) is a stretch of factor \(\frac{1}{2}\) horizontally (not 2). The "inside" changes are counterintuitive — they go in the opposite direction to what you might expect.
  • Not simplifying composites: When asked for \(gf(x)\) as a "simplified fraction", you must combine terms fully. Leaving an unsimplified expression loses the A mark.
  • Range vs Domain confusion: Range is the set of output values (on the y-axis). Domain is the set of allowed inputs (on the x-axis). Writing "range: \(x \geq 3\)" instead of "\(f(x) \geq 3\)" or "\(y \geq 3\)" loses the mark.

What Top Candidates Do

  • For composites with specific values (e.g. "find fg(2)"): evaluate the inner function at that value first, then substitute the numerical result into the outer function. Write both steps clearly.
  • For inverses of rational functions: cross-multiply first, then collect all terms containing the variable on one side and factorise.
  • Always use correct function notation: write \(g^{-1}(x) = \ldots\) not just \(y = \ldots\). State the domain using set notation or inequalities.
  • For transformations: state the type (translation/stretch/reflection) AND the specifics (direction and magnitude/vector). Both are needed for full marks.

Paper 2 — June 2018 — Question 1 (6 marks)

$$g(x) = \frac{2x + 5}{x - 3}, \quad x \in \mathbb{R}, \quad x \geq 5$$

(a) Find \(gg(5)\). [2 marks]

(b) State the range of \(g\). [1 mark]

(c) Find \(g^{-1}(x)\), stating its domain. [3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find g(5) ▶
\(gg(5)\) means \(g(g(5))\) — apply g twice. We work from the inside out: first find \(g(5)\), then substitute that result back into \(g\). This is a composite function evaluation at a specific numerical value.

Substitute \(x = 5\) into \(g(x) = \frac{2x+5}{x-3}\):

$$g(5) = \frac{2(5) + 5}{5 - 3} = \frac{15}{2} = 7.5$$

M1 — Attempting g(5) and substituting the result into g
Part (a) Step 2: Find g(g(5)) = g(7.5) ▶
Now we take the output from Step 1 (\(7.5\)) and feed it back into \(g\). This is the second application of the function, completing the composite \(gg(5) = g(g(5)) = g(7.5)\).

Substitute \(x = 7.5\) into \(g(x)\):

$$gg(5) = g(7.5) = \frac{2(7.5) + 5}{7.5 - 3} = \frac{20}{4.5} = \frac{40}{9}$$

$$\boxed{gg(5) = \frac{40}{9}}$$

A1 — Correct answer: \(\frac{40}{9}\) (accept \(4.\overline{4}\))
An exact answer is required here. Give \(\frac{40}{9}\) or \(4\frac{4}{9}\). Writing "4.4" without the recurring dot or fraction loses the A1 mark.
Part (b) Step 3: State the range of g ▶
The range is the set of all possible output values. For a rational function like \(\frac{2x+5}{x-3}\), as \(x\) gets very large the function approaches a horizontal asymptote. We can find this by dividing: the leading coefficients give \(\frac{2}{1} = 2\). With domain \(x \geq 5\), we substitute the boundary: \(g(5) = 7.5\). As \(x \to \infty\), \(g(x) \to 2\) from above. So the range is \(2 < g(x) \leq 7.5\).

As \(x \to \infty\): \(g(x) = \frac{2x+5}{x-3} \to \frac{2x}{x} = 2\) (from above, never reaching 2).

At \(x = 5\): \(g(5) = \frac{15}{2} = 7.5\) (this is the maximum output).

$$\boxed{2 < g(x) \leq \frac{15}{2}}$$

B1 — Correct range: \(2 < y \leq \frac{15}{2}\)
The range must use correct notation. Accept \(2 < g(x) \leq 7.5\) or set notation. Writing "\(2 \leq y \leq 7.5\)" loses the mark because \(g(x)\) never equals 2.
Part (c) Step 4: Find \(g^{-1}(x)\) — set up and cross-multiply ▶
To find the inverse, we write \(y = g(x)\) and rearrange to make \(x\) the subject. For rational functions, the key technique is cross-multiplication followed by collecting \(x\)-terms on one side and factorising. At the end we swap \(x\) and \(y\) (or equivalently, write \(g^{-1}(x) = \ldots\)).

Let \(y = \frac{2x+5}{x-3}\). Cross-multiply:

$$y(x - 3) = 2x + 5$$

Expand the left side:

$$yx - 3y = 2x + 5$$

M1 — Cross-multiplication followed by attempt to collect terms in x
Part (c) Step 5: Collect \(x\)-terms, factorise, and solve ▶
We need \(x\) on its own. Since \(x\) appears in two terms (\(yx\) and \(2x\)), we move them to the same side and factor out \(x\). Then divide to isolate \(x\). This is the standard technique for inverting any rational function.

Collect all terms containing \(x\) on one side:

$$yx - 2x = 3y + 5$$

Factor out \(x\):

$$x(y - 2) = 3y + 5$$

Divide both sides by \((y - 2)\):

$$x = \frac{3y + 5}{y - 2}$$

Now swap \(x\) and \(y\) (or simply write using inverse notation):

$$\boxed{g^{-1}(x) = \frac{3x + 5}{x - 2}, \quad 2 < x \leq \frac{15}{2}}$$

M1 — Complete method to find inverse (factorise and divide)
A1ft — Correct inverse with domain stated correctly (domain of \(g^{-1}\) = range of \(g\))
The domain of \(g^{-1}\) must be stated. It equals the range of \(g\) from part (b). Correct notation is required: "\(x \in \mathbb{R}, 2 < x \leq \frac{15}{2}\)" or equivalent. Simply writing the expression without domain loses the final mark.

Paper 2 — October 2021 — Question 2 (5 marks)

The functions f and g are defined by:

$$f(x) = 7 - 2x^2, \quad x \in \mathbb{R}$$

$$g(x) = \frac{3x}{5x - 1}, \quad x \in \mathbb{R}, \quad x \neq \frac{1}{5}$$

(a) State the range of \(f\). [1 mark]

(b) Find \(gf(1.8)\). [2 marks]

(c) Find \(g^{-1}(x)\). [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: State the range of f ▶
\(f(x) = 7 - 2x^2\) is a downward-opening parabola (negative coefficient of \(x^2\)). Its maximum value occurs at the vertex. Since the domain is all reals, \(x^2 \geq 0\), so \(-2x^2 \leq 0\), meaning \(f(x) = 7 - 2x^2 \leq 7\). The maximum is 7 (when \(x=0\)) and \(f(x)\) decreases without bound as \(|x| \to \infty\).

\(f(x) = 7 - 2x^2\). Since \(x^2 \geq 0\) for all real \(x\):

Maximum of \(f\) is at \(x = 0\): \(f(0) = 7\)

As \(|x| \to \infty\), \(f(x) \to -\infty\)

$$\boxed{f(x) \leq 7 \quad \text{(or equivalently, range of } f \text{ is } (-\infty, 7])}$$

B1 — Correct range: \(y \leq 7\) or \(f(x) \leq 7\)
Accept any correct notation: \(f(x) \leq 7\), \(y \leq 7\), \(\{y \in \mathbb{R}: y \leq 7\}\), \((-\infty, 7]\). Do not accept \(x \leq 7\) — range must be about outputs.
Part (b) Step 2: Find gf(1.8) — evaluate f(1.8) first ▶
\(gf(1.8)\) means \(g(f(1.8))\) — apply \(f\) first (it is closest to the \(x\)), then apply \(g\) to the result. This is the key: for numerical composites, always compute the inner function's value first, then feed that number into the outer function.

First, find \(f(1.8)\):

$$f(1.8) = 7 - 2(1.8)^2 = 7 - 2(3.24) = 7 - 6.48 = 0.52$$

M1 — Finding f(1.8) and substituting into g
Part (b) Step 3: Substitute into g to complete gf(1.8) ▶
We now have \(f(1.8) = 0.52\). The composite \(gf(1.8) = g(0.52)\). We substitute 0.52 into the formula for \(g\).

Now find \(g(0.52)\):

$$gf(1.8) = g(0.52) = \frac{3(0.52)}{5(0.52) - 1} = \frac{1.56}{2.6 - 1} = \frac{1.56}{1.6}$$

Simplify:

$$= \frac{156}{160} = \frac{39}{40} = 0.975$$

$$\boxed{gf(1.8) = \frac{39}{40}}$$

A1 — Correct answer: \(\frac{39}{40}\) or 0.975
Part (c) Step 4: Find \(g^{-1}(x)\) ▶
To find the inverse of \(g(x) = \frac{3x}{5x-1}\), we set \(y = \frac{3x}{5x-1}\) and rearrange to make \(x\) the subject. Since \(x\) appears in both numerator and denominator, cross-multiply first, then collect all \(x\)-terms together and factorise.

Let \(y = \frac{3x}{5x - 1}\). Cross-multiply:

$$y(5x - 1) = 3x$$

Expand:

$$5xy - y = 3x$$

Collect terms in \(x\) on one side:

$$5xy - 3x = y$$

Factor out \(x\):

$$x(5y - 3) = y$$

Divide by \((5y - 3)\):

$$x = \frac{y}{5y - 3}$$

Swap \(x\) and \(y\):

$$\boxed{g^{-1}(x) = \frac{x}{5x - 3}}$$

M1 — Cross-multiplying and attempting to factorise out x
A1 — Correct expression for the inverse
Accept equivalent forms such as \(\frac{-x}{3-5x}\) or \(\frac{1}{5} + \frac{3}{25x-15}\). The domain is not explicitly required in this question, but if given it must be correct.

Paper 1 — June 2019 — Question 5 (10 marks)

$$f(x) = 2x^2 + 4x + 9, \quad x \in \mathbb{R}$$

(a) Write \(f(x)\) in the form \(a(x+b)^2 + c\), where \(a\), \(b\) and \(c\) are integers to be found. [3 marks]

(b) Sketch the curve with equation \(y = f(x)\) showing any points of intersection with the coordinate axes and the coordinates of any turning point. [3 marks]

(c)(i) Describe fully the transformation that maps the curve with equation \(y = f(x)\) onto the curve with equation \(y = g(x)\) where \(g(x) = 2(x-2)^2 + 4x - 3\). [2 marks]

(c)(ii) Find the range of the function \(h(x) = \frac{21}{2x^2 + 4x + 9}\), \(x \in \mathbb{R}\). [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Complete the square on \(f(x) = 2x^2 + 4x + 9\) ▶
We need \(f(x)\) in vertex form \(a(x+b)^2 + c\). Since the leading coefficient is 2 (not 1), we factor it out from the \(x\)-terms first, then complete the square inside the bracket. This form directly gives us the minimum value \(c\) (at \(x = -b\)), which we will need for parts (b) and (c).

$$f(x) = 2x^2 + 4x + 9$$

Factor out the coefficient of \(x^2\) from the first two terms:

$$= 2(x^2 + 2x) + 9$$

Complete the square inside the bracket: \(x^2 + 2x = (x+1)^2 - 1\):

$$= 2\left[(x+1)^2 - 1\right] + 9$$

Expand the 2:

$$= 2(x+1)^2 - 2 + 9$$

$$= 2(x+1)^2 + 7$$

$$\boxed{a = 2, \quad b = 1, \quad c = 7}$$

B1 — States or implies \(a = 2\)
M1 — Correct method for completing the square with \(a \neq 1\)
A1 — \(2(x+1)^2 + 7\)
Part (b) Step 2: Sketch the curve ▶
From the completed square form \(2(x+1)^2 + 7\), we can read off: the vertex (minimum point) is at \((-1, 7)\), the parabola opens upward (\(a = 2 > 0\)), and the minimum \(y\)-value is 7 (above the \(x\)-axis, so no \(x\)-intercepts). The \(y\)-intercept is \(f(0) = 9\).

Key features of the sketch:

  • U-shaped parabola (since \(a = 2 > 0\))
  • Minimum (turning point) at \((-1, 7)\)
  • \(y\)-intercept at \((0, 9)\)
  • No \(x\)-intercepts (minimum is above the \(x\)-axis)
B1 — U-shaped curve in a reasonable position, not through (0,0)
B1 — \(y\)-intercept at (0, 9)
B1ft — Minimum at \((-1, 7)\), shown in the correct quadrant
Part (c)(i) Step 3: Identify the transformation from f(x) to g(x) ▶
We need to compare \(f(x) = 2(x+1)^2 + 7\) with \(g(x) = 2(x-2)^2 + 4x - 3\). The trick is to write \(g(x)\) in the same completed square form. Expanding \(g(x) = 2(x-2)^2 + 4x - 3 = 2(x^2 - 4x + 4) + 4x - 3 = 2x^2 - 8x + 8 + 4x - 3 = 2x^2 - 4x + 5\). Then complete the square: \(2(x^2 - 2x) + 5 = 2[(x-1)^2 - 1] + 5 = 2(x-1)^2 + 3\). Comparing \(f(x) = 2(x+1)^2 + 7\) with \(g(x) = 2(x-1)^2 + 3\): the vertex has moved from \((-1, 7)\) to \((1, 3)\).

First, simplify \(g(x)\):

$$g(x) = 2(x-2)^2 + 4x - 3 = 2(x^2 - 4x + 4) + 4x - 3$$

$$= 2x^2 - 8x + 8 + 4x - 3 = 2x^2 - 4x + 5$$

Complete the square:

$$= 2(x^2 - 2x) + 5 = 2[(x-1)^2 - 1] + 5 = 2(x-1)^2 + 3$$

Compare: \(f(x) = 2(x+1)^2 + 7\) has vertex \((-1, 7)\)

Compare: \(g(x) = 2(x-1)^2 + 3\) has vertex \((1, 3)\)

The vertex moves from \((-1, 7)\) to \((1, 3)\): that is 2 right and 4 down.

$$\boxed{\text{Translation by the vector } \begin{pmatrix} 2 \\ -4 \end{pmatrix}}$$

M1 — Deduces a translation with one correct component
A1 — Correct: "Translation by \(\begin{pmatrix}2\\-4\end{pmatrix}\)"
You must say "translation" (or "shift"/"move") AND give the vector \(\begin{pmatrix}2\\-4\end{pmatrix}\). Writing "2 right and 4 down" without the word "translation" or without the vector loses marks. Do NOT describe it as two separate transformations — it is one single translation.
Part (c)(ii) Step 4: Find the range of \(h(x) = \frac{21}{f(x)}\) ▶
We have \(h(x) = \frac{21}{2x^2 + 4x + 9} = \frac{21}{f(x)}\). From part (a), \(f(x) = 2(x+1)^2 + 7 \geq 7\) for all \(x\). Since \(f(x)\) is always at least 7, the fraction \(\frac{21}{f(x)}\) is at most \(\frac{21}{7} = 3\). And since \(f(x)\) is always positive and can grow without bound, \(h(x)\) is always positive but approaches 0 (never reaching it). So the range of \(h\) is \(0 < h(x) \leq 3\).

From part (a): \(f(x) = 2(x+1)^2 + 7 \geq 7\) for all \(x \in \mathbb{R}\).

Therefore the denominator of \(h\) is always at least 7:

Maximum value of \(h\): occurs when denominator is smallest (= 7):

$$h_{\max} = \frac{21}{7} = 3$$

As \(|x| \to \infty\), \(f(x) \to \infty\), so \(h(x) \to 0^+\) (approaches 0, never reaches it):

$$\boxed{0 < h(x) \leq 3}$$

M1 — Recognising that the maximum of \(h\) occurs at the minimum of \(f\), and calculating \(\frac{21}{7} = 3\)
A1ft — Correct range: \(0 < h(x) \leq 3\)
The upper bound is \(\leq 3\) (achieved at \(x = -1\)). The lower bound is strictly \(> 0\) (never reached). Writing \(0 \leq h(x) \leq 3\) loses the mark because \(h(x)\) can never actually equal 0.

Paper 1 — June 2024 — Question 8 (11 marks)

The functions f and g are defined by:

$$f(x) = 4 - 3x^2, \quad x \in \mathbb{R}$$

$$g(x) = \frac{5}{2x - 9}, \quad x \in \mathbb{R}, \quad x \neq \frac{9}{2}$$

(a) Find \(fg(2)\). [2 marks]

(b) Find \(g^{-1}(x)\), stating its domain. [3 marks]

(c)(i) Find \(gf(x)\), giving your answer as a simplified fraction. [2 marks]

(c)(ii) Deduce the range of \(gf(x)\). [1 mark]

(d) Given \(h(x) = 2x^2 - 6x + k\), find the range of values of \(k\) for which \(f(x) = h(x)\) has no real solutions. [3 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Find fg(2) — evaluate g(2) first ▶
\(fg(2)\) means \(f(g(2))\) — apply \(g\) first (innermost), then \(f\). We find \(g(2)\) numerically, then substitute that value into \(f\).

Find \(g(2)\):

$$g(2) = \frac{5}{2(2) - 9} = \frac{5}{4 - 9} = \frac{5}{-5} = -1$$

Now find \(f(-1)\):

$$fg(2) = f(-1) = 4 - 3(-1)^2 = 4 - 3(1) = 4 - 3 = 1$$

$$\boxed{fg(2) = 1}$$

M1 — Correct method: finding g(2) and substituting into f
A1 — Correct answer: 1
Part (b) Step 2: Find \(g^{-1}(x)\) with domain ▶
To invert \(g(x) = \frac{5}{2x-9}\), write \(y = \frac{5}{2x-9}\) and rearrange for \(x\). For this simpler form (constant numerator), we can multiply through and solve directly without needing to factorise.

Let \(y = \frac{5}{2x - 9}\). Rearrange for \(x\):

Multiply both sides by \((2x - 9)\):

$$y(2x - 9) = 5$$

Expand:

$$2xy - 9y = 5$$

Add \(9y\) to both sides:

$$2xy = 5 + 9y$$

Divide by \(2y\):

$$x = \frac{5 + 9y}{2y}$$

Write in inverse notation (swap \(x\) and \(y\)):

$$\boxed{g^{-1}(x) = \frac{5 + 9x}{2x}, \quad x \in \mathbb{R}, \quad x \neq 0}$$

M1 — Eliminates the fraction and isolates terms with x
A1 — Correct expression for the inverse
A1 — Correct notation including domain (\(x \neq 0\))
The domain of \(g^{-1}\) is the range of \(g\). Since \(g(x) = \frac{5}{2x-9}\) can take any non-zero value, the domain of \(g^{-1}\) is \(x \neq 0\). You must state this for full marks.
Part (c)(i) Step 3: Find gf(x) as a simplified fraction ▶
\(gf(x) = g(f(x))\): substitute the entire expression for \(f(x)\) into \(g\) wherever \(x\) appears. Since \(g(x) = \frac{5}{2x-9}\), we replace the \(x\) in \(g\) with \(f(x) = 4-3x^2\).

Substitute \(f(x) = 4 - 3x^2\) into \(g\):

$$gf(x) = g(4 - 3x^2) = \frac{5}{2(4 - 3x^2) - 9}$$

Expand the denominator:

$$= \frac{5}{8 - 6x^2 - 9} = \frac{5}{-1 - 6x^2}$$

Multiply numerator and denominator by \(-1\) to tidy up:

$$= \frac{-5}{1 + 6x^2}$$

$$\boxed{gf(x) = \frac{-5}{1 + 6x^2}}$$

M1 — Correct substitution of f into g
A1 — Correct simplified fraction
Part (c)(ii) Step 4: Deduce the range of gf(x) ▶
From the expression \(gf(x) = \frac{-5}{1+6x^2}\), we can deduce the range by analysing the denominator. Since \(x^2 \geq 0\), we have \(1 + 6x^2 \geq 1\). The fraction \(\frac{-5}{\text{something} \geq 1}\) is always negative. Its value closest to zero is when the denominator is largest (as \(x \to \infty\)), and its most negative value is when the denominator is smallest (= 1, when \(x = 0\)), giving \(\frac{-5}{1} = -5\).

The denominator \(1 + 6x^2 \geq 1\) for all real \(x\).

  • When \(x = 0\): \(gf(0) = \frac{-5}{1} = -5\) (most negative value)
  • As \(|x| \to \infty\): \(gf(x) \to 0^-\) (approaches 0 from below, never reaches it)

$$\boxed{-5 \leq gf(x) < 0}$$

B1 — Correct range deduced: \(-5 \leq gf(x) < 0\)
This can be scored even with an incorrect gf(x), provided the deduction is logically consistent. The key word "deduce" means you should not need new calculations — use what you already have from (c)(i).
Part (d) Step 5: Find values of k for which f(x) = h(x) has no real solutions ▶
"No real solutions" is a signal for the discriminant. If we set \(f(x) = h(x)\) and rearrange to \(ax^2 + bx + c = 0\), the equation has no real solutions when the discriminant \(b^2 - 4ac < 0\). We form the quadratic, identify \(a\), \(b\), \(c\) (in terms of \(k\)), then solve the inequality.

Set \(f(x) = h(x)\):

$$4 - 3x^2 = 2x^2 - 6x + k$$

Rearrange to standard form (all terms on one side):

$$0 = 5x^2 - 6x + (k - 4)$$

For no real solutions, discriminant \(< 0\):

$$b^2 - 4ac < 0$$

Here \(a = 5\), \(b = -6\), \(c = k-4\):

$$(-6)^2 - 4(5)(k-4) < 0$$

$$36 - 20(k-4) < 0$$

$$36 - 20k + 80 < 0$$

$$116 - 20k < 0$$

Add \(20k\) to both sides:

$$116 < 20k$$

Divide by 20:

$$k > \frac{116}{20} = \frac{29}{5} = 5.8$$

$$\boxed{k > \frac{29}{5}}$$

M1 — Setting f(x) = h(x) and collecting to form a 3-term quadratic = 0
dM1 — Using \(b^2 - 4ac < 0\) on their quadratic to find a range for k
A1 — Correct answer: \(k > \frac{29}{5}\) (or \(k > 5.8\))
The method mark for forming the quadratic is independent of the discriminant mark. However, the discriminant mark depends on the first M1 — you must have a correct quadratic (in terms of k) before applying the discriminant. Many candidates lost marks by making sign errors when rearranging.

How to Recognise This Question Type

Look for these signals in the question:

  • Functions defined with their domains stated explicitly (e.g. "\(f(x) = \ldots, x \in \mathbb{R}, x > 0\)")
  • The notation \(fg(x)\), \(gf(x)\), or \(ff(x)\) — always a composite
  • The notation \(f^{-1}(x)\) or \(g^{-1}\) — always an inverse
  • Keywords: "state the range", "state the domain"
  • Asking you to "describe fully the transformation" — expects type + specifics
  • Functions involving fractions like \(\frac{ax+b}{cx+d}\) — rational functions, typically inverse and composite
  • Usually appears in Q1–Q10, worth 5–11 marks, often the first substantial question on Paper 2

Key Transformation Rules

Given a point \(P(a, b)\) on \(y = f(x)\), here is where it maps to under each transformation:

  • \(y = f(x) + k\): translate \(k\) up \(\longrightarrow (a, \; b+k)\)
  • \(y = f(x + k)\): translate \(k\) left \(\longrightarrow (a-k, \; b)\)
  • \(y = kf(x)\): stretch vertically by factor \(k\) \(\longrightarrow (a, \; kb)\)
  • \(y = f(kx)\): stretch horizontally by factor \(\frac{1}{k}\) \(\longrightarrow (\frac{a}{k}, \; b)\)
  • \(y = -f(x)\): reflect in the \(x\)-axis \(\longrightarrow (a, \; -b)\)
  • \(y = f(-x)\): reflect in the \(y\)-axis \(\longrightarrow (-a, \; b)\)
  • \(y = |f(x)|\): reflect negative parts up \(\longrightarrow (a, \; |b|)\)
  • \(y = f(|x|)\): reflect right-half into left \(\longrightarrow\) y-axis symmetry

Memory trick: Changes "inside" the brackets (affecting \(x\)) go in the opposite direction and use reciprocal factors. Changes "outside" (affecting \(y\)) go in the expected direction.
