The Approach

Exponentials & logarithms questions fall into two main categories. Here is how to tackle each:

Type 1: Solving Log/Exponential Equations

1
Apply log laws to combine terms
Use \(\log a + \log b = \log(ab)\), \(\log a - \log b = \log\!\left(\frac{a}{b}\right)\), and \(n\log a = \log(a^n)\) to reduce to a single logarithm on each side.
2
Remove the logarithm
If \(\log_b(\text{expression}) = k\), then \(\text{expression} = b^k\). This converts the log equation into a polynomial (usually quadratic). If you have an exponential equation like \(a^x = k\), take logs of both sides.
3
Solve and check validity
Solve the resulting equation. Then check each solution: you cannot take the log of a negative number or zero. Reject any root that makes the argument of a logarithm non-positive, and state your reason.

Type 2: Exponential Models & Log Graphs

1
Take logarithms of the model equation
For \(y = ab^x\): take \(\log_{10}\) to get \(\log y = \log a + x\log b\). This is linear in \(x\) with gradient \(\log b\) and intercept \(\log a\).
For \(y = kx^n\): take \(\log_{10}\) to get \(\log y = \log k + n\log x\), which is linear in \(\log x\).
2
Read gradient and intercept from the graph
Use the two given points on the straight line to find the gradient and y-intercept. Match these to the transformed equation to find the constants.
3
Convert back to find original constants
If \(\log a = c\) then \(a = 10^c\). If \(\log b = m\) then \(b = 10^m\). State the complete model and interpret in context.

Common Pitfalls (from Examiner Reports)

  • Taking log of a negative: When solving log equations, candidates find both roots of the quadratic but forget to check whether they make the argument of the log negative. Always substitute back and reject invalid roots with a clear reason.
  • Confusing log laws: \(\log(a+b) \neq \log a + \log b\). The addition law is \(\log a + \log b = \log(ab)\). Similarly \(\log(a-b) \neq \log a - \log b\).
  • Forgetting to convert from \(\log a\) to \(a\): In modelling questions, candidates find \(\log_{10} a = 0.81\) but then state \(a = 0.81\) instead of \(a = 10^{0.81}\).
  • Mixing up log-linear vs log-log: For \(y = ab^x\), plot \(\log y\) against \(x\). For \(y = kx^n\), plot \(\log y\) against \(\log x\). Getting these confused leads to wrong identification of gradient/intercept.
  • Not giving exact answers: When asked for exact values, leave answers as \(\log_2 3\) or \(\frac{\ln 3}{\ln 2}\) rather than decimals.

What Top Candidates Do

  • Write down the log law being used at each step, e.g. "using \(\log a + \log b = \log(ab)\)" -- this earns method marks even if a slip follows.
  • When rejecting a root, explicitly state: "Reject \(x = ...\) because \(\log_2(\text{negative})\) is undefined" -- this earns the final accuracy mark.
  • In modelling questions, clearly state \(\log_{10} y = (\text{gradient}) \times x + (\text{intercept})\) and match terms -- this systematic approach avoids errors.
  • For equations like \(e^{0.05t} = k\), write \(0.05t = \ln k\) then \(t = \frac{\ln k}{0.05}\) -- each line is a method mark.

Q3 5 marks

(a) Given that \(2\log(4-x) = \log(x+8)\), show that \(x^2 - 9x + 8 = 0\).

(b)(i) Write down the roots of the equation \(x^2 - 9x + 8 = 0\).

(b)(ii) State which of the roots in (b)(i) is not a solution of the original equation, giving a reason.

View Question Paper View Mark Scheme
Part (a) Apply the power law for logarithms ▶
I see \(2\log(4-x)\) on the left. The power law says \(n\log a = \log(a^n)\), so I can rewrite this as a single log.

\(2\log(4-x) = \log(x+8)\)

Apply power law: \(2\log(4-x) = \log(4-x)^2\)

\(\log(4-x)^2 = \log(x+8)\)

M1 -- Uses the power law correctly to write \(\log(4-x)^2\)
Part (a) Remove the logarithms and expand ▶
Both sides are \(\log(\text{something})\). Since \(\log A = \log B \implies A = B\), I can equate the arguments.

\((4-x)^2 = x + 8\)

Expand the left side

\(16 - 8x + x^2 = x + 8\)

Collect all terms on one side

\(x^2 - 8x + 16 - x - 8 = 0\)

\(x^2 - 9x + 8 = 0 \quad \checkmark\)

M1 -- Removes logs and forms a correct equation without logs
A1 -- Correctly arrives at the printed answer with all working shown
Part (b) Factorise and check validity ▶
I need to factorise \(x^2 - 9x + 8\). I'm looking for two numbers that multiply to 8 and add to -9. That's -1 and -8.

\(x^2 - 9x + 8 = (x-1)(x-8) = 0\)

\(x = 1 \quad \text{or} \quad x = 8\)

Validity check:

If \(x = 8\): then \(4 - x = 4 - 8 = -4\), and \(\log(-4)\) is not defined.

Therefore \(x = 8\) is not a valid solution of the original equation.

Check \(x = 1\): \(\log(4-1) = \log 3\) (valid), \(\log(1+8) = \log 9\) (valid). And \(2\log 3 = \log 9\) -- confirmed.

B1 -- Both roots stated: \(x = 1\) and \(x = 8\)
B1 -- States \(x = 8\) is invalid because it makes the argument of log negative
Many candidates found both roots but failed to give a clear reason for rejecting \(x = 8\). Simply writing "doesn't work" is not enough. You must reference the fact that \(\log(\text{negative})\) is undefined.
Q3 5 marks

(a) Given that \(\log_2(x+3) + \log_2(x+10) = 2 + 2\log_2 x\), show that \(3x^2 - 13x - 30 = 0\).

(b)(i) Write down the roots of \(3x^2 - 13x - 30 = 0\).

(b)(ii) Hence state which root is not a solution of the original equation, giving a reason.

View Question Paper View Mark Scheme
Part (a) Apply log laws to both sides ▶
I need to combine the left side using the addition law, and rewrite the right side. The constant 2 can be written as \(\log_2 4\), and \(2\log_2 x = \log_2 x^2\).

Left side: \(\log_2(x+3) + \log_2(x+10) = \log_2[(x+3)(x+10)]\)

Addition law: \(\log a + \log b = \log(ab)\)

Right side: \(2 + 2\log_2 x = \log_2 4 + \log_2 x^2 = \log_2(4x^2)\)

Using \(2 = \log_2 4\) and power law then addition law

M1 -- Uses at least one correct log law (addition, power, or converting the constant)
Part (a) Equate arguments and expand to quadratic ▶
Now both sides are a single \(\log_2\). So I can equate the arguments and expand.

\(\log_2[(x+3)(x+10)] = \log_2(4x^2)\)

If \(\log_2 A = \log_2 B\) then \(A = B\)

\((x+3)(x+10) = 4x^2\)

Expand the left side

\(x^2 + 13x + 30 = 4x^2\)

Rearrange: bring everything to one side

\(0 = 4x^2 - x^2 - 13x - 30\)

\(3x^2 - 13x - 30 = 0 \quad \checkmark\)

dM1 -- Fully correct log work leading to an equation without logs (depends on first M1)
A1 -- Obtains the printed answer \(3x^2 - 13x - 30 = 0\) with no errors
Part (b) Solve the quadratic and reject invalid root ▶
I need to factorise \(3x^2 - 13x - 30\). Product = \(3 \times (-30) = -90\), sum = \(-13\). Factors: \(-18\) and \(+5\).

\(3x^2 - 13x - 30 = (3x + 5)(x - 6) = 0\)

\(x = 6 \quad \text{or} \quad x = -\tfrac{5}{3}\)

Validity check:

If \(x = -\tfrac{5}{3}\): the original equation has \(2\log_2 x\), which requires \(x > 0\).

Since \(-\tfrac{5}{3} < 0\), \(\log_2\!\left(-\tfrac{5}{3}\right)\) is not defined.

Therefore \(x = -\tfrac{5}{3}\) is not a valid solution.

B1 -- Both roots correct: \(x = 6\) and \(x = -\frac{5}{3}\)
B1 -- Rejects \(x = -\frac{5}{3}\) with reason: cannot take \(\log_2\) of a negative number
The mark scheme accepts any valid reason referencing the domain restriction: "\(\log_2\) of a negative is not real" or "\(x\) must be positive for \(\log_2 x\) to exist". Simply stating "negative so rejected" without linking to the log is insufficient.
Q9 9 marks

A research engineer tests braking distance. The braking distance, \(d\) metres, at speed \(V\) km/h is modelled by \(d = kV^n\) where \(k\) and \(n\) are constants.

A graph of \(\log_{10} d\) against \(\log_{10} V\) gives a straight line. A data point is \((30, 20)\) on the original \(d\) vs \(V\) graph.

(a) Explain how the log-log graph supports the model \(d = kV^n\), and show \(k \approx 0.017\). [3]

(b) Find a complete equation for the model giving \(n\) to 3 s.f. [3]

(c) Sean drives at 60 km/h in wet conditions and notices a puddle 100 m ahead. Reaction time 0.8 s. Can he stop in time? [3]

View Question Paper View Mark Scheme
Part (a) Take logs of the model to get a linear form ▶
If \(d = kV^n\), I take \(\log_{10}\) of both sides. This should give something linear, which explains the straight-line graph.

\(d = kV^n\)

Take \(\log_{10}\) of both sides

\(\log_{10} d = \log_{10}(kV^n)\)

Use \(\log(ab) = \log a + \log b\) and \(\log(a^n) = n\log a\)

\(\log_{10} d = \log_{10} k + n\log_{10} V\)

This has the form \(Y = nX + c\) where \(Y = \log_{10} d\), \(X = \log_{10} V\), gradient \(= n\), intercept \(= \log_{10} k\).

A straight line on the \(\log_{10} d\) vs \(\log_{10} V\) graph confirms the relationship \(d = kV^n\).

M1 -- Takes logs correctly to obtain a linear relationship
A1 -- Correctly identifies that a straight line on a log-log plot supports the power model
Part (a) Use the intercept to show \(k \approx 0.017\) ▶
The y-intercept of the log-log graph (where \(\log_{10} V = 0\)) gives \(\log_{10} k\). From the graph, this intercept is approximately \(-1.77\).

\(\log_{10} k \approx -1.77\)

Convert back: \(k = 10^{-1.77}\)

\(k \approx 0.017\)

A1 -- Shows \(k \approx 0.017\) from the intercept of the log-log graph
Part (b) Find \(n\) using the data point ▶
I know \(d = kV^n\) with \(k = 0.017\) and the data point \((V, d) = (30, 20)\). I substitute to find \(n\).

\(20 = 0.017 \times 30^n\)

Divide both sides by 0.017

\(\frac{20}{0.017} = 30^n\)

\(1176.47... = 30^n\)

Take \(\log_{10}\) of both sides

\(\log_{10}(1176.47) = n\log_{10}(30)\)

\(n = \frac{\log_{10}(1176.47)}{\log_{10}(30)} = \frac{3.0706...}{1.4771...}\)

\(n = 2.08 \text{ (3 s.f.)}\)

M1 -- Substitutes the data point into the model with \(k = 0.017\)
M1 -- Correct method to isolate \(n\) using logarithms
A1 -- \(n = 2.08\) (3 s.f.)
Part (c) Apply the model: can Sean stop in time? ▶
Sean is at 60 km/h. I need the braking distance from the model, plus the distance covered during 0.8 seconds of reaction time.

Braking distance:

\(d = 0.017 \times 60^{2.08}\)

\(d = 0.017 \times 4741.6... = 80.6 \text{ m}\)

Distance during reaction time:

Convert 60 km/h to m/s: \(60 \times \frac{1000}{3600} = \frac{50}{3}\) m/s

\(\text{Reaction distance} = \frac{50}{3} \times 0.8 = 13.3 \text{ m}\)

Total stopping distance:

\(80.6 + 13.3 = 93.9 \text{ m}\)

Since \(93.9 < 100\), Sean can stop before reaching the puddle.

M1 -- Substitutes \(V = 60\) into the model to find braking distance
M1 -- Calculates reaction distance and adds to braking distance
A1 -- Correct conclusion with supporting calculation
Candidates who forgot to convert km/h to m/s for the reaction distance calculation lost the final mark. Always check units are consistent. Note: 60 km/h = 16.67 m/s, so in 0.8 s the car travels 13.3 m.
Q10 8 marks

A scientist studies bees and wasps on an island.

The number of bees (thousands): \(N_b = 45 + 220e^{0.05t}\)

The number of wasps (thousands): \(N_w = 10 + 800e^{-0.05t}\)

where \(t\) is years from the start of the study.

(a) Find the number of bees at the start of the study. [1]

(b) Show that, 10 years after the start, the number of bees was increasing at approximately 18 thousand per year. [3]

(c) Find the value of \(T\) when there are equal numbers of bees and wasps, to 2 d.p. [4]

View Question Paper View Mark Scheme
Part (a) Find initial number of bees ▶
"At the start of the study" means \(t = 0\). I substitute into the formula, remembering that \(e^0 = 1\).

\(N_b = 45 + 220e^{0.05 \times 0} = 45 + 220e^0 = 45 + 220(1) = 265\)

The number of bees at the start is 265 thousand (or 265 000).

B1 -- 265 thousand (must indicate thousands, not just "265")
The mark scheme requires reference to thousands. Writing just "265" without indicating the unit risks losing this mark. The answer "265 000" is also acceptable.
Part (b) Differentiate and evaluate the rate of growth ▶
"Rate of increase" means I need \(\frac{dN_b}{dt}\). For \(Ce^{kt}\), the derivative is \(Cke^{kt}\). The constant 45 differentiates to zero.

\(N_b = 45 + 220e^{0.05t}\)

Differentiate with respect to \(t\)

\(\frac{dN_b}{dt} = 220 \times 0.05 \times e^{0.05t} = 11e^{0.05t}\)

Substitute \(t = 10\)

\(\frac{dN_b}{dt}\bigg|_{t=10} = 11e^{0.05 \times 10} = 11e^{0.5}\)

\(= 11 \times 1.6487... = 18.14...\)

\(\approx 18 \text{ thousand per year} \quad \checkmark\)

M1 -- Differentiates to get \(ke^{0.05t}\) where \(k \neq 220\)
M1 -- Substitutes \(t = 10\) into their derivative
A1 -- Shows intermediate value (e.g. \(11e^{0.5}\) or awrt 18.1) and concludes approximately 18 thousand per year
Part (c) Set equal and form a quadratic in \(e^{0.05t}\) ▶
I need to find when \(N_b = N_w\). Setting them equal gives an equation with both \(e^{0.05t}\) and \(e^{-0.05t}\). The key insight is that \(e^{-0.05t} = \frac{1}{e^{0.05t}}\), so I can multiply through to get a quadratic.

\(45 + 220e^{0.05t} = 10 + 800e^{-0.05t}\)

Rearrange

\(220e^{0.05t} - 800e^{-0.05t} + 35 = 0\)

Multiply every term by \(e^{0.05t}\) (always positive, so inequality unchanged)

\(220(e^{0.05t})^2 + 35e^{0.05t} - 800 = 0\)

This is a quadratic in \(e^{0.05t}\).

M1 -- Sets both models equal and simplifies constant terms
A1 -- Correct quadratic equation in \(e^{0.05t}\)
Part (c) Solve the quadratic and find \(T\) ▶
I use the quadratic formula with \(a = 220\), \(b = 35\), \(c = -800\). Since \(e^{0.05t} > 0\) always, I reject any negative solution.

Let \(u = e^{0.05t}\). Then \(220u^2 + 35u - 800 = 0\)

\(u = \frac{-35 \pm \sqrt{35^2 + 4(220)(800)}}{2(220)}\)

\(u = \frac{-35 \pm \sqrt{1225 + 704000}}{440}\)

\(u = \frac{-35 \pm \sqrt{705225}}{440} = \frac{-35 \pm 839.77...}{440}\)

\(u = 1.829... \quad \text{or} \quad u = -1.988...\)

Reject \(u = -1.988\) since \(e^{0.05t} > 0\)

\(e^{0.05t} = 1.829...\)

Take natural log of both sides

\(0.05t = \ln(1.829...) = 0.6041...\)

\(t = \frac{0.6041...}{0.05} = 12.08...\)

\(T = 12.08\) (to 2 decimal places)

M1 -- Solves quadratic for \(u = e^{0.05t}\), then takes \(\ln\) to find \(t\)
A1 -- \(T = 12.08\)
The substitution \(u = e^{0.05t}\) (or multiplying through by \(e^{0.05t}\)) is the key step most candidates missed. Those who attempted to take \(\ln\) directly of a sum like \(\ln(220e^{0.05t} + 35)\) made a fundamental error -- you cannot split \(\ln\) across addition.
Q2 3 marks

By taking logarithms of both sides, solve the equation

\[4^{3p-1} = 5210\]

giving the value of \(p\) to one decimal place.

View Question Paper View Mark Scheme
Solution Take logs and apply the power rule ▶
The unknown \(p\) is in the exponent, so I take logarithms of both sides. Any base works -- I will use \(\log_{10}\). Then I bring the exponent down using the power rule \(\log(a^n) = n\log a\).

\(4^{3p-1} = 5210\)

Take \(\log_{10}\) of both sides

\(\log(4^{3p-1}) = \log(5210)\)

Power rule: bring the exponent down

\((3p-1)\log 4 = \log 5210\)

M1 -- Takes logs correctly and applies the power rule to bring the exponent down
Solution Solve for \(p\) ▶
Now I isolate \(p\) using standard algebra -- divide by \(\log 4\), then solve the linear equation in \(p\).

\(3p - 1 = \frac{\log 5210}{\log 4}\)

Calculate: \(\log 5210 = 3.7168...\), \(\log 4 = 0.6021...\)

\(3p - 1 = 6.1726...\)

\(3p = 7.1726...\)

\(p = 2.3908...\)

\(p = 2.4 \text{ (1 d.p.)}\)

M1 -- Correct method to isolate \(p\) (divides by \(\log 4\) and solves linear equation)
A1 -- \(p = 2.4\)
This is a straightforward "take logs" question worth 3 easy marks. The most common error is writing \(\log(3p-1) \times \log 4\) instead of \((3p-1)\log 4\). Remember: the power rule says the entire exponent multiplies the log of the base.
