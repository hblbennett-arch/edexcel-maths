The Approach

A. Sign Change Argument (Locating a Root)

Used to show a root lies in an interval \([a, b]\):

1
Evaluate \(f(a)\) and \(f(b)\)
Substitute the endpoints into the function. Show your calculations clearly.
2
Identify the sign change
State that \(f(a) < 0\) and \(f(b) > 0\) (or vice versa). Be explicit about the signs.
3
State continuity and conclude
Write: "Since there is a change of sign and \(f\) is continuous, there is a root in \([a, b]\)." Both "sign change" and "continuous" are required for full marks.

B. Iteration Formula (Fixed Point Iteration)

Used to find approximate solutions by repeated substitution:

1
Apply the formula with \(x_1\)
Substitute the given starting value into \(x_{n+1} = g(x_n)\) to find \(x_2\).
2
Iterate repeatedly
Keep substituting: \(x_2 \to x_3 \to x_4 \to \ldots\) Show at least the values asked for to the required decimal places.
3
State the converged value
When successive values agree to the required accuracy, state the root. Show enough iterations to demonstrate convergence.

C. Newton-Raphson Method

More powerful iteration using the tangent line: \(\displaystyle x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\)

1
Find \(f'(x)\) if not given
Differentiate the function. This is often a preceding part of the question.
2
Evaluate \(f(x_0)\) and \(f'(x_0)\)
Substitute the given starting value into both \(f\) and \(f'\). Show these calculations.
3
Apply the formula
Compute \(x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\). Give the answer to the required decimal places.

D. Trapezium Rule

Approximating \(\int_a^b f(x)\,dx\) using trapezoid strips:

$$\int_a^b y\,dx \approx \frac{h}{2}\left[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})\right]$$

where \(h = \frac{b-a}{n}\) is the strip width.

1
Identify \(h\)
Calculate the strip width from the table: \(h = x_1 - x_0\). State this value explicitly (earns B1).
2
Apply the formula
First + Last \(y\) values, plus 2 times the sum of all middle \(y\) values, all multiplied by \(\frac{h}{2}\).
3
Calculate carefully
Show the bracket contents before multiplying by \(\frac{h}{2}\). Give answer to required accuracy.

Common Pitfalls (from Examiner Reports)

  • Not stating continuity in sign change: Writing "change of sign therefore root" without mentioning "f is continuous" loses the A1 mark every time. Both elements are required.
  • Insufficient iterations shown: When asked to find a root by iteration, you must show enough steps. Writing only the final answer with no intermediate values scores M0.
  • Wrong trapezium rule formula: Forgetting the factor of 2 on the middle terms, or confusing \(h\) with the number of strips. Also, missing the outer brackets around the formula.
  • Newton-Raphson with no evidence: Simply writing the answer without showing \(x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\) with substituted values earns M0.
  • Rounding too early in iteration: Use full calculator precision for intermediate values. Only round the final answer to the requested accuracy.
  • No conclusion in sign change: After calculating values and noting signs, you must write a conclusion such as "therefore root lies in [3.6, 3.7]".

What Top Candidates Do

  • For sign change: write a template sentence: "f(a) = ..., f(b) = .... Change of sign, f continuous, therefore root in [a, b]." This guarantees all marking points.
  • For iteration: show at least one more iteration than asked, to demonstrate convergence. Write values to at least one more decimal place than required.
  • For Newton-Raphson: explicitly write \(x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\) with numerical substitution shown, even if only one application is asked for.
  • For trapezium rule: state \(h = ...\) separately before substituting. This earns the B1 mark even if arithmetic goes wrong later.

Paper 1 - June 2018 - Question 4 (4 marks)

The curve with equation \(y = 2\ln(8 - x)\) meets the line \(y = x\) at a single point, \(x = \alpha\).

(a) Show that \(3 < \alpha < 4\) [2 marks]

(b) A student uses the iteration formula \(x_{n+1} = 2\ln(8 - x_n)\) with \(x_1 = 4\). Using the graph, determine whether or not this iteration formula can be used to find an approximation for \(\alpha\), justifying your answer. [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Set up the function and evaluate at endpoints ▶
The curves \(y = 2\ln(8-x)\) and \(y = x\) meet where \(2\ln(8-x) = x\), i.e. where \(2\ln(8-x) - x = 0\). Define \(f(x) = 2\ln(8-x) - x\). If \(f\) changes sign between 3 and 4, there must be a root in that interval.

At the intersection, \(2\ln(8-x) = x\). Let \(f(x) = 2\ln(8-x) - x\):

$$f(3) = 2\ln(8-3) - 3 = 2\ln 5 - 3 = 2(1.6094...) - 3 = 0.219... > 0$$

$$f(4) = 2\ln(8-4) - 4 = 2\ln 4 - 4 = 2(1.3863...) - 4 = -1.227... < 0$$

Since \(f(3) > 0\) and \(f(4) < 0\), there is a change of sign. As \(f\) is continuous on \([3, 4]\), there is a root \(\alpha\) in this interval. Hence \(3 < \alpha < 4\).

M1 - Attempts \(f(3)\) and \(f(4)\) and considers signs
A1* - Both values correct to 1 s.f., sign change stated, continuity referenced, conclusion given
You must mention "continuous" (or equivalent) AND give a conclusion. Just writing "sign change" without "continuous" or without "therefore root" loses the A1. Minimum acceptable: "change of sign, continuous, root in [3, 4]".
Part (b) Step 2: Determine convergence using a cobweb diagram ▶
The iteration \(x_{n+1} = 2\ln(8-x_n)\) means \(x_{n+1} = g(x_n)\) where \(g(x) = 2\ln(8-x)\). Starting at \(x_1 = 4\), we draw a cobweb on the given graph of \(y = g(x)\) and \(y = x\). If the cobweb spirals inwards, the iteration converges to the root.

Starting at \(x_1 = 4\) on the line \(y = x\):

  • Go vertically to curve \(y = 2\ln(8-x)\): gives \(y_1 = 2\ln 4 \approx 2.77\)
  • Go horizontally to line \(y = x\): gives \(x_2 \approx 2.77\)
  • Go vertically to curve: gives \(y_2 = 2\ln(5.23) \approx 3.31\)
  • Go horizontally to line: gives \(x_3 \approx 3.31\)

The cobweb diagram spirals inward toward the intersection point \(\alpha\).

The iteration formula can be used to find an approximation for \(\alpha\) because the cobweb spirals inwards (the iterations converge to the root).

M1 - Drawing/describing a cobweb diagram starting at \(x_1 = 4\) with at least two spirals
A1 - Correct cobweb spiralling inwards AND statement that the iteration can be used with reason
You must use the graph (draw the cobweb). A purely numerical approach was not the intended method. Suitable reasons for convergence: "it spirals inwards", "it gets closer to the root", "it converges".

Paper 2 - June 2018 - Question 5 (6 marks)

The equation \(2x^3 + x^2 - 1 = 0\) has exactly one real root.

(a) Show that, for this equation, the Newton-Raphson formula can be written as $$x_{n+1} = \frac{4x_n^3 + x_n^2 + 1}{6x_n^2 + 2x_n}$$ [3 marks]

(b) Using the formula in part (a) with \(x_1 = 1\), find the values of \(x_2\) and \(x_3\). [2 marks]

(c) Explain why, for this question, the Newton-Raphson method cannot be used with \(x_1 = 0\). [1 mark]

View Question Paper View Mark Scheme
Part (a) Step 1: Write down \(f(x)\) and \(f'(x)\) ▶
The Newton-Raphson formula is \(x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\). To derive the given form, we need both \(f(x)\) and \(f'(x)\), then combine into a single fraction.

From the equation \(2x^3 + x^2 - 1 = 0\):

$$f(x) = 2x^3 + x^2 - 1$$

Differentiate term by term:

$$f'(x) = 6x^2 + 2x$$

B1 - States \(f'(x) = 6x^2 + 2x\)
Part (a) Step 2: Substitute into Newton-Raphson and simplify ▶
We combine \(x_n - \frac{f(x_n)}{f'(x_n)}\) into a single fraction by writing \(x_n\) over the common denominator, then simplify the numerator.

$$x_{n+1} = x_n - \frac{2x_n^3 + x_n^2 - 1}{6x_n^2 + 2x_n}$$

Write as a single fraction:

$$= \frac{x_n(6x_n^2 + 2x_n) - (2x_n^3 + x_n^2 - 1)}{6x_n^2 + 2x_n}$$

Expand numerator: \(6x_n^3 + 2x_n^2 - 2x_n^3 - x_n^2 + 1\)

$$= \frac{4x_n^3 + x_n^2 + 1}{6x_n^2 + 2x_n}$$

$$\boxed{x_{n+1} = \frac{4x_n^3 + x_n^2 + 1}{6x_n^2 + 2x_n}} \quad \square$$

M1 - Substitutes \(f(x_n)\) and their \(f'(x_n)\) into the Newton-Raphson formula
A1* - Reaches the given answer with clear algebraic working
Part (b) Step 3: Calculate \(x_2\) and \(x_3\) ▶
Straightforward substitution. Put \(x_1 = 1\) into the formula to get \(x_2\), then \(x_2\) in to get \(x_3\).

With \(x_1 = 1\):

$$x_2 = \frac{4(1)^3 + (1)^2 + 1}{6(1)^2 + 2(1)} = \frac{4 + 1 + 1}{6 + 2} = \frac{6}{8} = \frac{3}{4}$$

With \(x_2 = \frac{3}{4}\):

$$x_3 = \frac{4\left(\frac{3}{4}\right)^3 + \left(\frac{3}{4}\right)^2 + 1}{6\left(\frac{3}{4}\right)^2 + 2\left(\frac{3}{4}\right)} = \frac{\frac{27}{16} + \frac{9}{16} + \frac{16}{16}}{\frac{54}{16} + \frac{24}{16}} = \frac{\frac{52}{16}}{\frac{78}{16}} = \frac{52}{78} = \frac{2}{3}$$

$$x_2 = \frac{3}{4}, \quad x_3 = \frac{2}{3}$$

M1 - Correct substitution of \(x_1 = 1\) into the formula
A1 - Both \(x_2 = \frac{3}{4}\) and \(x_3 = \frac{2}{3}\) correct
Part (c) Step 4: Explain why \(x_1 = 0\) cannot be used ▶
Newton-Raphson uses the tangent line at each point. If \(f'(x_0) = 0\), the tangent is horizontal and never crosses the x-axis, making the formula undefined (division by zero).

When \(x_1 = 0\):

$$f'(0) = 6(0)^2 + 2(0) = 0$$

The denominator of the Newton-Raphson formula is zero, so the formula is undefined.

There is a stationary point at \(x = 0\), so the tangent to the curve is horizontal and would never cross the x-axis. The method cannot produce a next approximation.

B1 - Valid explanation referring to the stationary point or horizontal tangent at \(x = 0\)
Acceptable: "stationary point at x = 0", "tangent is horizontal", "f'(0) = 0 so division by zero", "tangent would not meet the x-axis". All score B1.

Paper 2 - October 2020 - Question 2 (5 marks)

A table of values is given:

\(x\) 0.5 1 1.5 2 2.5
\(y\) 0.5774 0.7071 0.7746 0.8165 0.8452

(a) Use the trapezium rule, with all the values of \(y\) in the table, to find an estimate for \(\displaystyle\int_{0.5}^{2.5} \frac{\sqrt{x}}{\sqrt{1+x}}\,dx\), giving your answer to 3 significant figures. [3 marks]

(b) Using your answer to part (a), deduce an estimate for \(\displaystyle\int_{0.5}^{2.5} \frac{9\sqrt{x}}{\sqrt{1+x}}\,dx\) [1 mark]

(c) Given that \(\displaystyle\int_{0.5}^{2.5} \frac{9\sqrt{x}}{\sqrt{1+x}}\,dx = 4.535\) (4 s.f.), comment on the accuracy of your estimate. [1 mark]

View Question Paper View Mark Scheme
Part (a) Step 1: Identify the strip width \(h\) ▶
The trapezium rule needs \(h\), the width of each strip. Read from the table: x-values are 0.5, 1, 1.5, 2, 2.5 with gaps of 0.5. Stating \(h\) explicitly earns a B1 mark.

From the table, the \(x\) values are equally spaced:

$$h = 1 - 0.5 = 0.5$$

There are 5 ordinates giving 4 strips.

B1 - States or uses \(h = 0.5\)
Part (a) Step 2: Apply the trapezium rule formula ▶
The formula is \(\frac{h}{2}[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})]\). Identify "first and last" (\(y_0 = 0.5774\), \(y_4 = 0.8452\)) and "middle ones that get doubled" (\(y_1 = 0.7071\), \(y_2 = 0.7746\), \(y_3 = 0.8165\)).

$$\int_{0.5}^{2.5} y\,dx \approx \frac{h}{2}\left[y_0 + y_4 + 2(y_1 + y_2 + y_3)\right]$$

Substitute values:

$$= \frac{0.5}{2}\left[0.5774 + 0.8452 + 2(0.7071 + 0.7746 + 0.8165)\right]$$

First + Last = 1.4226; Middle sum = 2.2982; Doubled = 4.5964

$$= 0.25 \times [1.4226 + 4.5964]$$

$$= 0.25 \times 6.0190 = 1.50475...$$

$$\boxed{\approx 1.50 \text{ (3 s.f.)}}$$

M1 - Correct structure of trapezium rule with all y values used, no repeats
A1 - Answer of awrt 1.50
Writing \(\frac{h}{2} \times\) first + last + 2(middle) without enclosing outer brackets scores M0 unless the correct answer is obtained (implying the brackets). Always write the full formula clearly.
Part (b) Step 3: Deduce the related integral ▶
The new integrand \(\frac{9\sqrt{x}}{\sqrt{1+x}}\) is simply 9 times the original. Since \(\int 9f(x)\,dx = 9\int f(x)\,dx\), multiply your answer by 9. Do NOT apply the trapezium rule again.

Since \(\frac{9\sqrt{x}}{\sqrt{1+x}} = 9 \times \frac{\sqrt{x}}{\sqrt{1+x}}\):

$$\int_{0.5}^{2.5} \frac{9\sqrt{x}}{\sqrt{1+x}}\,dx \approx 9 \times 1.50 = 4.51 \text{ (3 s.f.)}$$

B1ft - Correctly deduces \(9 \times\) their answer to part (a)
Attempting the trapezium rule again here scores M0. The question says "using your answer to part (a)" - it requires you to recognise the factor of 9 relationship.
Part (c) Step 4: Comment on accuracy ▶
Compare our estimate (4.51) to the exact value (4.535). The difference tells us whether the trapezium rule overestimates or underestimates, and by how much.

Our estimate: \(\approx 4.51\)

Exact value: 4.535

$$\text{Percentage error} = \frac{|4.535 - 4.51|}{4.535} \times 100\% \approx 0.55\%$$

The trapezium rule gives an underestimate. This is because the curve \(y = \frac{\sqrt{x}}{\sqrt{1+x}}\) is concave in this region, so the straight-line tops of the trapezia lie below the curve.

B1 - States it is an underestimate with valid reason, or gives a sensible percentage error

Paper 2 - June 2022 - Question 6 (7 marks)

The curve has equation \(y = f(x)\) where

$$f(x) = 8\sin\left(\frac{x}{2}\right) - 3x + 9, \quad x > 0$$

(\(x\) in radians). The point P is a local maximum on the curve.

(a) Using calculus, find the \(x\) coordinate of P, giving your answer to 3 significant figures. [4 marks]

The curve crosses the \(x\)-axis at \(x = \alpha\). Given that \(f(4) = 4.274\) and \(f(5) = -1.212\) (to 3 d.p.):

(b) Explain why \(\alpha\) must lie in the interval \([4, 5]\). [1 mark]

(c) Taking \(x_0 = 5\) as a first approximation to \(\alpha\), apply the Newton-Raphson method once to obtain a second approximation. Give your answer to 3 significant figures. [2 marks]

View Question Paper View Mark Scheme
Part (a) Step 1: Differentiate \(f(x)\) ▶
At a local maximum, \(f'(x) = 0\). First differentiate \(f(x) = 8\sin(\frac{x}{2}) - 3x + 9\). The chain rule on \(\sin(\frac{x}{2})\) gives a factor of \(\frac{1}{2}\).

Differentiate \(f(x) = 8\sin\left(\frac{x}{2}\right) - 3x + 9\):

Chain rule on \(8\sin(\frac{x}{2})\): \(8 \times \cos(\frac{x}{2}) \times \frac{1}{2} = 4\cos(\frac{x}{2})\)

Derivative of \(-3x\) is \(-3\); derivative of constant 9 is 0.

$$f'(x) = 4\cos\left(\frac{x}{2}\right) - 3$$

M1 - Differentiating to get expression of form \(k\cos(\frac{x}{2}) + c\)
A1 - Correct derivative: \(4\cos(\frac{x}{2}) - 3\)
Part (a) Step 2: Solve \(f'(x) = 0\) and select the correct maximum ▶
Set the derivative to zero and solve. Since cosine is periodic, there are multiple solutions. We use the sketch to select the one corresponding to P.

$$4\cos\left(\frac{x}{2}\right) - 3 = 0$$

Rearrange:

$$\cos\left(\frac{x}{2}\right) = \frac{3}{4}$$

Principal value:

$$\frac{x}{2} = \cos^{-1}\left(\frac{3}{4}\right) = 0.7227...$$

Multiply by 2:

$$x = 2 \times 0.7227... = 1.445... \approx 1.45$$

Verify this is a maximum: \(f(1.45) = 8\sin(0.72) - 3(1.45) + 9 \approx 5.29 - 4.35 + 9 = 9.94 > 0\). The curve has a local maximum here, consistent with the sketch showing P above the x-axis before the root \(\alpha\).

$$\boxed{x = 1.45 \text{ (3 s.f.)}}$$

dM1 - Sets derivative = 0 and proceeds to find a value for \(x\)
A1 - Selects the correct turning point matching the sketch (3 s.f.)
There are multiple solutions to \(\cos(\frac{x}{2}) = \frac{3}{4}\). You must select the one matching the maximum P shown in the figure. A correct answer with no working scores 0 marks.
Part (b) Step 3: Explain why \(\alpha\) lies in \([4, 5]\) ▶
Standard sign change argument. We are given \(f(4) = 4.274 > 0\) and \(f(5) = -1.212 < 0\). A sign change with continuity guarantees a root.

We are given:

$$f(4) = 4.274 > 0$$

$$f(5) = -1.212 < 0$$

There is a change of sign. Since \(f\) is continuous on \([4, 5]\), there must be a root \(\alpha\) in this interval.

B1 - Change of sign stated, continuity referenced, conclusion given
All three elements required: sign change, continuity, conclusion. Minimum: "change of sign and continuous, so root in [4, 5]". Statements like "x is continuous" or "the interval is continuous" do not earn the mark.
Part (c) Step 4: Apply Newton-Raphson with \(x_0 = 5\) ▶
Newton-Raphson formula: \(x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\). We know \(f(5) = -1.212\). Calculate \(f'(5)\) using our derivative from part (a). Then substitute.

We need \(f(5)\) and \(f'(5)\):

$$f(5) = -1.212 \quad \text{(given)}$$

Calculate \(f'(5)\):

$$f'(5) = 4\cos\left(\frac{5}{2}\right) - 3 = 4\cos(2.5) - 3 = 4(-0.8011) - 3 = -6.204...$$

Apply Newton-Raphson:

$$x_1 = 5 - \frac{f(5)}{f'(5)} = 5 - \frac{-1.212}{-6.204}$$

Compute the fraction: \(\frac{-1.212}{-6.204} = 0.1954...\)

$$x_1 = 5 - 0.1954... = 4.8046...$$

$$\boxed{\alpha \approx 4.80 \text{ (3 s.f.)}}$$

M1 - Attempts \(x_1 = 5 - \frac{f(5)}{f'(5)}\) with evidence of the formula being used
A1 - \(x_1 =\) awrt 4.80 (must be 4.80, not just 4.8)
You must show evidence that the Newton-Raphson formula is being applied. Simply writing "4.80" with no working earns M0 A0. The accuracy of your answer (e.g. 4.805) may imply the method if consistent with the N-R calculation.

How to Recognise This Question Type

Look for these signals in the question:

  • "Show that \(\alpha\) lies in the interval..." - sign change argument
  • "Use the iteration formula \(x_{n+1} = ...\)" - fixed point iteration
  • "Apply the Newton-Raphson method" - need \(f(x)\) and \(f'(x)\)
  • "Use the trapezium rule with all values..." - direct application of formula
  • "Using your answer to part (a), deduce..." - scale/transform the trapezium result (do NOT re-apply)
  • A table of \(x\) and \(y\) values with equal spacing - trapezium rule
  • "Find \(x_2\), \(x_3\), ..." or "find \(\alpha\) to \(n\) decimal places" - iteration
  • "Explain why Newton-Raphson cannot be used with..." - stationary point / horizontal tangent

Key Formulas

  • Trapezium Rule: \(\displaystyle\int_a^b y\,dx \approx \frac{h}{2}\left[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})\right]\) where \(h = \frac{b-a}{n}\)
  • Newton-Raphson: \(\displaystyle x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\)
  • Iteration converges if: \(|g'(\alpha)| < 1\) near the root
  • Sign change test: If \(f(a)\) and \(f(b)\) have opposite signs and \(f\) is continuous on \([a,b]\), then \(f\) has at least one root in \((a,b)\)
